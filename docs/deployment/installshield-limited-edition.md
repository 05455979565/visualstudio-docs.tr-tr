# <a name="installshield-limited-edition"></a>InstallShield Limited Edition
InstallShield Limited Edition'ı kullanarak, bir kurulum dosyası oluşturabilir ve böylece bir ağa bağlı olmayan kullanıcılar bir masaüstü uygulamanın veya bileşenin yükleyebilir kullanıcılara dağıtabilirsiniz. Visual Studio Professional ve Enterprise sürümleri kullanıcılar için InstallShield Limited Edition ücretsizdir. Visual Studio artık destekleyen Windows Installer teknolojisi değiştirir. Alternatif olarak, uygulamaları ve bileşenleri, ağ bağlantısı gerektiren ClickOnce kullanarak dağıtabilirsiniz. Bkz: [ClickOnce güvenliği ve dağıtımı](../deployment/clickonce-security-and-deployment.md).  
  
> [!NOTE]
>  Visual Studio'nun önceki sürümleri, Visual Studio yükleyici projeleri uzantısı yükleyerek oluşturulan Windows Yükleyici projeleri kullanmaya devam edebilirsiniz. Bkz: [Visual Studio yükleyicisi projeleri uzantısı](http://blogs.msdn.com/b/visualstudio/archive/2014/04/17/visual-studio-installer-projects-extension.aspx).  
  
### <a name="to-enable-installshield-limited-edition"></a>InstallShield Limited Edition etkinleştirmek için  
  
1.  Menü çubuğunda seçin **dosya**, **yeni**, **proje**.  
  
2.  İçinde **yeni proje** iletişim kutusunda, genişletin **diğer proje türleri** düğümünü ve ardından **Kurulum ve dağıtım** düğümü.  
  
3.  Şablon listede seçin **etkinleştirmek InstallShield Limited Edition**ve ardından **Tamam** düğmesi.  
  
4.  Açılır tarayıcı penceresinde yönergeleri okuyun ve ardından **indirme web sitesine gidin** bağlantı.
