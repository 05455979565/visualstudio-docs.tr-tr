# <a name="c6305"></a>C6305
uyarı C6305: sizeof ve CountOf miktarları arasında olası uyuşmazlık

 Bu uyarı `sizeof` sonucunu tutan bir değişkenin bir işaretçiye veya `countof` ifadesine eklendiğini veya bu ifadeye bırakıldığını gösterir. Bu, işaretçi aritmetiğinin beklenmedik şekilde ölçeklendirilmesine neden olur.

## <a name="example"></a>Örnek
 Aşağıdaki kod bu uyarıyı oluşturur:

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:

```
void f(int *p)
{
  // code...
  p += 1;
}
```
