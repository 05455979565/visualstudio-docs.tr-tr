# <a name="c28196"></a>C28196
C28196 Uyarı: Gereksinim karşılanmıyor. (İfade true olarak değerlendirilmiyor.)

 Bu uyarı, işlev prototipi analiz edilen işlev için olduğunu gösterir. bir `__notnull`, `__null` veya `__drv_valueIs` üzerinde bir `_Out_` parametre veya dönüş değeri, ancak değeri döndürdü, ek açıklama ile tutarsız.
