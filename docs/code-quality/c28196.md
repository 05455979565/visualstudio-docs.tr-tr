# <a name="c28196"></a>C28196
C28196 Uyarı: gereksinim sağlanmıyorsa. (İfade true olarak değerlendirmez.)  
  
 Bu uyarı çözümlenmekte işlevi için işlev prototipi olduğunu gösterir bir `__notnull`, `__null` veya `__drv_valueIs` üzerinde bir `_Out_` parametresi veya dönüş değeri ancak değeri döndürdü ek açıklama ile tutarsız.
