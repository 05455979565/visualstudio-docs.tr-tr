# <a name="c28196"></a>C28196
Uyarı C28196: Gereksinim karşılanmıyor. (İfade true olarak değerlendirilmez.)

 Bu uyarı, çözümlenmekte olan işlevin işlev prototipinin bir `_Out_` parametresi veya dönüş değeri üzerinde `__notnull`, `__null` veya `__drv_valueIs` olduğunu gösterir, ancak döndürülen değer bu ek açıklama ile tutarsız olur.
