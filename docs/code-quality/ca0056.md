# <a name="ca0056"></a>CA0056
Proje dosyası yanlış bir çözümleme aracı sürümü vardır.

 FxCopCmd.exe el ile bir hatalı kod çözümleme aracı sürümüne sahip bir proje dosyası ile çağrılırsa, bu uyarıyı oluşur.
