# <a name="c28214"></a>C28214
C28214 Uyarı: Öznitelik parametre adları p1... olmalıdır p9

 Bu uyarı, bir ek açıklama bildirimi oluşturulması, parametre adları p1... sınırlı olduğunu gösterir. p9. Bu uyarı, normal kullanımda asla gerçekleşmelidir.
