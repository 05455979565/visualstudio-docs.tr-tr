# <a name="c28103"></a>C28103
Uyarı C28103: sızıntı kaynağı

 Belirtilen nesne, serbest bırakılmamış bir kaynak içeriyor. Çağrılan bir işlev `__drv_acquiresResource` veya `__drv_acquiresResourceGlobal` ile açıklanmıştı ve bu uyarı, ek açıklamada adı geçen kaynağın serbest bırakılmadığını gösterir.

## <a name="example"></a>Örnek
 Aşağıdaki kod örneği bu uyarıyı oluşturur:

```
res = KeSaveFloatingPointState(buffer);
```

 Aşağıdaki kod örneği bu uyarıyı önler:

```
res = KeSaveFloatingPointState(buffer);
if (NT_SUCCESS(res))
{
    res = KeRestoreFloatingPointState(buffer);
}
```

 Bu uyarı yanlış pozitif olarak bildirilirse, en olası neden kaynağı serbest bırakma işlevinin `__drv_releasesResource` veya `__drv_releasesResourceGlobal` ile açıklanmamasına neden olur. Sistem işlevleri için sarmalayıcı işlevleri kullanıyorsanız, sarmalayıcı işlevlerinin sistem işlevlerinin kullandığı aynı ek açıklamaları kullanması gerektiğini unutmayın. Şu anda, model dosyasında birçok sistem işlevi açıklanmakta olduğundan ek açıklamalar başlık dosyalarında görünür değil.
