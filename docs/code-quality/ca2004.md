# <a name="ca2004-remove-calls-to-gckeepalive"></a>CA2004: GC.KeepAlive'a çağrıları kaldırın

|||
|-|-|
|TypeName|RemoveCallsToGCKeepAlive|
|CheckId|CA2004|
|Kategori|Microsoft. güvenilirliği|
|Son değişiklik|Kırılmamış|

## <a name="cause"></a>Sebep
Sınıflar `SafeHandle` kullanır, ancak yine de `GC.KeepAlive`çağrıları içerir.

## <a name="rule-description"></a>Kural açıklaması
`SafeHandle` kullanımı için dönüştürüyorsanız, tüm `GC.KeepAlive` (nesne) çağrılarını kaldırın. Bu durumda, bir sonlandırıcının olmadığı varsayıldığında, ancak bunların işletim sistemi tanıtıcısını tamamlaması için `SafeHandle` güvenen sınıfların `GC.KeepAlive`çağrısı olması gerekmez.  `GC.KeepAlive` çağrısında bırakma maliyeti performansa göre ölçülebilse de, `GC.KeepAlive` bir çağrının gerekli olduğunu veya artık mevcut olmayan bir yaşam süresi sorununu gidermek için yeterli olması gerektiğini belirten bir durum, kodun bakımını daha zor hale getirir.

## <a name="how-to-fix-violations"></a>İhlalleri çözme
`GC.KeepAlive`çağrılarını kaldırın.

## <a name="when-to-suppress-warnings"></a>Uyarıların ne zaman bastırılamıyor
Bu uyarıyı, yalnızca sınıfınıza `SafeHandle` kullanımı için teknik olarak doğru bir şekilde düzelmez.
