# <a name="ca0505"></a>CA0505
Superceding özelliği tanımlı olduğu için kullanım dışı bırakılan özellik gözardı edilecek.  
  
 Uyarı CA505 hem belirtilen kullanım ve neden Kod Analizi özellikleri projesinde değerler içeriyor veya yapı yapılandırma dosyası oluşturulur. Bu sorunu çözmek için kullanım dışı özelliği kaldırın.
