# <a name="c28310"></a>C28310
Uyarı 28310: @No__t-0function > \<parametresindeki < annotation_name > ek açıklaması SAL sürümüne sahip değil.

 Kaynak kod içinde kullanılan tüm satış ek açıklamalar SAL_name kullanımı tarafından uygulanan bir eklenti sürümü olması gerekir. Makro tanımında düzeltilmesi gerekiyor.

 Bu uyarı bildirimi yalnızca bir kez bildirilir. Daha fazla eski satış bildiriminin geri kalanını inceleyin.
