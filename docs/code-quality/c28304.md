# <a name="c28304"></a>C28304
C28304 Uyarı: için C++ başvuru-parametre < parametre_adý >, yanlış yerleştirilmiş `_Notref_` işleci uygulanan bulundu \<token >.  
  
 `_Notref_` İşleci, yalnızca C++ başvuru parametreleri içeren özel durumlar ve yalnızca sistem tarafından sağlanan makroları kullanılmalıdır. Bunu hemen tarafından uyulması gereken bir `_Deref_` işleci veya işlevsel bir ek açıklama.
