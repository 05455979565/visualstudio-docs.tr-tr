# <a name="c28283"></a>C28283
Uyarı C28283: Sembol için belirtilen boyut belirtimi henüz desteklenmiyor

 Uyarı boyut belirtimi "sentinel" ek açıklama sıfır dışında bir değer aldığını gösterir. Aslında, çağırana dizesi ikili sıfır dışında bir karakterle sonlandırılır söyleyin denedi.
