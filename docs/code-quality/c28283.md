# <a name="c28283"></a>C28283
C28283 Uyarı: simge için belirtilen boyut belirtimi henüz desteklenmiyor  
  
 Uyarı boyutu belirtimi "sentinel" ek açıklama sıfır dışında bir değer aldığını gösterir. Esas olarak, çağıran dize ikili sıfır dışında bir karakterle sonlandırılır söyleyin çalışıldı.
