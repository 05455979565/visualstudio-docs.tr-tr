# <a name="c28243"></a>C28243
Uyarı C28243: Daha fazla parametre üzerindeki işlev gerektirir asıl türün izin verdiğinden başvuru

 Ek açıklamada `__deref` işleçleri sayısı, parametre türü tarafından tanımlanan işaretçinin seviyelerinin sayısından daha fazla. Ek açıklamanın numarasını ya da başvurulan parametrenin işaretçi düzeylerini değiştirerek bu uyarıyı düzeltin.
