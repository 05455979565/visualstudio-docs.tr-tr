# <a name="c6271"></a>C6271
Uyarı C6271: \<function > için çok bağımsız değişken geçirildi: parametre \<sayı > biçim dizesi tarafından kullanılmıyor

 Bu uyarı, biçim dizesi tarafından belirtilenlerin ötesinde ek bağımsız değişkenlerin sağlandığını gösterir. Bu hatanın kendisi tarafından herhangi bir görünür etkiye sahip olmayacaktır, ancak programcının amacı koda yansıtılmadığını gösterir.

## <a name="example"></a>Örnek
 Aşağıdaki örnek kod bu uyarıyı oluşturur:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d",1,2);
}
```

 Bu uyarıyı düzeltmek için aşağıdaki örnek kodu kullanın:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d, %d",1,2);
}
```

 Aşağıdaki örnek kod, bu uyarıyı düzeltmek için `sprintf_s` olan güvenli dize işleme işlevini çağırır:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf_s( buff, 5,"%s %d", 1,2 ); //safe version
}
```
