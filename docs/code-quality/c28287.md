# <a name="c28287"></a>C28287
Uyarı C28287: İşlev için söz dizimi hatası, \_adresindeki\_() ek açıklama (tanınmayan parametre adı)

 @No__t-0 (`__drv_at`) ek açıklaması kullanıldığında ve parametre ifadesi geçerli bağlamda yorumlanamadığından, Kod Analizi Aracı bu uyarıyı raporlar. Bu, yanlış yazılmış bir parametre veya üye adı ya da "return" veya "This" anahtar sözcüklerinden oluşan hatalı bir işlem içerebilir.
