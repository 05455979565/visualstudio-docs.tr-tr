# <a name="c28285"></a>C28285
Uyarı C28285: İşlev için parametrede söz dizimi hatası

 Model dosyasında bir olası hata oluştuğunda bu uyarı kod analizi aracı bildirir. Birkaç kaynak dosyası hatalarını da bu tür hatalara neden olabilir.
