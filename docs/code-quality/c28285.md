# <a name="c28285"></a>C28285
Uyarı C28285: İşlev için parametrede söz dizimi hatası

 Model dosyasında olası bir hatayla karşılaşıldığında, Kod Analizi Aracı bu uyarıyı bildirir. Birkaç kaynak dosya hatası da bu tür hatalara neden olabilir.
