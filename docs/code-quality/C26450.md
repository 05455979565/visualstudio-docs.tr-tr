# <a name="arithmetic-overflow-operator-operation-causes-overflow-at-compile-time-use-a-wider-type-to-store-the-operands"></a>Aritmetik Taşma: '% işleci %' işlemi derleme zamanında taşmaya neden olur. İşlenenleri depolamak için daha geniş bir tür kullanın

 Bu uyarı, bir aritmetik işlem derleme zamanında kanıtlanabilir kayıplı olduğunu gösterir. İşlenen tüm derleme zamanı sabit olduğunda bu onaylanan.  Şu anda, sola kaydırma, çarpma, toplama ve çıkarma işlemlerinde böyle taşıyor denetleyin.

Not: C4307 Microsoft C++ derleyicisinde benzer bir denetimdir.

## <a name="example-1"></a>Örnek 1

```cpp
int multiply()
{
    const int a = INT_MAX;
    const int b = 2;
    int c = a * b; // C26450 reported here
    return c;
}
```
 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın.

```cpp
long long multiply()
{
    const int a = INT_MAX;
    const int b = 2;
    long long c = (long long)a * b; // OK
    return c;
}
```
