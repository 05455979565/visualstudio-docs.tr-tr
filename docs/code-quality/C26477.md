# <a name="c26477-usenullptrnotconstant"></a>C26477 USE_NULLPTR_NOT_CONSTANT

`nullptr` özel bir türe sahip `nullptr_t` , özel null işleme ile aşırı yüklemeler sağlar. Kullanarak `0` veya `NULL` yerine `nullptr` tür güvenliği ve kesinti atlar, `nullptr` sağlar.
