# <a name="c6235"></a>C6235
Uyarı C6235: (\<sıfır olmayan sabit > &#124; &#124; \<expression >) her zaman sıfır olmayan bir sabit değerdir

 Bu uyarı, bir test bağlamında oluşan mantıksal veya işlemin sol tarafında sıfır olmayan bir sabit değer olduğunu gösterir. Elde edilen ifade her zaman true olarak değerlendirildiğinden, mantıksal veya işlemin sağ tarafı değerlendirilmez. Bu, "kısa devre değerlendirmesi" olarak adlandırılır.

 Sıfır olmayan bir sabit değer dışında bir değeri, bit düzeyinde AND işlecinin (`&`) amaçlanan gibi olabileceğini önerir. Bu uyarı, sıfır olmayan sabit değer 1 olduğunda ortak deyim için oluşturulmaz, çünkü seçmeli olarak kod yollarının etkinleştirilmesi için kullanılır, ancak sıfır olmayan sabit değer 1 ' e, örneğin 1 + 0 olarak değerlendirilir.

## <a name="example"></a>Örnek
 Aşağıdaki kod bu uyarıyı oluşturur çünkü `INPUT_TYPE` 2 ' dir:

```
#define INPUT_TYPE 2
void f(int n)
{
   if(INPUT_TYPE || n) //warning 6235 issued
   {
      puts("Always gets here");
   }
   else
   {
      puts("Never gets here");
   }
}
```

 Aşağıdaki kod, bu uyarıyı düzeltmek için bit düzeyinde AND (`&`) işlecini kullanır:

```
#define INPUT_TYPE 2
void f(int n)
{
   if((INPUT_TYPE & n) == 2)
   {
      puts("bitwise-AND comparison true");
   }
   else
   {
      puts("bitwise-AND comparison false");
   }
}
```
