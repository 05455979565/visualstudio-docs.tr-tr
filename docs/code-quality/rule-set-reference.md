# <a name="code-analysis-rule-set-reference"></a>Kod çözümleme kural kümesi başvurusu

Visual Studio'daki yönetilen kod projeleri için Kod Analizi yapılandırdığınızda, yerleşik listesinden seçebilirsiniz *kural kümeleri*. Bu yerleşik kural kümeleri birini ya da kullanabilir ya da yapabilecekleriniz [bir kural kümesi özelleştirme](../code-quality/how-to-create-a-custom-rule-set.md) proje gereksinimlerinizi karşılayacak şekilde.

Bu bölümdeki konular, yerleşik kuralı açıklayın kümeleri ve kurallar (veya Uyarıları) içerir.
