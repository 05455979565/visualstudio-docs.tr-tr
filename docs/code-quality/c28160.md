# <a name="c28160"></a>C28160
C28160 Uyarı: Hata ek açıklaması

 Bu uyarı bildirilen zaman bir `__drv_error` ek açıklama karşılaşıldı. Bu ek açıklama, düzeltilmesi gereken ve kullanılabilir kodlama uygulamalarını bayrağı için kullanılan bir `__drv_when` belirli parametre birleşimleri belirtmek için ek açıklama.
