# <a name="c28160"></a>C28160
Uyarı C28160: Hata ek açıklaması

 Bu uyarı `__drv_error` ek açıklamasına rastlandığı zaman bildirilir. Bu ek açıklama, düzeltilmesi gereken kodlama uygulamalarını işaretlemek için kullanılır ve belirli parametre birleşimlerini göstermek için `__drv_when` ek açıklaması ile kullanılabilir.
