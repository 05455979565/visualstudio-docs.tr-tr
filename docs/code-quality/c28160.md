# <a name="c28160"></a>C28160
C28160 uyarısı: hata ek açıklaması

 Bu uyarı bildirilen olduğunda bir `__drv_error` ek açıklama karşılaştı. Bu ek açıklama düzeltilmesi gereken ve ile kullanılacak kodlama uygulamalarını bayrak için kullanılan bir `__drv_when` parametreleri belirli birleşimlerini belirtmek için ek açıklama.
