# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>Yönetilen Kodun Ölçüm Karmaşıklığı ve Bakımı
Modern yazılım uygulamaları artan karmaşıklık kodu güvenilir ve korunabilir hale getirme zorluk da artırır. Son yıllarda kod ölçümleri bilinen birçok yazılım ölçüler, geliştirilmiştir geliştiricileri kendi kodlarını olacak yeniden çalışma ihtimalinden veya artan sınama burada gerekiyor anlamak yardımcı olabilir.  
  
 Geliştiricileri kullanabilir [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] karmaşıklığı ve bunların yönetilen kod bakımı ölçmek kod ölçümleri verileri üretme için. Kod ölçümleri verileri çözümün tamamında veya tek bir proje için oluşturulabilir.  
  
 Bu bölümde kod ölçümleri oluşturmak ve nasıl kullanılacağını açıklar [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
