# <a name="c6274"></a>C6274
Uyarı C6274: karakter \<işlevine yapılan çağrıda gerekli olduğunda \<sayı > parametre olarak geçildi >

 Bu uyarı, biçim dizesinin bir karakterin gerektiğini (örneğin, bir `%c` veya `%C` belirtimi) belirtir, ancak bir float, String veya struct gibi tamsayı olmayan bir sayıdır. Bu hata, büyük olasılıkla hatalı çıktıya neden olabilir.

## <a name="example"></a>Örnek
 Aşağıdaki kod bu uyarıyı oluşturur:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 Aşağıdaki kod, bu uyarıyı düzeltmek için `sprintf_s` olan güvenli dize işleme işlevini kullanır:

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
