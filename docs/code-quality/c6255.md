# <a name="c6255"></a>C6255
Uyarı C6255: _alloca, yığın taşması özel durumunu yükselterek hata gösterir. Bunun yerine _malloca kullanmayı düşünün

 Bu uyarı, yerel özel durum işlemenin dışında `_alloca` ' a bir çağrının algılandığını gösterir. `_alloca`, hata durumunda bir yığın taşması özel durumu yükseltebildiğinden, her zaman bir özel durum işleyicisinin korunan aralığı içinden çağrılmalıdır. Mümkünse `_alloca` kullanmak yerine, `_alloca` ' nin daha güvenli bir sürümü olan `_malloca` kullanmayı düşünün.

## <a name="example"></a>Örnek
 @No__t-0 özel durum oluşturabileceği için aşağıdaki kod bu uyarıyı oluşturur:

```
#include <windows.h>

void f( )
{
  void *p = _alloca(10);
  // code ...
}
```

 Bu uyarıyı düzeltmek için, `_malloca` ' ı kullanın ve aşağıdaki kodda gösterildiği gibi özel durum işleyicisi ekleyin:

```
#include <windows.h>
#include <malloc.h>

void f( )
{
  void *p;
  int errcode;
  __try
  {
    p = _malloca(10);
    // code...
    _freea(p);
  }
  __except( (GetExceptionCode() == STATUS_STACK_OVERFLOW ) ?
              EXCEPTION_EXECUTE_HANDLER : EXCEPTION_CONTINUE_SEARCH )
  {
    errcode = _resetstkoflw();
    // code ...
  }
}
```
