# <a name="c28235"></a>C28235
C28235 Uyarı: işlevi açıklamanın tek başına olamaz

 Bu uyarı, açıklamanın bir ek açıklamanın bağlamda olarak bildirilmedi işlevi kullanmak için girişimde bulunuldu olduğunu gösterir. Bu, bir tek başına bağlamında ilkel işlemi (PrimOp) kullanarak içerir. Bu standart makroları ek açıklamalar için kullanılıyorsa olası olmamalıdır. Bu uyarı genel kullanımda görüntülenmesine beklenmiyor.
