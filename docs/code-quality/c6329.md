# <a name="c6329"></a>C6329
uyarı C6329: @No__t-0function > çağrısı için dönüş değeri \<sayı ile denetlenmemelidir >

 Program bir sayıyı `CreateFile` ' a çağrıdan dönüş değeri ile karşılaştırır. @No__t-0 başarılı olursa, nesnesine bir açık tanıtıcı döndürür. Başarısız olursa, `INVALID_HANDLE_VALUE` döndürür.

## <a name="example"></a>Örnek
 Bu kod, uyarıya neden olabilir:

```cpp
if (CreateFile() == NULL)
    {
        return;
    }
```

## <a name="example"></a>Örnek
 Bu kod hatayı düzeltir:

```cpp
if (CreateFile() == INVALID_HANDLE_VALUE)
    {
        return;
    }
```
