# <a name="c6995"></a>C6995
Uyarı C6995: XML günlük dosyası kaydedilemedi

 Bu uyarı, kod analizi aracının kod analizinin çıktısı olan hata günlüğünü oluşturamayacağını gösterir.

 Bu hata bir disk hatası olduğunu veya belirtilen dizinde dosya oluşturma izniniz olmadığını gösteriyor olabilir.
