# <a name="c28290"></a>C28290
Uyarı C28290: işlev için ek açıklama gerçek parametre sayısından daha fazla External içeriyor

 Kod Analizi aracına işlevi için ek açıklama gerçek parametre sayısından daha fazla External içeriyor. Bu uyarı bildirir.
