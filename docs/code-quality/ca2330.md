# <a name="ca2330-ensure-that-jsonserializer-has-a-secure-configuration-when-deserializing"></a>CA2330: JsonSerializer 'ın serisi kaldırılırken güvenli bir yapılandırmaya sahip olduğundan emin olun

|||
|-|-|
|TypeName|Donotuseınsecuredeserializerjsonnetwithoutciltçi|
|CheckId|CA2330|
|Kategori|Microsoft.Security|
|Son değişiklik|Kırılmamış|

## <a name="cause"></a>Sebep

Bu kural, bir [Newtonsoft. JSON. JsonSerializer](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_JsonSerializer.htm) örneği için aşağıdaki koşulların her ikisi de doğru olduğunda ateşlenir:
- [Typenamehandling](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializer_TypeNameHandling.htm) özelliği `None`dışında bir değerdir.
- [Serializationciltçi](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializer_SerializationBinder.htm) özelliği null.

[Jsonserializer](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_JsonSerializer.htm) örneği şu koşullarda kullanıldığında:
- Bir sınıf alanı veya özellik olarak başlatıldı.
- Bir seri kaldırma yöntemini çağırmak için.

Bu kural [CA2329](ca2329.md)ile benzerdir ancak analiz, serileştiricinin kesin olarak güvenli bir şekilde yapılandırılıp yapılandırılmadığını belirleyemez.

## <a name="rule-description"></a>Kural açıklaması

[!INCLUDE[insecure-deserializers-description](includes/insecure-deserializers-description-md.md)]

Bu kural, girişten belirtilen türlerin serisini kaldırmak için yapılandırılmış olabilecek [Newtonsoft. JSON. JsonSerializer](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_JsonSerializer.htm) örneklerini bulur, ancak [ Newtonsoft. JSON. Serialization. ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm). Girişten tamamen belirtilen türlerin serisini kaldırma iznini kaldırmak istiyorsanız, [CA2327](ca2327.md), [CA2328](ca2328.md), [CA2329](ca2329.md)ve CA2330 kurallarını devre dışı bırakın ve bunun yerine kural [CA2326](ca2326.md) etkinleştirin.

## <a name="how-to-fix-violations"></a>İhlalleri çözme

- Mümkünse, [Typenamehandling](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_TypeNameHandling.htm)'ın `None` değerini kullanın.
- Seri hale getirilen verileri prova yapın. Serileştirmeden sonra, serileştirilmiş verileri şifreli olarak imzalayın. Seri durumdan önce, şifreleme imzasını doğrulayın. Şifreleme anahtarını, önemli döndürmeler için açıklanmasını ve tasarıma karşı koruyun.
- Seri durumdan çıkarılan türleri kısıtla. Özel bir [Newtonsoft. JSON. Serialization. ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm)uygulayın. Json.NET ile seri durumdan çıkarmak için, [Newtonsoft. JSON. JsonSerializer. Serializationciltçi](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializer_SerializationBinder.htm) özelliğinde özel [ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm) belirtildiğinden emin olun. Geçersiz kılınan [Newtonsoft. JSON. Serialization. ISerializationBinder. BindToType](https://www.newtonsoft.com/json/help/html/M_Newtonsoft_Json_Serialization_ISerializationBinder_BindToType.htm) yönteminde tür beklenmiyorsa, `null` döndürün veya seri durumdan çıkarmayı durdurmak için bir özel durum oluşturun.

## <a name="when-to-suppress-warnings"></a>Uyarıların ne zaman bastırılamıyor

Şu durumlarda bu kuraldan bir uyarı bastırmak güvenlidir:
- Girişin güvenilir olduğunu bilirsiniz. Uygulamanızın güven sınırının ve veri akışlarının zaman içinde değişebilir olduğunu göz önünde bulundurun.
- [İhlallerin nasıl düzeltileceğini](#how-to-fix-violations)gösteren önlemlerle bir tane aldınız.
- [Typenamehandling](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializerSettings_TypeNameHandling.htm) özelliği `None`dışında bir değer olduğunda [Newtonsoft. JSON. Jsonserializer. serializationciltçi](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializer_SerializationBinder.htm) özelliğinin her zaman ayarlandığını bilirsiniz.

## <a name="pseudo-code-examples"></a>Sözde kod örnekleri

### <a name="violation"></a>Edildiği

```csharp
using System;
using Newtonsoft.Json;
using Newtonsoft.Json.Serialization;

public class BookRecordSerializationBinder : ISerializationBinder
{
    // To maintain backwards compatibility with serialized data before using an ISerializationBinder.
    private static readonly DefaultSerializationBinder Binder = new DefaultSerializationBinder();

    public void BindToName(Type serializedType, out string assemblyName, out string typeName)
    {
        Binder.BindToName(serializedType, out assemblyName, out typeName);
    }

    public Type BindToType(string assemblyName, string typeName)
    {
        // If the type isn't expected, then stop deserialization.
        if (typeName != "BookRecord" && typeName != "AisleLocation" && typeName != "WarehouseLocation")
        {
            return null;
        }

        return Binder.BindToType(assemblyName, typeName);
    }
}

public class BookRecord
{
    public string Title { get; set; }
    public object Location { get; set; }
}

public abstract class Location
{
    public string StoreId { get; set; }
}

public class AisleLocation : Location
{
    public char Aisle { get; set; }
    public byte Shelf { get; set; }
}

public class WarehouseLocation : Location
{
    public string Bay { get; set; }
    public byte Shelf { get; set; }
}

public static class Binders
{
    public static ISerializationBinder BookRecord = new BookRecordSerializationBinder();
}

public class ExampleClass
{
    public BookRecord DeserializeBookRecord(JsonReader reader)
    {
        JsonSerializer jsonSerializer = new JsonSerializer();
        jsonSerializer.TypeNameHandling = TypeNameHandling.Auto;
        jsonSerializer.SerializationBinder = Binders.BookRecord;
        return jsonSerializer.Deserialize<BookRecord>(reader);    // CA2330 -- SerializationBinder might be null
    }
}
```

### <a name="solution"></a>Çözüm

```csharp
using System;
using Newtonsoft.Json;
using Newtonsoft.Json.Serialization;

public class BookRecordSerializationBinder : ISerializationBinder
{
    // To maintain backwards compatibility with serialized data before using an ISerializationBinder.
    private static readonly DefaultSerializationBinder Binder = new DefaultSerializationBinder();

    public void BindToName(Type serializedType, out string assemblyName, out string typeName)
    {
        Binder.BindToName(serializedType, out assemblyName, out typeName);
    }

    public Type BindToType(string assemblyName, string typeName)
    {
        // If the type isn't expected, then stop deserialization.
        if (typeName != "BookRecord" && typeName != "AisleLocation" && typeName != "WarehouseLocation")
        {
            return null;
        }

        return Binder.BindToType(assemblyName, typeName);
    }
}

public class BookRecord
{
    public string Title { get; set; }
    public object Location { get; set; }
}

public abstract class Location
{
    public string StoreId { get; set; }
}

public class AisleLocation : Location
{
    public char Aisle { get; set; }
    public byte Shelf { get; set; }
}

public class WarehouseLocation : Location
{
    public string Bay { get; set; }
    public byte Shelf { get; set; }

    public static ISerializationBinder BookRecord = new BookRecordSerializationBinder();

public class ExampleClass
{
    public BookRecord DeserializeBookRecord(JsonReader reader)
    {
        JsonSerializer jsonSerializer = new JsonSerializer();
        jsonSerializer.TypeNameHandling = TypeNameHandling.Auto;

        // Ensure that SerializationBinder is assigned non-null before deserializing
        jsonSerializer.SerializationBinder = Binders.BookRecord ?? throw new Exception("Expected non-null");

        return jsonSerializer.Deserialize<BookRecord>(reader);
    }
```
