# <a name="c28301"></a>C28301
Uyarı C28301: \<function > ilk bildirimi için ek açıklama yok. \<note1 > bkz. \<filename > (\<line >). \<note2 >

 Bu uyarı, belirli bir işlevin ilk bildiriminde ek açıklamalar bulunamadığında raporlanır.
