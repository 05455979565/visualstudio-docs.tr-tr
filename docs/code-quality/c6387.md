# <a name="c6387"></a>C6387
uyarı C6387: \<argument > \<Değer > olabilir: Bu, \<function adı >: Lines: x, y işlevi için belirtime bağlı kalmıyor

 Bu uyarı, bir açıklamalı işlev parametresine beklenmeyen bir değer geçiriliyorsa tetiklenir. Örneğin, `_In_` ek açıklaması ile işaretlenen bir parametreye potansiyel olarak null bir değer geçirilerek bu uyarı oluşturulur.

## <a name="example"></a>Örnek
 @No__t-0 ' a null bir parametre geçirildiğinden aşağıdaki kod bu uyarıyı oluşturur:

```cpp

#include <sal.h>

_Post_ _Null_ char * g();

void f(_In_ char *pch);

void main()
{
    char *pCh = g();
    f(pCh); // Warning C6387
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:

```cpp

#include <sal.h>

_Post_ _Notnull_ char * g();

void f(_In_ char *pch);

void main()
{
    char *pCh = g();
    f(pCh);
}
```
