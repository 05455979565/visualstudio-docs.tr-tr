# <a name="ca1506-avoid-excessive-class-coupling"></a>CA1506: Aşırı sınıf bağlantısından kaçın

|||
|-|-|
|TypeName|AvoidExcessiveClassCoupling|
|CheckId|CA1506|
|Kategori|Microsoft.Maintainability|
|Son dakika değişikliği|Yeni|

## <a name="cause"></a>Nedeni

Bir tür veya yöntem diğer birçok türile birleştiğinde. Derleyici oluşturulan türleri bu metrik dışında tutulur.

## <a name="rule-description"></a>Kural açıklaması

Bu kural türü veya yöntemini içeren benzersiz türde başvuru sayısı belirlenerek eşlenmesiyle sınıfı ölçer.

Sınıf bağlantı yüksek derecede tür ve yöntemler korumak zor olabilir. Düşük bağlantı ve yüksek uyum sergileyen tip ve yöntemlere sahip olmak iyi bir uygulamadır.

## <a name="how-to-fix-violations"></a>İhlalleri düzeltme

Bu ihlali düzeltmek için, birleştiği tür sayısını azaltmak için türü veya yöntemi yeniden tasarlamayı deneyin.

## <a name="when-to-suppress-warnings"></a>Uyarıları ne zaman bastırır

Türü veya yöntemi diğer türlere bağımlılıkları çok sayıda rağmen sürdürülebilir olarak kabul edildiğinde bu uyarıyı hariç tut.
