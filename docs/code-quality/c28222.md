# <a name="c28222"></a>C28222
28222 Uyarı: \_Evet\_, \_Hayır\_, veya \_belki\_ ek açıklaması için beklenen

 Bu uyarı, bir ek açıklama parametresi simgelerden birini olması beklenir gösterir `_Yes_`, `_No_`, veya `_Maybe_`, ve başka bir simge ile karşılaşıldı. Bu, genellikle bir yanlış kodlanmış ek açıklama makrosu gösterir.
