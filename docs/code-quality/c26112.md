# <a name="c26112"></a>C26112
Uyarı C26112: Çağıran \<func > çağrılmadan önce herhangi bir kilit tutamaz.

 @No__t-0 ek açıklaması, çağıran işlevi çağırdığında herhangi bir kilit tutmaması gereken bir önkoşul uygular. Uyarı C26112, bir işlev başka bir işlevi çağırmadan önce tüm kilitleri serbest bırakamazsa verilir.

## <a name="example"></a>Örnek
 Aşağıdaki örnek, `_Requires_no_locks_held_` önkoşulunu kilitli bölüm içindeki `NoLocksAllowed` ' e çağırarak ihlal ettiğinden, uyarı C26112 oluşturur.

```

typedef struct _DATA
{
    CRITICAL_SECTION cs;
} DATA;

_Requires_no_locks_held_

void NoLocksAllowed(DATA* p)
{
     // Lock sensitive operations here
}

void LocksHeldFunction(DATA* p)
{
    EnterCriticalSection(&p->cs);
    NoLocksAllowed(p); // Warning C26112
    LeaveCriticalSection(&p->cs);
}
```
