# <a name="c28244"></a>C28244
Uyarı C28244: İşlevi için ek açıklama bir ayrılamaz parametre/dış ek açıklamaya sahip

 Bu şu anda standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
