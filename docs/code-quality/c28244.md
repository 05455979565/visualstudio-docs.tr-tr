# <a name="c28244"></a>C28244
Uyarı C28244: İşlevi için ek açıklamanın ayrıştırılamayan bir parametre/dış ek açıklaması vardır

 Bu, şu anda ek açıklamalar için standart makrolar kullanılıyorsa mümkün değildir; Bu uyarının tipik kullanımda görülebilmesi beklenmez.
