# <a name="c28232"></a>C28232
C28232 Uyarı: _Pre\_, _post'u\_, veya _Deref\_ hiçbir ek açıklama uygulanmadı  
  
 Bu uyarı bildiren bir `_Pre_`, `_Post_`, veya `_Deref_` sonraki işlev ek açıklama olmayan bir ek açıklamanın ifade işleci görünür; değiştiricisi yoksayıldı, ancak bu yanlış kodlanmış açıklamanın gösterir.
