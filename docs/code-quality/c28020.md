# <a name="c28020"></a>C28020
C28020 Uyarı: İfade \<expr > Bu çağrıda doğru değil

 Bu uyarı bildirilen zaman \_Satisfies\_ listelenen ifade true değil. Bu genellikle hatalı bir parametre belirtir.

 Bu işlev bildiriminde meydana gelirse, ek açıklamalar mümkün olmayan bir durumu gösterir.
