# <a name="c28020"></a>C28020
Uyarı C28020: @No__t-0expr > ifadesi bu çağrıda doğru değil

 Bu uyarı, listelenen \_no__t-1 ifadesi doğru olmadığında bildirilir. Bu, genelde yanlış bir parametre olduğunu gösterir.

 Bu bir işlev bildiriminde oluşursa, ek açıklamalar imkansız bir koşul gösterir.
