# <a name="c6306"></a>C6306
Uyarı C6306: \<function > için yanlış çağrı: bağımsız değişken olarak bir va_list kabul eden \<function > kullanmayı düşünün

 Bu uyarı yanlış bir işlev çağrısını gösterir. @No__t_0 ailesi, bağımsız değişkenlerin değişken bir listesini alan çeşitli işlevler içerir; Ancak, bu işlevler `va_list` bir bağımsız değişkenle çağrılamaz. Bu tür çağrılar için kullanılabilen karşılık gelen bir `vprintf` işlev ailesi vardır. Yanlış Print işlevinin çağrılması hatalı çıktıya neden olur.

## <a name="example"></a>Örnek
 Aşağıdaki kod bu uyarıyı oluşturur:

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   printf("%s", v); // warning 6306
   va_end(v);
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   vprintf_s("%d",v);
   va_end(v);
}
```
