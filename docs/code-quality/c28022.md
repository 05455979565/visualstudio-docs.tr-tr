# <a name="c28022"></a>C28022
C28022 Uyarı: işlevi sınıfları \<classlist1 > fonksiyon işlevi sınıfları eşleşmiyor \<classlist2 > bunu tanımlamak için kullanılan typedef üzerinde.  
  
 Ek açıklamalar hata olduğunda bu uyarı bildirilir. Typedef ve işlev `_Function_class_` ek açıklamalar, ancak bunlar eşleşmiyor. Bunların her ikisi de kullanılıyorsa eşleşmesi gerekir.
