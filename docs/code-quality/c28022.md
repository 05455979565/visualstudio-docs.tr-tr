# <a name="c28022"></a>C28022
Uyarı C28022: İşlev sınıfları \<classlist1 > işlev sınıfları bu işlev üzerinde eşleşmiyor \<classlist2 > tanımlamak için kullanılan typedef üzerindeki.

 Ek açıklamalarda bir hata olduğunda bu uyarı bildirilir. Hem typedef hem de işlev `_Function_class_` ek açıklamaları, ancak bunlar eşleşmiyor. Her ikisi de kullanılıyorsa bunların eşleşmesi gerekir.
