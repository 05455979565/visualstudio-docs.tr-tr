# <a name="ca0057"></a>CA0057
Geçerli hedefleri ve kurallar kümesine ihlalleri eşlenemez.

 FxCopCmd.exe el ile bir proje dosyası ile çağrılırsa, bu hata oluşur veya bir / alınan geçerli hedefleri ve kurallar kümesine eşlenemez ihlalleri sahip XML rapor.

 Yalnızca analiz yapıldığında oluşur komut satırı aracı sürümüyle.
