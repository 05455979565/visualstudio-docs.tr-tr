# <a name="ca0053"></a>CA0053
Kural derlemesi yüklenemedi.

 Bu kural derlemesi tarafından yansıma yüklenemedi gösterir. Kullanıcının doğru şekilde oluşturulmuş değil özel kurallar varsa bu oluşabilir veya dosya sistemi erişimi reddedildi gibi analiz aracı diğer nedenlerle derlemesi yüklenemiyor.
