# <a name="ca2305-do-not-use-insecure-deserializer-losformatter"></a>CA2305: Güvenli olmayan seri kaldırıcı LosFormatter kullanmayın

|||
|-|-|
|TypeName|DoNotUseInsecureDeserializerLosFormatter|
|CheckId|CA2305|
|Category|Microsoft.Security|
|Son değişiklik|Kırılmamış|

## <a name="cause"></a>Nedeni

@No__t-0 seri kaldırma yöntemi çağrıldı veya başvuruluyor.

## <a name="rule-description"></a>Kural açıklaması

Bu kural <xref:System.Web.UI.LosFormatter?displayProperty=nameWithType> seri kaldırma yöntemi çağrıları veya başvuruları bulur.

## <a name="pseudo-code-examples"></a>Sözde kod örnekleri

### <a name="violation"></a>Edildiği

```csharp
using System.IO;
using System.Web.UI;

public class ExampleClass
{
    public object MyDeserialize(byte[] bytes)
    {
        LosFormatter formatter = new LosFormatter();
        return formatter.Deserialize(new MemoryStream(bytes));
    }
}
```

```vb
Imports System.IO
Imports System.Web.UI

Public Class ExampleClass
    Public Function MyDeserialize(bytes As Byte()) As Object
        Dim formatter As LosFormatter = New LosFormatter()
        Return formatter.Deserialize(New MemoryStream(bytes))
    End Function
End Class
```
