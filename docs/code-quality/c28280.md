# <a name="c28280"></a>C28280
Uyarı C28280: sembol Için, eşleşen bir ' begin' olmadan bir ' End ' bulundu

 Ek açıklama dili, bir BEGIN ve End (`{` ve `}`) yapısını destekler ve eşleştirmeden dengesiz bir yer vardır. Makrolar kullanılıyorsa bu durum kaçınılabilir.
