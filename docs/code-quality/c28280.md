# <a name="c28280"></a>C28280
C28280 Uyarı: Sembol için bir 'end' eşleşmeyen bir 'begin' bulundu

 Ek Açıklama Dili başlangıç ve bitiş tarihi destekler (`{` ve `}` C'de) yapısı ve eşlemesine değil dengesiz. Bu durum, makroların kullanılması durumunda önlenebilir.
