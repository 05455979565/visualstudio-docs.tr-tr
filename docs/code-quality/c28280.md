# <a name="c28280"></a>C28280
Uyarı C28280: sembol Için, eşleşen bir ' begin' olmadan bir ' End ' bulundu

 Ek açıklama dili, başlangıç ve bitiş (`{` ve `}` C) yapısını destekler ve eşleştirme, dengesiz bir şekilde dengelenmemiş. Makrolar kullanılıyorsa bu durum kaçınılabilir.
