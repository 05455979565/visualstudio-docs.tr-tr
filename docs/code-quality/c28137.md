# <a name="c28137"></a>C28137
Uyarı C28137: değişken bağımsız değişkeni bunun yerine bir (sabit değer) sabit olmalıdır

 Bu uyarı, bir işlev çağrısında gerekli bir (değişmez değer) sabit değeri eksik olduğunda bildirilir. İşlevine yönelik belgelere başvurun.

## <a name="example"></a>Örnek
 Örneğin, [ExAcquireResourceExclusiveLite](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-exacquireresourceexclusivelite) yordamı `Wait` PARAMETRESI için true veya false değerini gerektirir. Aşağıdaki örnek kod bu uyarıyı oluşturur:

```
ExAcquireResourceExclusiveLite(Resource, Wait);
```

 Aşağıdaki kod örneği bu uyarıyı önler:

```
ExAcquireResourceExclusiveLite(Resource, TRUE);
```
