# <a name="c28291"></a>C28291
Uyarı C28291: Deref düzey 0 ' da null/NotNull değeri, param \<Number > işlevinde \<x > işlevi için anlamsız bir küçüktür.

 Bir başvuru düzeyi sıfır parametresinin Post koşulu null/null olmayan bir özelliğe sahip olacak şekilde belirtildiğinde, Kod Analizi Aracı bu uyarıyı bildirir. Bu hata, başvuru düzeyindeki sıfır değeri değiştiremediği için oluşur.
