# <a name="c28291"></a>C28291
C28291 Uyarı: POST null/notnull deref düzeyi 0 ise, işlev için anlamsız \<x > adresindeki param \<sayı >

 Kod Analizi aracı, posta koşulu başvuru bir düzey sıfır parametresi null/boş olmayan özellik belirtildiğinde bu uyarı bildirir. Bu hata düzeyi sıfır değiştirilemez bir değerde başvuru oluşur.
