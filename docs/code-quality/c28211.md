# <a name="c28211"></a>C28211
Uyarı C28211: SAL_context için beklenen statik bağlam adı

 Bu uyarı, `_Static_context_` ek açıklamasına ait işlenenin araç tanımlı bir bağlamın adı olması gerektiğini gösterir. Bu uyarı, normal kullanımda asla gerçekleşmemelidir.
