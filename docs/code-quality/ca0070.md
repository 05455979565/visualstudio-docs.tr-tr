# <a name="ca0070"></a>CA0070
Bu hata, Kod Analizi bütünleştirilmiş analiz eder ve izin öznitelikleri karşılaşırsa ve aşağıdaki koşulların en az biri doğru olduğunda oluşur:

-   Kod çözümleme özniteliklerini içeren derleme bulunamıyor.

-   Kod çözümleme derleme yükleme izni yok.

-   Kod Analizi tarafından yüklenen derleme özniteliği içermiyor.

 Çoğu durumda, kod çözümleme araçları en son sürümüne sahip olduğunuzdan emin yaparak sorun çözülebilir.
