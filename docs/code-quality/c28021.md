# <a name="c28021"></a>C28021
Uyarı C28021: @No__t-0param > \<anno > ile açıklanmakta olan parametre bir işaretçi olmalıdır

 Bu uyarı, açıklama eklenen nesne bir işaretçi türü olmadığında bildirilir. Bu ek açıklama `void` veya integral türleriyle kullanılamaz.
