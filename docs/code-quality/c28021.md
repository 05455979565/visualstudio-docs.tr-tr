# <a name="c28021"></a>C28021
C28021 Uyarı: Parametre \<param > ile açıklanan \<Milattan > bir işaretçi olmalıdır

 Ek açıklamalı olan nesneye bir işaretçi türü olmadığında bu uyarı bildirilir. Bu ek açıklaması kullanılamaz `void` veya tam sayı türleri.
