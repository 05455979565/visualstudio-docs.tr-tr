# <a name="ca0504"></a>CA0504
Kural dizini yok sayıldı

 Bir dizin belirtilen olduğunda uyarı CA504 oluşturulan **CodeAnalysisRuleDirectories** bir proje veya derleme yapılandırma dosyası bir özellik bulunamadı. Bu sorunu çözmek için geçerli bir yol girin.
