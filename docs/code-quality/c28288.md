# <a name="c28288"></a>C28288
C28288 Uyarı: işlev, söz dizimi hatası sürü_müne\_() ek açıklama (geçersiz parametre adı)  
  
 Kod çözümleme aracı olduğunda bu uyarı raporları `SAL_at` (`__drv_at`) ek açıklama kullanılır ve parametre ifadesi geçerli bağlamda yorumlanamaz. Bu, yanlış yazılmış bir parametre veya üye adı veya bir yazım hatası "return" veya "Bu" anahtar sözcükleri kullanarak içerebilir.
