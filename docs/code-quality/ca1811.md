# <a name="ca1811-avoid-uncalled-private-code"></a>CA1811: Çağrılmayan özel kodlardan kaçının

|||
|-|-|
|TypeName|AvoidUncalledPrivateCode|
|CheckId|CA1811|
|Kategori|Microsoft.Performance|
|Son dakika değişikliği|Kırılmaz|

## <a name="cause"></a>Nedeni
Özel veya dahili (derleme düzeyinde) üyenin derlemede arayanları yoktur, ortak dil çalışma süresi tarafından çağrılmaz ve bir temsilci tarafından çağrılmaz. Aşağıdaki üyeler bu kural tarafından denetlenmez:

- Açık arabirim üyeleri.

- Statik yapıcılar.

- Serileştirme yapıcılar.

- Veya <xref:System.Runtime.InteropServices.ComUnregisterFunctionAttribute?displayProperty=fullName>. <xref:System.Runtime.InteropServices.ComRegisterFunctionAttribute?displayProperty=fullName> ile işaretlenmiş yöntemler

- Geçersiz kılan üyeler.

## <a name="rule-description"></a>Kural açıklaması
Bu kural, şu anda kural mantığı tarafından tanımlanmamış giriş noktaları oluşursa yanlış pozitif leri bildirebilir. Ayrıca, derleyici çağrılamayan kodu bir derlemeye yarayabilir.

## <a name="how-to-fix-violations"></a>İhlalleri düzeltme
Bu kuralın ihlalini düzeltmek için, çağrılamayan kodu kaldırın veya onu çağıran kod ekleyin.

## <a name="when-to-suppress-warnings"></a>Uyarıları ne zaman bastırır
Bu kuraldan bir uyarıyı bastırmak güvenlidir. Bunu yapmak için [kod çözümlemesi uyarılarını bastır'a](../code-quality/in-source-suppression-overview.md)bakın.
