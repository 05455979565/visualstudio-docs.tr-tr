# <a name="ca0067"></a>CA0067
Dizin bulunamadı.

 Değerini **/dizin** FxCopCmd komut satırı seçeneği bulunamadı.

 Bu uyarı kullanıyorsanız gerçekleşebilir **CodeAnalysisDependentAssemblyPaths** özelliği [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] ek başvuru bütünleştirilmiş kod arama yolları ve bu yollar birini belirtmek için yok.

- Uyarı CA0067 ile görünürse [CA0058](ca0058.md) hataları veya [CA0060](ca0060.md) uyarı, genellikle diğer hataları çözümleme CA0067 giderir.

- Uyarı CA0067 hata veya uyarı görünürse, çünkü dizin gerekmeyen genellikle uyarıyı yoksayabilirsiniz.
