# <a name="ca0067"></a>CA0067
Dizin bulunamadı.

 Değeri **/directory** FxCopCmd komut satırı seçeneği bulunamadı.

 Bu uyarı, kullanmakta olduğunuz oluşabilir **CodeAnalysisDependentAssemblyPaths** özelliği [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] ek başvuru derleme arama yolları ve bu yollar birini belirtmek için mevcut değil.

-   Uyarı CA0067 ile görünüyorsa [CA0058](ca0058.md) hataları veya [CA0060](ca0060.md) uyarılar, genellikle diğer hataları çözümleme CA0067 giderir.

-   Uyarı CA0067 diğer hatalar veya uyarılar görünüyorsa, dizin gerekmeyen çünkü genellikle uyarıyı yoksayabilirsiniz.
