# <a name="c6302"></a>C6302
Uyarı C6302: biçim dizesi uyuşmazlığı: \<işlevine yapılan çağrıda geniş karakter dizesi gerektiğinde \<sayı > parametresi olarak geçirilen karakter dizesi >

 Bu uyarı, biçim dizesinin geniş bir karakter dizesinin gerekli olduğunu belirtir. Ancak, bir karakter dizesi geçiriliyor. Bu hata, bir kilitlenme veya bir formun bozulmasına neden olabilir.

## <a name="example"></a>Örnek
 Aşağıdaki örnek kod bu uyarıyı oluşturur çünkü bir karakter dizesi `wprintf` işlevine geçirilir:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%s", buff);
}
```

 Aşağıdaki örnek kod, `wprintf` işleviyle tek baytlı bir karakter dizesi belirtmek için `%hs` kullanır:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%hs", buff);
}
```

 Aşağıdaki örnek kod, bu uyarıyı düzeltmek için `wprintf_s` olan güvenli dize işleme işlevini kullanır:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf_s(L"%hs", buff);
}
```
