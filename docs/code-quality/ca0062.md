# <a name="ca0062"></a>CA0062
Kural '*RuleId*'Kural kümesinde başvurulan'*RuleSetName*' bulunamadı.  
  
 Belirtilen kural bulunamadı Bu hata gösterir.  
  
 Ruleset dosyayı el ile düzenlendi bu hata genellikle oluşur. Bu hatayı önlemek için [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] kural kümesi Düzenleyici kurallarını belirtin. Daha fazla bilgi için bkz: [özel kural kümeleri oluşturma](../code-quality/creating-custom-code-analysis-rule-sets.md).  
  
 Bu sorunu çözmek için ruleset dosyasında belirtilen tüm onay kimlikleri geçerli olduğundan emin olun. Standart olmayan kural dosyaları kullanıyorsanız, marka ve uygun kuralı yolları ipucu ruleset dosyanızda belirtilir.
