# <a name="ca0062"></a>CA0062

Kural '*RuleId*'Kural kümesinde başvurulan'*RuleSetName*' bulunamadı.

Bu hata, belirtilen kural bulunamadı gösterir.

Bu hata genellikle bir kural kümesi dosyasına el ile düzenlendi kaynaklanır. Bu hatadan kaçınmak için Visual Studio kullanabilirsiniz [kural kümesi düzenleyici](../code-quality/working-in-the-code-analysis-rule-set-editor.md) Kod Analizi kuralları yapılandırmak için.

Bu sorunu çözmek için emin tüm onay belirtilen kimlikleri, *.ruleset* dosya geçerlidir. bir standart bir kural kümesi dosyası kullanıyorsanız, ilgili kural ipucu yolları dosyasında belirtilir emin olun.
