# <a name="c28224"></a>C28224
C28224 Uyarı: Ek açıklama parametreleri gerektirir

 Bu uyarı, adlı ek açıklama, parametre olmadan kullanılır ve en az bir parametresi gerekiyor gösterir. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
