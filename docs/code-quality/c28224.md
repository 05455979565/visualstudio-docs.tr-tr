# <a name="c28224"></a>C28224
Uyarı C28224: Ek açıklama parametreleri gerektirir

 Bu uyarı, adlandırılmış ek açıklamanın parametre olmadan kullanıldığını ve en az bir parametre gerektiğini gösterir. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının normal kullanımda görülebilmesi beklenmez.
