# <a name="c28273"></a>C28273
Uyarı C28273: İşlev için ipuçları işlev bildirimiyle tutarsız

 Bu uyarı, ek açıklamalar, analiz ediliyor. kodu değil, bir hata gösterir. Bir işlev tanımı üzerinde görünen ek açıklamalar bildiriminde görünen değerlerle tutarsız. İki ek açıklamalar, eşleştirilecek çözülmesi gerekir.
