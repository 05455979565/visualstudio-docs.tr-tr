# <a name="c28204"></a>C28204
Uyarı C28204: \<function >: Bu aşırı yükten yalnızca biri ve \<filename > (\<line >), \<paramname > için açıklama eklendi: her ikisi de birlikte açıklanmamalıdır.

 Bu uyarı, ek açıklamalarda bir hata olduğunda bildirilir.
