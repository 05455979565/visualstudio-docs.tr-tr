# <a name="c28236"></a>C28236
C28236 Uyarı: ek açıklamanın bir ifadede kullanılamaz  
  
 Bu uyarı bir ifade bağlamda ek açıklama olarak bildirilen bir işlevi kullanmak için girişimde bulunuldu olduğunu gösterir. Bu standart makroları ek açıklamalar için kullanılıyorsa olası olmamalıdır; Bu uyarı genel kullanımda görüntülenmesine beklenmiyor.
