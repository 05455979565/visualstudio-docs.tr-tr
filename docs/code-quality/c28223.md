# <a name="c28223"></a>C28223
Uyarı C28223: Ek açıklaması için beklenen belirteç/tanımlayıcı bulunamadı parametresi

 Bu uyarı, bir ek açıklama parametre tanımlayıcı olması beklenir ve başka bir simge karşılaşıldı gösterir. Bu, genellikle bir yanlış kodlanmış ek açıklama makrosu gösterir. C veya C++ anahtar sözcükleri bu konumda kullanımını bu hataya neden olur.
