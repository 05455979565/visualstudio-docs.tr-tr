# <a name="c26111"></a>C26111
Uyarı C26111: Çağıran, \<func > işlevini çağırmadan önce \<lock > kilit serbest bırakamıyor.

 @No__t-0 ek açıklaması, belirtilen kilit için kilit sayısının, işlev çağrıldığında sıfırdan büyük olamaz. Bir işlev başka bir işlevi çağırmadan önce kilidi serbest bırakamazsa, uyarı C26111 verilir.

## <a name="example"></a>Örnek
 Aşağıdaki örnek, `_Requires_lock_not_held_` önkoşulunu kilitli bölüm içindeki `DoNotLock` ' e çağırarak ihlal ettiğinden, uyarı C26111 oluşturur.

```

typedef struct _DATA
{
    CRITICAL_SECTION cs;
    int d;
} DATA;

_Requires_lock_not_held_(p->cs)

void DoNotLock(DATA* p)
{
    EnterCriticalSection(&p->cs);
    p->d = 0;
    LeaveCriticalSection(&p->cs);
}

void LockedFunction(DATA* p)
{
    EnterCriticalSection(&p->cs);
    DoNotLock(p); // Warning C26111
    LeaveCriticalSection(&p->cs);
}
```
