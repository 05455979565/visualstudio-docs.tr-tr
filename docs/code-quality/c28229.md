# <a name="c28229"></a>C28229
C28229 Uyarı: ek açıklama parametreleri desteklemiyor  
  
 Bu uyarı, ek açıklamanın parametresiz bildirilmişse açıklamanın bir parametre kullanıldı gösterir. Bu standart makroları ek açıklamalar için kullanılıyorsa olası olmamalıdır; Bu uyarı genel kullanımda görüntülenmesine beklenmiyor.
