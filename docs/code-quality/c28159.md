# <a name="c28159"></a>C28159
Uyarı C28159: bunun yerine başka bir işlev kullanmayı düşünün.

 Bu uyarı, sürücüler için bildirilir ve bu, sürücünün çağrıldığı işleve anlamsal olarak eşdeğer bir tercih edilen işlev çağrısı kullanmanızı önerir. Bu, genel bir uyarı iletisidir; `__drv_preferredFunction` ek açıklaması kullanıldı (büyük olasılıkla bir `__drv_when` () ek açıklamasına sahip olabilir) ve hatalı bir kodlama uygulamasını Bayrakla işaretleyebilirsiniz.

## <a name="example"></a>Örnek
 Aşağıdaki kod örneği bu uyarıyı oluşturur:

```
char buff[MAX_PATH];

OemToChar(buff, input);

    // if strlen(input) > MAX_PATH
....// leads to buffer overrun
```

 Aşağıdaki kod örneği bu uyarıyı önler:

```
char buff[MAX_PATH];

OemToCharBuff(buff, input, MAX_PATH);
```
