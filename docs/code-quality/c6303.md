# <a name="c6303"></a>C6303
Uyarı C6303: biçim dizesi uyuşmazlığı: \<işlevine yapılan çağrıda karakter dizesi gerekliyse \<sayı > parametresi olarak geçirilen geniş karakter dizesi >

 Bu uyarı, biçim dizesinin bir karakter dizesinin gerekli olduğunu belirtir. Ancak, geniş bir karakter dizesi geçirilir. Bu hata, büyük olasılıkla kilitlenme veya bazı form bozulması oluşmasına neden olabilir.

## <a name="example"></a>Örnek
 Aşağıdaki örnek kod bu uyarıyı oluşturur:

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%s", buff);
}
```

 Bu uyarıyı düzeltmek için aşağıdaki örnek kodda gösterildiği gibi `%ls` kullanın:

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%ls", buff);
}
```

 Aşağıdaki örnek kod, bu uyarıyı düzeltmek için `printf_s` olan güvenli dize işleme işlevini kullanır:

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf_s("%ls",buff);
}
```
