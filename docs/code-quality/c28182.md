# <a name="c28182"></a>C28182
Uyarı C28182: NULL işaretçisinin başvurusunu kaldırma.

 **Ek bilgi**: *\<pointer1 >* *\<pointer2 >* DID *\<note* ile aynı null değeri içerir >

 Kod Analizi Aracı, işaretçinin NULL olduğunu doğruladığında bu uyarıyı bildirir. Hatanın izlemenin başlarında gerçekleşebileceği onaylanan olmayan örnekler varsa, kod çözümleme aracı ilk örneğin satır numarasını uyarı iletisine ekleyerek kodu tüm örnekleri ele almak üzere değiştirebilirsiniz.

 *\<pointer2 >* POTANSIYEL olarak null olacak şekilde onaylandı. *\<pointer1 >* *pointer2* ile aynı değeri içeriyor ve başvurusu kaldırılmış. Bu işaretçiler kodun çok farklı konumlarında olabileceğinden, her ikisi de, kod analizi aracının neden bu uyarıyı raporladığını belirleyebilmeniz için raporlanır.

 Koşulun onaylanmamış bir önceki örneği varsa *\<note >* şu metinle değiştirilmiştir: "Bu, gerçekleşebileceği önceki bir konum için bkz. satır *\<number >* ."

## <a name="example"></a>Örnek
 Aşağıdaki örnek, kod analizi aracının bu uyarı iletisini oluşturmasına neden olabilecek kodu gösterir. Bu örnekte, Kod Analizi Aracı `pNodeFree` ' ın `if` ifadesinde NULL olduğunu ve `if` ' nin gövdesinde kod yolunun alındığını belirler. Ancak, `nBlockSize` potansiyel olarak sıfır olduğundan, `for` ifadesinin gövdesi yürütülmez ve `pNodeFree` değiştirilmemiş olarak kalır. `pNodeFree` `pNode` ' e atanır ve NULL bir başvuru oluşması durumunda `pNode` kullanılır.

```
typedef struct xlist {
    struct xlist *pNext;
    struct xlist *pPrev;
} list;

list *pNodeFree;
list *masterList;
int nBlockSize;

void fun()
{
    if (pNodeFree == 0)
    {
        list *pNode = masterList;

        for (int i = nBlockSize-1; i >= 0; i--, pNode--)
        {
            pNode->pNext = pNodeFree;
            pNodeFree = pNode;
        }
    }

    list* pNode = pNodeFree;
    pNode->pPrev = 0;
}
```

 Kod çözümleme aracı aşağıdaki uyarıyı raporlar:

```
:\sample\testfile.cpp(24) : warning C28182: Dereferencing NULL pointer. 'pNode' contains the same NULL value as 'pNodeFree' did.: Lines: 12, 14, 16, 23, 24
```

## <a name="example"></a>Örnek
 Daha önceki örneği düzeltmeniz için bir yol, bir NULL başvurusunun Averted olması için silinmeden önce sıfır için @no__t denetiyisidir. Aşağıdaki kod bu düzeltmeyi göstermektedir.

```
typedef struct xlist {
    struct xlist *pNext;
    struct xlist *pPrev;
} list;

list *pNodeFree;
list *masterList;
int nBlockSize;

void fun()
{
    if (pNodeFree == 0)
    {
        list *pNode = masterList;

        for (int i = nBlockSize-1; i >= 0; i--, pNode--)
        {
            pNode->pNext = pNodeFree;
            pNodeFree = pNode;
        }
    }

    list* pNode = pNodeFree;
    if(pNode != 0)
      pNode->pPrev = 0;
}
```
