# <a name="c28210"></a>C28210
28210 Uyarı: Ek açıklamalar için `_On_failure_` bağlam açık bağlam içinde olmamalıdır

 Ek açıklamalar `_On_failure_` açıkça veya örtük olarak bildirilmelidir `__post` işlev döndürdükten sonra uygulanacak bağlamı, diğer bir deyişle,.  Kullanım `_drv_out` bu emin olmak için.
