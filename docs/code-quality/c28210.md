Uyarı 28210: ek açıklamalarını `_On_failure_` bağlamı açık öncesi bağlamda olmamalıdır  
  
 Ek açıklamalar `_On_failure_` açıkça veya örtük olarak bildirilmelidir `__post` işlevi döndükten sonra uygulanacak bağlamı, diğer bir deyişle,.  Kullanım `_drv_out` bu emin olmak için.
