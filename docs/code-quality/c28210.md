# <a name="c28210"></a>C28210
uyarı 28210: @No__t-0 bağlamının ek açıklamaları açık bir ön bağlamda olmamalıdır

 @No__t-0 `__post` bağlamında açık ya da örtük olarak belirtilmelidir, diğer bir deyişle, işlev dönüşünden sonra uygulanabilir olmalıdır.  Bunu sağlamak için `_drv_out` kullanın.
