# <a name="c6250"></a>C6250
Uyarı C6250: MEM_RELEASE bayrağı olmadan \<function > VirtualFree çağrısı, belleği serbest bırakabilir, ancak adres tanımlayıcılarını (VADs) içeremez. Adres alanı sızıntılarına ilişkin sonuçlar

 Bu uyarı, `MEM_RELEASE` bayrağına sahip olmayan `VirtualFree` ' a bir çağrının yalnızca sayfaların geri ve serbest bırakıldığını gösterir. Sayfaları silmek ve serbest bırakmak için, `VirtualFree` ' e yapılan çağrıda `MEM_RELEASE` bayrağını kullanın. Bölgedeki herhangi bir sayfa taahhütdeyse, işlev ilk olarak onları serbest bırakır ve sonra bırakır. Bu işlemden sonra sayfalar boş durumdadır. Bu bayrağı belirtirseniz, `dwSize` sıfır olmalıdır ve `lpAddress`, bölge ayrıldıysa `VirtualAlloc` işlevi tarafından döndürülen temel adresi göstermelidir. Bu koşullardan biri karşılanmazsa işlev başarısız olur.

 Kodunuz daha sonra `MEM_RELEASE` bayrağıyla `VirtualFree` ' ı çağırarak adres alanını serbest bırakıyorsanız bu uyarıyı yoksayabilirsiniz.

 Daha fazla bilgi için bkz. [VirtualAlloc](http://go.microsoft.com/fwlink/?LinkId=181201) ve [VirtualFree](http://go.microsoft.com/fwlink/?LinkId=181202).

 VirtualAlloc ve VirtualFree kullanımı, bellek sızıntıları ve özel durumlar bakımından birçok ekde sahiptir. Bu tür sızıntıları ve özel durum sorunlarını tamamen önlemek için, C++ standart şablon KITAPLıĞı (STL) tarafından sunulan mekanizmaları kullanın. Bunlar şunlardır [shared_ptr](/cpp/standard-library/shared-ptr-class), [unique_ptr](/cpp/standard-library/unique-ptr-class)ve [Vector](/cpp/standard-library/vector). Daha fazla bilgi için bkz. [akıllı işaretçiler](/cpp/cpp/smart-pointers-modern-cpp) ve [ C++ standart kitaplık](/cpp/standard-library/cpp-standard-library-reference).

## <a name="example"></a>Örnek
 Aşağıdaki örnek kod bu uyarıyı oluşturur:

```
#include <windows.h>
#include <stdio.h>
#define PAGELIMIT 80

DWORD dwPages = 0;  // count of pages
DWORD dwPageSize;   // page size

VOID f( )
{
  LPVOID lpvBase;            // base address of the test memory
  SYSTEM_INFO sSysInfo;      // system information

  GetSystemInfo(&sSysInfo);
  dwPageSize = sSysInfo.dwPageSize;

  // Reserve pages in the process's virtual address space
  lpvBase = VirtualAlloc(
                         NULL,                // system selects address
                         PAGELIMIT*dwPageSize, // size of allocation
                         MEM_RESERVE,
                         PAGE_NOACCESS);
  //
  // code to access memory
  // ...

  if (lpvBase != NULL)
  {
    if (VirtualFree( lpvBase, 0, MEM_DECOMMIT )) // decommit pages
      {
        puts ("MEM_DECOMMIT Succeeded");
      }
      else
      {
       puts("MEM_DECOMMIT failed");
      }
  }
  else
  {
    puts("lpvBase == NULL");
  }
}
```
