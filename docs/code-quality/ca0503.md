# <a name="ca0503"></a>CA0503
Özellik kullanım dışı bırakılmıştır. Öncekinin yerini alan özelliğini kullanın

 Kullanım dışı kod çözümleme özelliği derleme veya proje yapılandırma dosyasında bir değer içeriyor uyarı CA0503 oluşturulur. Bu uyarıyı çözmek için bu yerine geçiyor belirtilen özelliğini kullanın.
