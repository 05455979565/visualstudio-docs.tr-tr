# <a name="c28230"></a>C28230
Uyarı C28230: Parametresinin türü üyesi yok.

 Bu uyarı, erişmeyi denediği bir ek açıklama bağımsız değişkeni gösterir. bir `struct`, `class`, veya `union` ve adlandırılmış üyesinin yok.
