# <a name="c6102"></a>C6102
Uyarı C6102: satır \<konum > başarısız işlev çağrısından \<değişken > kullanılıyor.

 Bu uyarı, başarısız olan bir önceki işlev çağrısında `_Out_` parametresi olarak işaretlendiğinden, bir değişken ayarlanmamışsa [C6001](../code-quality/c6001.md) yerine bu uyarı raporlanır.

 Sorun önceden çağrılan işleve tamamen açıklama eklenmediğinden olabilir. @No__t-0, `_Outptr_result_nullonfailure_` (COM kodu için `_COM_Outptr_`) veya ilgili ek açıklama gerekebilir.
