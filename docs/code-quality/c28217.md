# <a name="c28217"></a>C28217
C28217 Uyarı: İşlev için ek açıklama için parametre sayısı dosyada bulunanla eşleşmiyor.

 Ek açıklamalar geçerli satırdaki ve ileti satırda tutarsız. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
