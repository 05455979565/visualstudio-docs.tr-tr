# <a name="c6282"></a>C6282
Uyarı C6282: Hatalı işleç: Boolean bağlamında sabit atama. Bunun yerine ' = = ' kullanmayı düşünün

 Bu uyarı, bir değişkene bir sabit atamanın bir test bağlamında algılandığını gösterir. Bir sabitin bir test bağlamındaki değişkenine atanması neredeyse her zaman hatalı olur. @No__t-0 ' ı `==` ile değiştirin ya da bu uyarıyı çözmek için test bağlamından atamayı kaldırın.

## <a name="example"></a>Örnek
 Aşağıdaki kod bu uyarıyı oluşturur:

```
void f( int i )
{
   while (i = 5)
   {
   // code
   }
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:

```
void f( int i )
{
   while (i == 5)
   {
   // code
   }
}
```
