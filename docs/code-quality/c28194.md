# <a name="c28194"></a>C28194
C28194 Uyarı: işlevi değişkenin değeri yumuşatma bildirilir ve bunu yapmadan çıkıldı

 Bu uyarı çözümlenmekte işlevi için işlev prototipi olduğunu gösterir bir `__drv_isAliased` olacak olduğunu belirten ek açıklama *diğer* belirtilen bağımsız değişken (diğer bir deyişle, varlığını sürdürmesini şekilde değeri atayın işlevinden döndüren). Ancak, işlevi diğer adı olmayan bağımsız değişkeni tarafından ek açıklama belirtilen yolda yapar. Diğer bir değişken değeri genel veri yapısına kaydetmeniz çoğu işlev.
