# <a name="c28279"></a>C28279
Uyarı C28279: sembol Için, eşleşen ' End ' olmadan bir ' begin' bulundu

 Ek açıklama dili, başlangıç ve bitiş (`{` ve `}`) yapısını destekler ve eşleştirme, dengesiz bir şekilde dengelenmemiş. Makrolar kullanılıyorsa bu durum kaçınılabilir.
