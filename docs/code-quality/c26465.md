# <a name="c26465-no_const_cast_unnecessary"></a>C26465 NO_CONST_CAST_UNNECESSARY

`const`dönüştürmek için `const_cast` kullanmayın. `const_cast` gerekli değildir; sabitlik veya Vola, bu dönüşüm tarafından kaldırılmıyor. Bkz [ C++ . temel yönergeler türü. 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-constcast).
