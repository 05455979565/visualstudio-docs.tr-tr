# <a name="arithmetic-overflow-left-shift-count-is-negative-or-greater-than-or-equal-to-the-operand-size-which-is-undefined-behavior"></a>Aritmetik Taşma: Sola kaydırma sayısı negatif veya tanımsız bir davranıştır işlenen boyutuna eşit veya daha büyük

 Kaydırma sayısı negatif veya işlenen olma bit sayısına eşit veya daha büyük kaydırılacağı uzaklık, tanımsız davranışlara neden olur, bu uyarı gösterir.
Not: C4293 Microsoft C++ derleyicisinde benzer bir denetimdir.

## <a name="example"></a>Örnek

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return (hi << 32) | lo; // C26252 here
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return ((unsigned __int64)hi << 32) | lo; // OK
}
```
