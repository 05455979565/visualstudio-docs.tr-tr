# <a name="warning-c26253-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>Uyarı C26253: Aritmetik Taşma: Sola kaydırma, negatif işaretli bir sayı tanımsız bir davranıştır

Bu uyarı, kötü bir fikir olan ve uygulama tanımlı davranış tetikleyen bir negatif işaretli tamsayı değeri, kaydırma biz sol gösterir.

## <a name="example"></a>Örnek

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
