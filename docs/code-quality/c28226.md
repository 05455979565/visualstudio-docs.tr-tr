# <a name="c28226"></a>C28226
Uyarı C28226: Ek açıklama da aynı zamanda (geçerli bildirim içinde) PrimOp olamaz

 Bu uyarı, adlandırılmış ek açıklamanın bir PrimOp olarak bildirildiği ve daha önce normal bir ek açıklama olarak bildirildiği anlamına gelir. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının normal kullanımda görülebilmesi beklenmez.
