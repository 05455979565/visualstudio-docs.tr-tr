# <a name="c28226"></a>C28226
Uyarı C28226: Ek açıklama ayrıca PrimOp (bildiriminde geçerli) olamaz

 Bu uyarı adlı ek açıklama PrimOp bildirilmiş ve aynı zamanda normal bir ek açıklama daha önce bildirildi gösterir. Bu standart makroları ek açıklamalar için kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görüntülenmesine beklenmiyor.
