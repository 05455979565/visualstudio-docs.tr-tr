# <a name="c6064"></a>C6064
Uyarı C6064: dönüştürme tanımlayıcısına karşılık gelen > \<function için tamsayı bağımsız değişkeni eksik \<sayı >

 Bu uyarı, bir biçim dizesiyle eşleşmesi için yeterli sayıda bağımsız değişken sağlandığını ve eksik bağımsız değişkenlerden birinin bir tamsayı olduğunu gösterir. Bu hata yanlış çıktıya neden olabilir.

## <a name="example"></a>Örnek
 Aşağıdaki kod bu uyarıyı oluşturur çünkü `sprintf` çağrısında yanlış sayıda bağımsız değişken kullanıldı ve eksik bağımsız değişken bir tamsayı:

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string="Hello, World";

  sprintf(buff,"%s %d", string);
}
```

 Bu uyarıyı düzeltmek için, aşağıdaki kodda gösterildiği gibi eksik bağımsız değişkenleri belirtin:

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string = "Hello, World";

  sprintf(buff,"%s %d",string, strlen(string));
}
```

 Aşağıdaki kod, bu uyarıyı düzeltmek için `sprintf_s` olan güvenli dize işleme işlevini kullanır:

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string="Hello World";

  sprintf_s(buff,sizeof(buff),"%s %d", string, strlen(string));
}
```
