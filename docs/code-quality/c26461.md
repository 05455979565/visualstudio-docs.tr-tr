# <a name="c26461-use_const_pointer_arguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:
  '% Function% ' işlevi için '% Argument% ' işaretçi bağımsız değişkeni, `const` işaretçisi olarak işaretlenebilir. Bkz: [C++ temel yönergeleri con.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
