# <a name="c6272"></a>C6272
Uyarı C6272: \<işlevine yapılan çağrıda float gerekli olduğunda \<sayı > bağımsız değişken olarak zaman uyumsuz olarak geçirildi >

 Bu uyarı, biçim dizesinin bir float gerektiğini, örneğin `printf,` için bir `%f` veya `%g` belirtimi olduğunu ancak bir tamsayı veya dize gibi bir kayan noktalı olmayan bir değer olduğunu belirtir. Bu hata, büyük olasılıkla yanlış çıkışa neden olabilir; Ancak, bazı durumlarda kilitlenmeyle sonuçlanabilir.

## <a name="example"></a>Örnek
 Aşağıdaki kod bu uyarıyı oluşturur:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodda gösterildiği gibi `%f` belirtimi yerine `%i` kullanın:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 Aşağıdaki kod, bu uyarıyı düzeltmek için `sprintf_s` olan güvenli dize işleme işlevini kullanır:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
