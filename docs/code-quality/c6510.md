# <a name="c6510"></a>C6510
Uyarı C6510: geçersiz ek açıklama: ' Nullsonlandırılmış ' özelliği yalnızca öğeleri integral veya işaretçi türü olan arabellekler üzerinde kullanılabilir: Işlev ' \<işlev > ' \<parametre >.

 Bu uyarı, **Nullsonlandırılmış** özelliğin (' `_z` ' içinde biten) yanlış bir kullanımını gösterir. Bu özellik türünü yalnızca işaretçi veya dizi türleri üzerinde kullanabilirsiniz.

## <a name="example"></a>Örnek
 Aşağıdaki kod bu uyarıyı oluşturur:

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 Bu uyarıyı düzeltmek için aşağıdaki kodu kullanın:

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
