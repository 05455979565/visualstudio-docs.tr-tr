# <a name="how-to-view-managed-code-defects"></a>Nasıl yapılır: Yönetilen Kod Kusurlarını Görüntüleme
Kod Analizi hataları, uyarıları ve iletileri Kod Analizi penceresinde raporlanır. Kod çözümleme proje ya da çözüm çalıştırdığınızda Kod Analizi penceresi otomatik olarak görüntülenir.  
  
 Kod Analizi penceresinde aşağıdakileri yapabilirsiniz:  
  
-   **Uyarı başlığını seçin.**  
  
     Uyarı hakkında ek bilgi Kod Analizi penceresinde görünür. Uyarıyı tetikleyen kaynak dosyasındaki satır görüntülenir ve kaynak kod düzenleyicisinde vurgulanır.  
  
-   **Uyarı kimliği seçin.** Uyarı kimlikleri öneki olarak **CA** ve dört basamaklı bir sayı içerir. Örneğin, CA1234.  
  
     MSDN Kitaplığı Yardım konusu bir uyarı görüntülenir. Yardım konusu uyarı nedeni hakkında ayrıntılı bilgiler içerir ve nasıl uyarı sorununu oluşur ve çözülmüş gösteren örnekleri içerir.
