# <a name="c26473-noidentitycast"></a>C26473 NO_IDENTITY_CAST
"Kaynak türü ve hedef türünün aynı olduğu işaretçi türleri arasında dönüştürme yapmayın."

**C++ temel yönergeleri**: Type.1: Yayınları kaçının

Bu kural, gereksiz veya şüpheli atamaları kaldırmak için yardımcı olur. Kuşkusuz, tür kendisine dönüştürüldüğünde, tür dönüştürme etkisiz henüz türleri gelecek değiştirirseniz cast kullanılır olgu Zarif tasarım sorunu veya regresyon potansiyeli gösterebilir. Her zaman mümkün olduğu kadar az yayınları kullanmak daha güvenlidir.

## <a name="remarks"></a>Açıklamalar
- Bu kural için statik uygulanır ve reinterpret çevirir ve işaretçi türleri denetler.
  ## <a name="example"></a>Örnek
  tehlikeli düzeyde bir genel arama

```cpp
gsl::span<server> servers_;

template<class T>
server* resolve_server(T tag) noexcept {
    auto p = reinterpret_cast<server*>(tag); // C26473, also 26490 NO_REINTERPRET_CAST
    return p >= &(*servers_.begin()) && p < &(*servers_.end()) ? p : nullptr;
}

void promote(server *s, int index) noexcept {
    auto s0 = resolve_server(s);
    auto s1 = resolve_server(index);
   if (s0 && s1)
        std::swap(s0, s1);
}
```

## <a name="example"></a>Örnek
tehlikeli düzeyde bir genel arama - yeniden işledi

```cpp
// ...
server* resolve_server(server *p) noexcept {
    return p >= &(*servers_.begin()) && p < &(*servers_.end()) ? p : nullptr;
}

server* resolve_server(ptrdiff_t i) noexcept {
    return !servers_.empty() && i >= 0 && i < servers_.size() ? &servers_[i] : nullptr;
}
// ...
```
