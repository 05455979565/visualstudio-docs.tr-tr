> [!NOTE]
> Otomatik UI tabanlı işlevsel test için kodlanmış UI Testi amortismana hazırdır. Visual Studio 2019, Kodlu UI Testi'nin sunulabildiği son sürümüdür. Masaüstü ve UWP uygulamalarını test etmek için Web uygulamalarını ve [Appium'u WinAppDriver ile](https://github.com/Microsoft/WinAppDriver) test etmek için [Selenium](https://docs.seleniumhq.org/) kullanmanızı öneririz. NUnit test çerçevesini kullanarak iOS ve Android uygulamalarını test etmek için [Xamarin.UITest'i](/appcenter/test-cloud/uitest/) göz önünde bulundurun.
