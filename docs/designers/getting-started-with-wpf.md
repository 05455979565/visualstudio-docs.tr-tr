# <a name="get-started-with-wpf"></a>WPF kullanmaya başlama

Windows Presentation Foundation (WPF), masaüstü istemci uygulamaları oluşturan bir kullanıcı bir kullanıcı birimi çerçevesidir. WPF geliştirme platformu, uygulama modeli, kaynaklar, denetimler, grafikler, düzen, veri bağlama, belgeler ve güvenlik dahil olmak üzere çok çeşitli uygulama geliştirme özelliklerini destekler. Çerçeve .NET'in bir parçasıdır, bu nedenle daha önce ASP.NET veya Windows Formlar kullanarak .NET ile uygulamalar oluşturmuşsanız, programlama deneyimi tanıdık olmalıdır. WPF, uygulama programlama için bildirimsel bir model sağlamak için Genişletilebilir Uygulama Biçimlendirme Dili'ni (XAML) kullanır.

Bu bölümde WPF'yi tanıtmanıza ve işe başlamanıza yardımcı olan konular vardır.
