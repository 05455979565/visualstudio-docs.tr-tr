1. VS içinde GitHub havuzların çalışmak için Visual Studio için Github uzantısını yüklemeniz gerekir. Bunu yapmak için Visual Studio yükleyicisi, belirleyin **Değiştir**seçip **bileşenleri tek tek** sekmesi. Ekranı aşağı kaydırarak **kod Araçları** bölümünde, select **Visual Studio için GitHub uzantısı**ve seçin **Değiştir**.

    ![VS Yükleyicisi'nde GitHub uzantısı seçme](../media/installation-github-extension.png)
