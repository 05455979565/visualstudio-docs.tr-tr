# <a name="one-or-more-selected-database-objects-return-a-schema-that-does-not-match-the-schema-of-the-target-class"></a>Seçili veritabanı nesnelerinin biri veya daha fazlası, hedef sınıfın şemasıyla eşleşmeyen bir şema döndürüyor

Bir veya daha fazla seçili veritabanı nesnelerini hedef sınıfın şemasıyla eşleşmeyen bir şema döndürüyor. Hiçbir şey tasarımcıya eklendi.

Veritabanı nesneleri mevcut varlık sınıfları sürüklediğinizde, veritabanı nesnesi tarafından döndürülen veriler hedef varlık sınıfın şemasıyla eşleşmesi gerekir. Doğru veritabanı nesnesi seçili olduğunu ve doğru varlık sınıfı hedeflenmiş olduğunu doğrulayın.

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Tıklayın **Tamam** iletişim kutusunu kapatın.

2. Hedef sınıfın şemasıyla eşleşen veri döndüren bir veritabanı nesnesi seçin (veritabanı nesnesi bırakılıyor üzerine içinde sınıf **O/R Tasarımcısı**).
