# <a name="rtvs-current-version"></a>RTVS geçerli sürümü

Visual **Studio'da Yardım** > **Hakkında'ya** giderek ve Visual Studio girişi için R **Tools'a** kaydırarak sürümünüzü kontrol edebilirsiniz:

![Visual Studio'da RTVS sürümünün incelenmesi (VS2017)](media/current-version.png)

Visual Studio 2017'de RTVS sürümünüzü güncellemek için Visual Studio yükleyicisine gidin ve **Güncelleştir** düğmesini seçin.

Visual Studio 2015 için en son yükleyici [aka.ms/rtvs-current](https://rtvs.blob.core.windows.net/download/RTVS_2017-12-18.1.exe) 'dan (doğrudan indirin) edinilebilir.

Bkz. Visual Studio ve Windows gereksinimleri için [Yükleme.](installing-r-tools-for-visual-studio.md)

Microsoft R Open ve Microsoft Machine Learning Server (eski adıyla Microsoft R Server) hakkındaki güncelleştirmeler için bkz: [Microsoft R ürünleri.](https://azure.microsoft.com/?ocid=cloudplat_hp)
