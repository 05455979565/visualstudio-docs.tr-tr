Bazı araçlar diğer paylaşılan [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] diller ve C# Derleyici gibi bazı Visual C# için benzersiz. Bu konu, en önemli Visual C# araçlarına bağlantılar sağlar.
