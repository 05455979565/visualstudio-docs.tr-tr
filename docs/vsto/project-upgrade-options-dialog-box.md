# <a name="project-upgrade-options-dialog-box"></a>Proje yükseltme, Seçenekler iletişim kutusu
  Kullanım **projeyi yükseltmesine** sayfasında Visual Studio otomatik olarak geliştirme bilgisayarında yüklü Office sürümünü hedeflemek için VSTO eklentisi projelerine değişiklikler olup olmadığını belirlemek için.  
  
 Bu sayfaya erişmeye **Araçları** menüsünde tıklatın **seçenekleri**. Genişletin **Office Araçları** klasörünü ve ardından **projeyi yükseltmesine**.  
  
## <a name="uielement-list"></a>UIElement listesi  
 **Her zaman yüklü Office sürümü yükseltme**  
 Bu seçenek devre dışıdır [!INCLUDE[vs_dev12](../vsto/includes/vs-dev12-md.md)] Office projelerinde çünkü [!INCLUDE[vs_dev12](../vsto/includes/vs-dev12-md.md)] yalnızca hedefleyebilir [!INCLUDE[office14_long](../vsto/includes/office14-long-md.md)] veya Microsoft [!INCLUDE[Office_15_short](../vsto/includes/office-15-short-md.md)] uygulamalar. Daha fazla bilgi için [yükseltme ve Office çözümlerini geçirme](../vsto/upgrading-and-migrating-office-solutions.md).
