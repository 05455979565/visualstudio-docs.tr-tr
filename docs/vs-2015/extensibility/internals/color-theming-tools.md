# <a name="color-theming-tools"></a>Renk Teması Oluşturma Araçları
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [renk teması oluşturma araçları](https://docs.microsoft.com/visualstudio/extensibility/internals/color-theming-tools).  
  
VS SDK'yi oluşturup renk temaları yönetmenize yardımcı olmak için çeşitli araçlar sağlar.  
  
-   [VSIX renk Düzenleyicisi](../../extensibility/internals/vsix-color-editor.md) aracı oluşturabilir ve Visual Studio için özel renkler düzenleyebilirsiniz.  
  
-   [VSIX renk derleyicisi](../../extensibility/internals/vsix-color-compiler.md) aracı .pkgdef Uzantınız için mevcut Visual Studio temasını .xml dosyalarına dönüştürür.
