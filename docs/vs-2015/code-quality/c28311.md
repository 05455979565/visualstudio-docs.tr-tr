# <a name="c28311"></a>C28311

Uyarı 28311: üzerindeki < annotation_name > ek açıklama \<işlevi > \<parametresi > SAL'ın eskimiş bir sürümüdür.  
  
 Ek açıklama eski bir sürümdür ve eşdeğer güncelleştirilmelidir [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md).
