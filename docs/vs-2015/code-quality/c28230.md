# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28230: parametre türünün üyesi yok.  
  
 Bu uyarı, bir ek açıklamanın bağımsız değişkeninin `struct`, `class`veya `union` erişmeyi denediğini ve adlandırılmış üyenin mevcut olmadığını gösterir.
