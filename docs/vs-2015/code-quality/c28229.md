# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28229: Ek açıklama parametreleri desteklemiyor  
  
 Bu uyarı, ek açıklama parametre olmadan bildirildiğinde açıklamanın bir parametreyle kullanıldığını gösterir. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
