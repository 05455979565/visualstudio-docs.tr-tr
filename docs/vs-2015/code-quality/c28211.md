# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28211: statik bağlam adı SAL_context için bekleniyor  
  
 Bu uyarıyı gösterir işleneni `_Static_context_` ek açıklama aracı tarafından tanımlanan bir bağlam adı olması gerekir. Bu uyarı, normal kullanımda asla gerçekleşmelidir.
