# <a name="ca0055"></a>CA0055

Dosya yüklenemedi:\<yolu >.

Kod analiz aracı analiz için belirtilen hedef dosyasını yükleyemedi. Bu dosya bulunamadı veya dosyaya erişim reddedildi ortaya çıkar. Bu genellikle, kullanıcı Visual Studio'da giriş bütünleştirilmiş kodun özelliği geçersiz kılmayı seçtiğinizi ya da fxcopcmd.exe el ile çağrılan oluşur.
