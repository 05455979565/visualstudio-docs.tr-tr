# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [C28225](https://docs.microsoft.com/visualstudio/code-quality/c28225).  
  
Uyarı C28225: açıklamada doğru sayıda gerekli parametre bulunamadı  
  
 Bu uyarı, adlandırılmış ek açıklamanın yanlış sayıda parametre kullanıldığını gösterir. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
