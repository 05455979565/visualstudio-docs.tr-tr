# <a name="ca0056"></a>CA0056

Bir proje dosyası, analiz aracı yanlış bir sürümü vardır.

FxCopCmd.exe el ile bir hatalı kod analizi aracı sürümü olan bir proje dosyası ile çağrılması durumunda bu uyarı oluşur.
