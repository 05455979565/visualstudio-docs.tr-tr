# <a name="ca0505"></a>CA0505

Geçersiz kılınan Özellik tanımlandığından, kullanım dışı bırakılan özellik yok sayılacak.

Uyarı CA505, hem belirtilen kullanım dışı hem de bir proje ya da derleme yapılandırma dosyasında değerler içerdiğinde oluşturulur. Bu sorunu çözmek için kullanımdan kaldırılan özelliği kaldırın.
