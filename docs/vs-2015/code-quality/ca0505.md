# <a name="ca0505"></a>CA0505

Superceding özelliği tanımlanmadığından kullanım dışı bırakılan özellik gözardı edilecek.

Uyarı CA505 hem belirtilen kullanım dışı ve kod çözümleme özelliklerini neden bir projedeki değerleri içeren ya da derleme yapılandırma dosyası oluşturulur. Bu sorunu çözmek için kullanım dışı özelliği kaldırın.
