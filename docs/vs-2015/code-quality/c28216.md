# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28216: \_Check_return\_ ek açıklaması yalnızca belirli işlev parametresi için son koşullar için geçerlidir.  
  
 `_Check_return_` ek açıklaması `__post`dışındaki bir bağlamda uygulandı; `__post` (veya `__drv_out`) değiştiricisi gerekebilir veya hatalı yerleştirilmiş olabilir.
