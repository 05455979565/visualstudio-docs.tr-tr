# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28216: _Check_return\_ ek açıklaması belirli işlev parametresi için koşullara yalnızca geçerlidir.  
  
 `_Check_return_` Ek açıklama uygulanmış bir bağlamda dışında `__post`; gereksinim duymayabilirsiniz bir `__post` (veya `__drv_out`) değiştiricisi veya yanlış yerleştirilebilir.
