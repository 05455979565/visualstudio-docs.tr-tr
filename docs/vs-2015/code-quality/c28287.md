# <a name="c28287"></a>C28287
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28287: işlev Için \_\_() ek açıklamasında söz dizimi hatası (Tanınmayan parametre adı)  
  
 `SAL_at` (`__drv_at`) ek açıklaması kullanıldığında ve parametre ifadesi geçerli bağlamda yorumlanamadığından, Kod Analizi Aracı bu uyarıyı raporlar. Bu, yanlış yazılmış bir parametre veya üye adı ya da "return" veya "This" anahtar sözcüklerinden oluşan hatalı bir işlem içerebilir.
