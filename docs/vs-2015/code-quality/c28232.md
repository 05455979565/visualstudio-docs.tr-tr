# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28232: _Pre\_, _post'u\_, veya _Deref\_ herhangi bir ek açıklamaya uygulanmadı  
  
 Bu uyarı bildiren bir `_Pre_`, `_Post_`, veya `_Deref_` işleci bir sonraki işlev Açıklama olmadan bir ek açıklama ifadesi görüntülenir; değiştiricisi yoksayıldı, ancak bu açıklamanın yanlış kodlanmış gösterir.
