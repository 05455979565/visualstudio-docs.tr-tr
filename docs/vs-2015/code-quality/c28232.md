# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28232: \_ön\_, \_Post\_veya \_deref\_ hiçbir açıklamaya uygulanmadı  
  
 Bu uyarı, bir `_Pre_`, `_Post_`veya `_Deref_` işlecinin sonraki işlevsel ek açıklama olmadan bir ek açıklama ifadesinde göründüğünü belirtir; değiştirici yoksayıldı, ancak bu, yanlış kodlanmış bir ek açıklamayı gösterir.
