# <a name="ca0070"></a>CA0070

Kod Analizi bütünleştirilmiş analiz eder ve izin öznitelikleri ile karşılaşırsa ve en az biri aşağıdaki koşullardan biri doğru olduğunda bu hata oluşur:

-   Kod Analizi öznitelikleri içeren derleme bulunamıyor.

-   Kod Analizi bütünleştirilmiş kod yükleme izni yok.

-   Kod Analizi tarafından yüklenen derleme özniteliği içermiyor.

Çoğu durumda, kod Çözümleme Araçları'nın güncel sürümlerine sahip olduğunuzdan emin olmak tarafından sorun çözülebilir.
