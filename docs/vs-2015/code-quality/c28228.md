# <a name="c28228"></a>C28228
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28228: Ek açıklama parametresi: ek açıklamalarda tür kullanılamaz  
  
 Bu uyarı, desteklenmeyen türde bir parametre olduğunu gösterir. Ek açıklamalar, yalnızca sınırlı sayıda türleri parametre olarak kullanabilirsiniz. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
