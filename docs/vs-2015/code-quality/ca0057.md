# <a name="ca0057"></a>CA0057

İhlaller geçerli hedef ve kural kümesiyle eşlenemez.

Bu hata, FxCopCmd. exe ' nin geçerli hedef ve kural kümesiyle eşleştirilemez ihlalleri olan bir proje dosyasıyla veya/Imported XML raporuyla el ile çağrılması durumunda meydana gelir.

Yalnızca aracın komut satırı sürümü kullanılarak analiz yapıldığında gerçekleşir.
