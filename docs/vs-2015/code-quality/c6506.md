# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C6506: Geçersiz ek açıklama: \<name > özelliği yalnızca işaretçi veya dizi türü değerleri üzerinde kullanılabilir  
  
 Bu uyarı, bir özellik işaretçi veya dizi türleri farklı bir türünde kullanıldığını gösterir. Erişim, Tainted ve geçerli özellikler, tüm veri türlerinde kullanılabilir. ValidBytesConst, ValidElementsConst ElementSize ve NullTerminted gibi diğer özellikleri işaretçisiyse, işaretçe üyeleri veya dizi türleri için destek. Özellikleri ve desteklenen veri türlerini tam listesi için bkz: [ek açıklama özellikleri](http://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki kod bu uyarıyı üretir:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 Bu uyarıyı düzeltmek için bir işaretçi ya da bir dizi türü aşağıdaki örnek kodda gösterildiği gibi kullanın:  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
