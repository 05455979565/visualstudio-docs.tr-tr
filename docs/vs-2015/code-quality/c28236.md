# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [C28236](https://docs.microsoft.com/visualstudio/code-quality/c28236).  
  
Uyarı C28236: ek açıklama bir ifadede kullanılamaz  
  
 Bu uyarı, bir ek açıklama bir ifadenin bağlamındaki olarak bildirilmiş bir işlev kullanmayı girişiminde bulunuldu gösterir. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
