# <a name="suppress-warnings-by-using-the-suppressmessage-attribute"></a>SuppressMessage Özniteliğini Kullanarak Uyarıları Bastırma
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Genellikle uyarının, takım üyelerinin kodun gözden geçirdiğini ve uyarının bastırıldığına ilişkin olduğunu bilmesini sağlamak için, uyarının uygulanamadığına işaret etmek yararlı olur. Kaynak gizleme (ISS) bölümünde, bir geliştiricinin uyarıyı oluşturan konuma yakın bir uyarı belirten özniteliği yerleştirmesini sağlar. Doğrudan kaynak dosyasına ISS özniteliğini ekleyebilir veya [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] IDE 'de kısayol menüsünü kullanabilirsiniz.
