# <a name="c28243"></a>C28243
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [C28243](https://docs.microsoft.com/visualstudio/code-quality/c28243).  
  
Uyarı C28243: parametre üzerindeki işlev daha gerektirir asıl türün izin verdiğinden başvuru  
  
 Sayısını `__deref` açıklamanın işleçlerde olduğu birden çok parametre türünün tanımlanan işaretçisinin düzeyi sayısı. Bu uyarı ya da değiştirerek, ek açıklama ya da başvurulan parametresi işaretçi düzeyleri numarayı düzeltin.
