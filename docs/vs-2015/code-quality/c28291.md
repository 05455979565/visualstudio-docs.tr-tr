# <a name="c28291"></a>C28291
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [C28291](https://docs.microsoft.com/visualstudio/code-quality/c28291).  
  
Uyarı C28291: deref düzeyi Post null/notnull 0 ise, işlev için anlamsız \<x > param en \<sayı >  
  
 Kod Analizi aracı, posta koşulu başvuru bir düzey sıfır parametresi null/boş olmayan özellik belirtildiğinde bu uyarı bildirir. Bu hata düzeyi sıfır değiştirilemez bir değerde başvuru oluşur.
