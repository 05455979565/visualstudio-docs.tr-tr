# <a name="concurrency-namespace-concurrency-visualizer"></a>Eşzamanlılık Ad Alanı (Eşzamanlılık Görselleştiricisi)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

`Concurrency` Size işlevleri erişmek için eşzamanlılık çalışma zamanı C++ için eşzamanlı programlama çerçevesi ve ad alanı sınıflar sağlar. Daha fazla bilgi için [eşzamanlılık çalışma zamanı](https://msdn.microsoft.com/library/874bc58f-8dce-483e-a3a1-4dcc9e52ed2c).  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
namespace Concurrency;  
```
