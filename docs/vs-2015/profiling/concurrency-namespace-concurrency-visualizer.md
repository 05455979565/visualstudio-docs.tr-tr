# <a name="concurrency-namespace-concurrency-visualizer"></a>Eşzamanlılık Ad Alanı (Eşzamanlılık Görselleştiricisi)

Bu konuda en son sürümünü şu yolda bulunabilir: [eşzamanlılık Namespace (eşzamanlılık görselleştiricisi)](https://docs.microsoft.com/visualstudio/profiling/concurrency-namespace-concurrency-visualizer).  
  
`Concurrency` Size işlevleri erişmek için eşzamanlılık çalışma zamanı C++ için eşzamanlı programlama çerçevesi ve ad alanı sınıflar sağlar. Daha fazla bilgi için [eşzamanlılık çalışma zamanı](http://msdn.microsoft.com/library/874bc58f-8dce-483e-a3a1-4dcc9e52ed2c).

