# <a name="how-to-filter-profiling-tools-report-views-to-display-just-my-code"></a>Nasıl yapılır: Yalnızca Kendi Kodumu Görüntülemek için Profil Oluşturma Araçları Raporu Görünümlerini Filtreleme
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio 2010 Premium, böylece bunlar yalnızca uyguladıysanız kodunu görüntülemek için performans raporları filtreleme sağlar. Yalnızca kendi kodum adlandırılır, bu özellik varsayılan olarak etkindir. Üzerinden yapılandırılan **yalnızca kendi kodum** seçeneğini **seçenekleri** iletişim kutusu.  
  
### <a name="to-enable-just-my-code"></a>Just My Code'u etkinleştirmek için  
  
1.  Üzerinde **Araçları** menüsünü tıklatın **seçenekleri**.  
  
2.  İçinde **seçenekleri** iletişim kutusunda **performans araçları** seçip **genel** düğümü.  
  
3.  Altında **genel ayarlar**seçin **yalnızca benim kodumu etkinleştir** onay kutusu.  
  
### <a name="to-disable-just-my-code"></a>Just My Code'u devre dışı bırakmak için  
  
1.  Üzerinde **Araçları** menüsünü tıklatın **seçenekleri**.  
  
2.  İçinde **seçenekleri** iletişim kutusunda **performans araçları** seçip **genel** düğümü.  
  
3.  Altında **genel ayarlar**temizleyin **yalnızca benim kodumu etkinleştir** onay kutusu.
