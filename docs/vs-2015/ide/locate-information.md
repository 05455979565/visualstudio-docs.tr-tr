# <a name="locate-information"></a>Bilgilerin Konumunu Bulma
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Yardım Görüntüleyici Yardım içerisinde bilgileri bulmanız birkaç farklı yol sunar. Bilgi türüne bağlı olarak, stratejiler diğerlerinden daha etkili olabilir belirli arıyorsanız.  
  
|Hakkında bilgi için|Deneme|  
|----------------------------|---------|  
|Kullanıcı arabirimi öğesi|F1 tuşu|  
|Geçerli konuyla aynı içindekiler alanının bulunan diğer konular|**İçindekilerde konuyu Göster** araç çubuğunda|  
|Bir özellik veya teknoloji|Dizin|  
|Yöntemler ve olaylar gibi dil öğeleri|Ara|
