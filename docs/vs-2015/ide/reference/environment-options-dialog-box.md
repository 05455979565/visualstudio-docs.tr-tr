# <a name="environment-options-dialog-box"></a>Ortam Seçenekleri İletişim Kutusu
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [ortam Seçenekleri iletişim kutusu](https://docs.microsoft.com/visualstudio/ide/reference/environment-options-dialog-box).  
  
  
Sayfaları kullanabilirsiniz **ortam** klasöründe **seçenekleri** nasıl tümleşik geliştirme ortamı (IDE) belirli öğelerinin görüntülenme ve davranış ayarlamak için iletişim kutusu. Menü çubuğundan seçme, ortam sayfalarına erişebilirsiniz **Araçları**, **seçenekleri**seçip **ortam**.  
  
> [!NOTE]
>  Etkin ayarlarınıza ve kullandığınız Visual Studio sürümüne bağlı olarak, bilgisayarınızda görünen iletişim kutuları ve menü komutları Yardım'da açıklananlardan farklı olabilir. Ayarlarınızı değiştirmek için seçin **içeri ve dışarı aktarma ayarları** üzerinde **Araçları** menüsü. Daha fazla bilgi için [Visual Studio'da geliştirme ayarlarını özelleştirme](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3).
