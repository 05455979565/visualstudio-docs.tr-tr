# <a name="diff"></a>/Diff
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

İki dosyayı karşılaştırır. Farklar özel bir Visual Studio penceresinde görüntülenir.

## <a name="syntax"></a>Sözdizimi

```
devenv /Diff SourceFile, TargetFile, [SourceDisplayName],[TargetDisplayName]
```

## <a name="arguments"></a>Arguments
 `SourceFile` gerekiyor. Karşılaştırılacak ilk dosyanın tam yolu ve adı.

 `TargetFile` gerekiyor. Karşılaştırılacak ikinci dosyanın tam yolu ve adı

 Isteğe bağlı `SourceDisplayName`. İlk dosyanın görünen adı.

 Isteğe bağlı `TargetDisplayName`. İkinci dosyanın görünen adı.
