# <a name="how-to-open-the-data-sources-window"></a>Nasıl yapılır: Veri Kaynakları penceresini açma
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**Veri kaynakları** penceresi projenizdeki veri kaynaklarını görüntüler. Kullandığınız **veri kaynakları** pencere öğeleri pencereden projenizdeki formlara sürükleyerek veriye bağlı denetim içeren kullanıcı arabirimi oluşturmak için. Bkz: [Visual Studio'da verilere Windows Forms bağlama denetimleri](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 Doldurabilirsiniz **veri kaynakları** çalıştırarak penceresi **veri kaynağı yapılandırması** Sihirbazı. Daha fazla bilgi için [veri kaynağı Yapılandırma Sihirbazı](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f).  
  
### <a name="to-open-the-data-sources-window"></a>Veri Kaynakları penceresini açmak için  
  
-   Menü çubuğunda, **görünümü**, **diğer Windows**, **veri kaynakları** (veya Shift + Alt + D tuşlarına basın).  
  
    > [!IMPORTANT]
    >  Bir projeye veri kaynağı ekleyebilmek için projenin veri kaynakları oluşturmayı ve bunlarla çalışmayı desteklemesi gerekir. Örneğin, açılamıyor **veri kaynakları** penceresini bir Windows Store uygulaması için bir proje.
