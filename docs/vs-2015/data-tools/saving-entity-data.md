# <a name="saving-entity-data"></a>Varlık Verilerini Kaydetme
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Oluşturmak, güncelleştirmek veya WCF Veri Hizmetleri veritabanına bağlanma varlıklarını silme, değişiklikleri veritabanına geri açıkça kaydetmeniz gerekir. Aşağıdaki konular, WCF Veri Hizmetleri uygulamalarında varlık veri değişiklikleri kaydetmek nasıl açıklar.  
  
|Konu|Açıklama|  
|-----------|-----------------|  
|[Nasıl yapılır: Varlık Ekleme, Değiştirme ve Silme](http://msdn.microsoft.com/library/a00f8933-b232-4445-95ba-adc634f055d8)|Bir veri hizmeti aracılığıyla geri varlık verilerindeki değişiklikleri kaydetmek gösterilmektedir.|  
|[Nasıl yapılır: Zaman Uyumsuz Veri Hizmeti Sorguları Yürütme](http://msdn.microsoft.com/library/902a2dc1-d0e9-4b00-90a8-becc4cb1f6a7)|Zaman uyumsuz olarak bir veri hizmeti aracılığıyla değişiklikleri kaydetme işlemi gösterilmektedir.|
