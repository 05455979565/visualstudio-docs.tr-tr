# <a name="what39s-new-in-the-xml-editor"></a>Hangi&#39;XML Düzenleyicisi'nde yeni s
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [ne&#39;s yeni XML Düzenleyicisi'nde](https://docs.microsoft.com/visualstudio/xml-tools/what-s-new-in-the-xml-editor).  
  
  
Aşağıdaki özellikleri, Visual Studio 2010'da yenidir.  
  
## <a name="xslt-intellisense"></a>XSLT IntelliSense  
 XSLT IntelliSense özelliği için otomatik tamamlama öznitelik kümesi adları sağlar; Şablon modları ve adları; ve parametre adları için belirtilen bir mod ya da belirtilen bir şablonun adı. Daha fazla bilgi için [XML Düzenleyicisi IntelliSense özellikleri](../xml-tools/xml-editor-intellisense-features.md).  
  
## <a name="xslt-profiler"></a>XSLT Profil Oluşturucusu  
 XSLT Profiler ayrıntılı XSLT performansı ölçmesine, değerlendirmek ve performans ile ilgili sorunları XSLT kodda hedef yardımcı olan raporlar oluşturur. XSLT Profiler, XSL ve XSLT stil sayfası iyileştirmeler için faydalı ipuçları da içerir. Daha fazla bilgi için [izlenecek yol: XSLT Profiler](../xml-tools/walkthrough-xslt-profiler.md).  
  
## <a name="xslt-hierarchy"></a>XSLT hiyerarşisi  
 XSLT hiyerarşisi araç dahil stil sayfaları ve/veya yerleşik bir şablon kuralları kesme noktaları eklemenize olanak sağlar. Daha fazla bilgi için [izlenecek yol: XSLT hiyerarşisi kullanma](../xml-tools/walkthrough-using-xslt-hierarchy.md).
