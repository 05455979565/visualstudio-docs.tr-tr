# <a name="xslt-profiler"></a>XSLT Profil Oluşturucusu
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

XSLT Profiler, geliştirme ve hata ayıklama XSLT belge yardımcı olan bir Performans Analizi profil oluşturucu aracıdır. Ölçün ve XSLT kodundaki performansla ilgili sorunları ayrıntılı XSLT performans raporlar oluşturarak değerlendirmek geliştiriciler XSLT Profiler sağlar. XSLT Profiler yararlı ipuçları, en yüksek performans gerektiren XSLT tabanlı uygulamalar için gerekli olan XSL ve XSLT stil sayfası iyileştirmeleri içerir.  
  
 XSLT Profiler Visual Studio 2010 'un bir parçasıdır ve Visual Studio'dan kullanılabilir **XML** menüsü.  
  
 ![XSLT Profiler](../xml-tools/media/xsltprofilermenu.gif "XSLTProfilerMenu")  
  
 XSLT Profiler ilk bir eklenti için Visual Studio 2008 sunulmuştur. Daha fazla bilgi için şirket ilgili kaynaklar bölümüne bakın. [XSLT Profiler Şubat 08 Visual Studio Community Technology Preview](http://go.microsoft.com/fwlink/?LinkId=142987) sayfası.  
  
> [!NOTE]
>  Bazı görsel öğeler, Visual Studio 2008'den beri değiştirilmiş olduğundan bu belgelerdeki arkadaşlarınıza kıyasla bazı ekranlar farklı şekilde Visual Studio 2010'da görüntülenir.
