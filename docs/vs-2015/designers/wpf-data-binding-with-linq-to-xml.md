# <a name="wpf-data-binding-with-linq-to-xml"></a>LINQ to XML ile WPF verilerini bağlama

Bu konuda en son sürümünü şu yolda bulunabilir: [WPF veri bağlama LINQ to XML ile](https://docs.microsoft.com/visualstudio/designers/wpf-data-binding-with-linq-to-xml).
