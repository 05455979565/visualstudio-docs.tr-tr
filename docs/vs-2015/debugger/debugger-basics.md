# <a name="debugger-basics"></a>Hata Ayıklayıcı Temel Bilgileri
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [hata ayıklayıcı temel bilgileri](https://docs.microsoft.com/visualstudio/debugger/debugger-basics). 

Visual Studio hata ayıklayıcı, programınızın çalışma zamanı davranışını gözlemeyi ve mantık hatalarını bulmayı sağlar. Hata ayıklayıcı, tüm Visual Studio programlama dilleri ve bunların ilişkili kitaplıkları ile çalışır. Hata ayıklayıcı, kodunuzda adım adım ilerleyin ve uygulamanızda değişkenleri değerlendirmek için yürütmeyi kesebilirsiniz.
