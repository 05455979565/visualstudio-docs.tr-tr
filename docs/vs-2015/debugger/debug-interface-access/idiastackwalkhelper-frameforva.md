# <a name="idiastackwalkhelperframeforva"></a>IDiaStackWalkHelper::frameForVA
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [IDiaStackWalkHelper::frameForVA](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiastackwalkhelper-frameforva).  
  
Belirtilen sanal adres içeren yığın çerçevesini alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp#  
HRESULT frameForVA(   
   ULONGLONG        va,  
   IDiaFrameData**  ppFrame  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `va`  
 [in] Çerçeve veriler için sanal adres.  
  
 `ppFrame`  
 [out] Bir [Idiaframedata](../../debugger/debug-interface-access/idiaframedata.md) belirtilen adreste yığın çerçevesini temsil eden nesne.  
  
## <a name="return-value"></a>Dönüş Değeri  
 Başarılı olursa döndürür `S_OK`; Aksi takdirde bir hata kodu döndürür.
