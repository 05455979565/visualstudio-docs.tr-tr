# <a name="idiaenumsourcefilesget_count"></a>IDiaEnumSourceFiles::get_Count

## <a name="syntax"></a>Sözdizimi

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>Parametreler
 pRetVal

dışı Kaynak dosya sayısını döndürür.
