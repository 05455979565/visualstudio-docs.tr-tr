# <a name="idiasymbolgetinterruptreturn"></a>IDiaSymbol::get_interruptReturn
İşlevi bir dönüş kesme yönerge içeren olup olmadığını belirten bir bayrak alır (örneğin, X86 bütünleştirilmiş kodu `iret`).  
  
## <a name="syntax"></a>Sözdizimi  
  
```C++  
HRESULT get_interruptReturn(  
   BOOL *pFlag  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `pFlag`  
 [out] Döndürür `TRUE` işlevi bir dönüş kesme yönerge; varsa, aksi takdirde, döndürür `FALSE`.  
  
## <a name="return-value"></a>Dönüş Değeri  
 Başarılı olursa, döndürür `S_OK`; Aksi halde döndürür `S_FALSE` veya bir hata kodu.  
  
> [!NOTE]
>  Dönüş değeri `S_FALSE` özelliği simgesi kullanılabilir olmadığı anlamına gelir.
