# <a name="windows-view"></a>Pencereler Görünümü
İlk Spy ++ açtığınızda, Windows görünüm sistemdeki tüm windows ve denetimleri ağacı görüntüler. Pencere tanıtıcısı ve sınıf adı gösterilir. Geçerli masaüstü ağacının en üstünde bir penceredir. Diğer tüm windows masaüstünün alt öğeleri ve standart pencere hiyerarşi göre listelenmiştir. Üst Girintili expansible listelerindeki eşdüzey windows görünür.  
  
 Aşağıdaki şekilde, en üst düğüme genişletilmiş tipik bir Spy ++ Windows görünümü gösterir.  
  
 ![Spy &#43; &#43; Pencereler görünümü](../debugger/media/spy--_windowsview.png "Spy ++ _WindowsView")  
Spy ++ pencereler görünümü  
  
 Geçerli masaüstü ağacının en üstünde bir penceredir. Diğer tüm windows masaüstünün alt öğeleri ve standart pencere hiyerarşi göre Z düzeni tarafından sıralanan eşdüzey windows ile birlikte listelenir. Genişletme veya herhangi bir üst düğümün ağacının tıklayarak daraltmak + veya - düğüm yanındaki simge.  
  
 Pencereler görünümü odağa sahip olduğunda Bulucu aracında kullanabilmeniz için [pencere arama iletişim kutusu](../debugger/window-search-dialog-box.md) sisteminizde herhangi penceresi açık bilgilerini görüntülemek için.
