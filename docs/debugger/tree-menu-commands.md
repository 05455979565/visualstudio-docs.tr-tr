# <a name="tree-menu-commands"></a>Ağaç Menü Komutları
**Ağaç** menü komutları Genişlet veya daralt görünümleri sağlar.  
  
 **Bir düzey Genişlet**  
 Seçili öğenin sistem görünümünde bir düzey altındaki görüntüleyerek genişletir.  
  
 **Dalını genişletin**  
 Seçili öğenin altındaki tüm düzeyleri genişletir.  
  
 **Tümünü Genişlet**  
 Genişletir ve bir sistem görünümünde tümünü görüntüler.  
  
 **Daralt**  
 Seçili öğenin altındaki tüm düzeyleri daraltır.
