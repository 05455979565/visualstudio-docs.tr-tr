# <a name="how-can-i-use-debugger-windows-while-debugging-a-foreground-program"></a>Ön Plan Programında Hata Ayıklarken Hata Ayıklayıcı Penceresini Nasıl Kullanabilirim?
## <a name="problem-description"></a>Sorun açıklaması  
 Bir ekran boyama sorun hata ayıklamak çalışıyorum. Bu sorunu izlemek için hata ayıklama windows erişim izinleri olmayan anlamına gelir ön planda programımı tutmak sahibim. Ne yapabilirim?  
  
## <a name="solution"></a>Çözüm  
 İkinci bir bilgisayar varsa, uzaktan hata ayıklama kullanabilirsiniz. İki bilgisayarlı kurulum ile ana bilgisayarda hata ayıklayıcı çalışması sırasında uzak bilgisayardaki ekran boyama izleyebilirsiniz. Uzaktan hata ayıklama hakkında daha fazla bilgi için bkz: [uzaktan hata ayıklama](../debugger/remote-debugging.md).
