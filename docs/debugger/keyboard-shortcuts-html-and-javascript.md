# <a name="keyboard-shortcuts-html-and-javascript"></a>Klavye kısayolları (HTML ve JavaScript)
  
 HTML ve JavaScript kullanarak UWP uygulamaları hata ayıklama için klavye kısayolları Visual Studio'da.  
  
## <a name="general-shortcuts"></a>Genel kısayolları  
 HTML ve JavaScript kullanarak UWP uygulamaları için tüm Araçlar üzerinden bu kısayolları çalışır.  
  
|Eylem|Kısayol|  
|------------|--------------|  
|Windows uygulaması Yenile|Ctrl+Shift+R|  
|Windows uygulamasını yeniden başlatın|Ctrl+Shift+F5|  
|Hata ayıklamayı durdurun|Shift+F5|  
|Performans ve tanılama hub'ı açın|Alt+F2|  
  
## <a name="javascript-console-shortcuts"></a>JavaScript Konsolu kısayollarını  
 Bu kısayolların JavaScript konsol penceresinde kullanın.  
  
|Eylem|Kısayol|  
|------------|--------------|  
|Bir konsol komutunu çalıştırın|Enter|  
|Konsol temizleyin|Esc|  
|Tek satırlı ve çok satırlı modu arasında geçiş|Ctrl+Alt+M|  
  
## <a name="dom-explorer-shortcuts"></a>DOM Gezgini kısayolları  
 Bu kısayolların DOM Gezgini'ni kullanın.  
  
|Eylem|Kısayol|  
|------------|--------------|  
|Öğe Seç|Ctrl+B|  
|Görüntüleme ve Web sayfası Vurgu kutuları kullanırken gizleme arasında geçiş **öğesi seçin**|Ctrl+Shift+I|  
|Kullanarak değişiklikleri **HTML olarak Düzenle**|Ctrl+Enter|  
|Kullanarak değişiklikleri atmak **HTML olarak Düzenle**|Esc|  
|1 ile seçilen CSS değerini artırın|Yukarı Ok|  
|Seçili CSS değeri 1 ile azaltma|Aşağı Ok|
