# <a name="overview-of-the-visual-studio-debugger"></a>Visual Studio hata ayıklayıcısı genel bakış

Visual Studio hata ayıklayıcısı, programın çalışma zamanı davranışını inceleyin ve mantık hataları bulmak sağlar. Hata ayıklayıcı tüm Visual Studio programlama dilleri ve bunların ilişkili kitaplıkları ile çalışır. Hata ayıklayıcısını kullanmaya kodunuzun içinde ilerleyebilir ve uygulamanızda değişkenleri değerlendirmek için yürütme bozulabilir.  

|         |         |
|---------|---------|
|  ![video kamera simgesine film](../install/media/video-icon.png "bir videoyu izleyin")  |    [Bir video izlemek](https://mva.microsoft.com/en-US/training-courses-embed/getting-started-with-visual-studio-2017-17798/Debugger-Feature-tour-of-Visual-studio-2017-sqwiwLD6D_1111787171) hata ayıklamayı benzer adımları gösterir. |

Hata ayıklayıcı kesme noktalarını ayarlayın, değişkenleri inceleyin ve saatlerde ayarlayın, çağrı yığını görüntülemek, uzak cihazda hata ayıklama, birden çok iş parçacıklı uygulamalar hata ayıklama ve diğer birçok özelliklerden izin verir.
