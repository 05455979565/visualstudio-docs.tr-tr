# <a name="executable-for-debugging-session-dialog-box"></a>Hata Ayıklama Oturumu İçin Yürütülebilir İletişim Kutusu

Bir DLL'nin hatasını ayıklamaya çalıştığınızda hangi hiçbir yürütülebilir dosyası için belirtilen bu iletişim kutusu görüntülenir. Visual Studio, DLL doğrudan başlatılamıyor. Bunun yerine, Visual Studio belirtilen yürütülebilir dosyayı çalıştırır. Yürütülebilir dosya tarafından çağrıldığında DLL hatalarını ayıklayabilir.

 **Yürütülebilir dosya adı** ayıkladığınız DLL'yi çağıran bir yürütülebilir için yol adı girin.

 **Burada proje URL (sadece ATL Sunucusu için) erişilebilir** ATL Sunucu DLL ayıklıyorsanız, projenin bulunduğu URL'yi girin.

 Sonraki hata ayıklama oturumları için bunları yeniden girmeniz gerekmez giriş yapıldığında, bu ayarları projenin özellik sayfaları depolanır. Ayarları değiştirmeniz gerekiyorsa, özellik sayfaları açın ve değerlerini değiştirin. Hata ayıklama oturumu için yürütülebilir bir belirtme hakkında daha fazla bilgi için bkz. [DLL'lerin hata ayıklama](../debugger/how-to-debug-from-a-dll-project.md).
