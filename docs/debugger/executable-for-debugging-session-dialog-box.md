# <a name="executable-for-debugging-session-dialog-box"></a>Hata Ayıklama Oturumu İçin Yürütülebilir İletişim Kutusu
DLL hata ayıklamak çalıştığınızda hangi hiçbir yürütülebilir dosyası için belirtilen bu iletişim kutusu görüntülenir. Visual Studio DLL doğrudan başlatılamıyor. Bunun yerine, belirtilen yürütülebilir dosya başlatır. Yürütülebilir dosya tarafından çağrıldığında DLL ayıklayabilirsiniz.  
  
 **Yürütülebilir dosya adı**  
 Hata ayıklama yaptığınız DLL çağıran bir yürütülebilir dosya yol adı girin.  
  
 **Proje burada olabilir URL erişilen (yalnızca ATL Sunucu)**  
 ATL Sunucu DLL hata ayıklama, proje bulunabileceği URL'sini girin.  
  
 Sonraki hata ayıklama oturumları için bunları yeniden girmeniz gerekmez şekilde girilen sonra bu ayarları proje özellik sayfalarını depolanır. Ayarları değiştirmek gerekiyorsa, özellik sayfaları açın ve değerlerini değiştirin. Hata ayıklama oturumu için yürütülebilir bir dosya belirtme hakkında daha fazla bilgi için bkz: [DLL'lerde hata ayıklama](../debugger/how-to-debug-from-a-dll-project.md).
