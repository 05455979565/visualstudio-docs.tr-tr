# <a name="window-menu-commands"></a>Pencere Menü Komutları
**Penceresi** menü Spy ++ windows yönetmek için komutlar sağlar.

 **Art arda** cascade desenindeki tüm windows yeniden düzenler.

 **Yatay Döşe** Yatay Döşe desenindeki tüm windows yeniden düzenleyin.

 **Dikey Döşe** Dikey Döşe desenindeki tüm windows yeniden düzenler.

 **Simgeleri Düzenle** windows küçülttüğünüzde görünümünü simgeleri yeniden düzenler.

 **Kapat / Tümünü Kapat** Spy ++'ta bir ya da tüm windows kapatır.

 **Yenileme** seçili Görünümü yeniler.

 **Pencere** belirli bir pencere seçilmesine izin verir.
