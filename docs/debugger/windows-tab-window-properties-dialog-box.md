# <a name="windows-tab-window-properties-dialog-box"></a>Pencereler Sekmesi, Pencere Özellikleri İletişim Kutusu
Kullanım **Windows** Windows'da bilgi göstermek için sekmesinde seçilen pencereyi ilgili. Görüntülenecek [pencere Özellikleri iletişim kutusu](../debugger/window-properties-dialog-box.md), odağı Taşı [Windows görünümü](../debugger/windows-view.md) penceresi. Herhangi bir pencere düğüm ağaçta seçin ve ardından **özellikleri** gelen **görünümü** menüsü.

 Aşağıdaki ayarlar kullanılabilir **Windows** sekmesinde:

|Giriş|Açıklama|
|-----------|-----------------|
|**Sonraki pencere**|Pencere ağaç görünümünde ("none" sonraki penceresi yok ise) gösterilen aynı sırasını (Z düzeninde) sonraki eşdüzey penceresinde tanıtıcısı. Sonraki pencere özelliklerini görüntülemek için bu girdiyi seçin.|
|**Önceki pencere**|Pencere ağaç görünümünde ("none" önceki penceresi yok ise) gösterilen aynı sırasını (Z düzeninde) önceki eşdüzey penceresinde tanıtıcısı. Önceki pencere özelliklerini görüntülemek için bu girdiyi seçin.|
|**Ana penceresi**|Bu pencerenin üst penceresine ("none" üst öğe yoksa) tanıtıcısı. Üst pencere özelliklerini görüntülemek için bu girdiyi seçin.|
|**İlk alt**|Pencere ağaç görünümünde ("none" hiçbir alt pencereleri varsa) gösterilen Bu pencerenin ilk alt penceresinde (Z düzeninde) dizisi tanıtıcısı. İlk alt penceresi özelliklerini görüntülemek için bu değeri seçin.|
|**Sahip penceresi**|Bu pencerenin sahibini pencere tanıtıcısı. Bir uygulamanın ana pencere genellikle sistem kalıcı iletişim windows, örneğin ("none" sahibi varsa) sahip. Sahip penceresinin özelliklerini görüntülemek için bu girdiyi seçin.|
