# <a name="cannot-change-value-dialog-box"></a>Değer Değiştirilemez İletişim Kutusu
## <a name="error"></a>Hata  
 `The value of this variable cannot be changed`&#124; `The name` *adı* `does not exist in the current context` &#124; *diğer çeşitli iletileri*  
  
 Hata ayıklayıcı penceresinde (otomobiller, izleme veya Yereller windows) veya QuickWatch iletişim kutusu geçersiz bir değer için bir değişken içeriğini değiştirmeye çalıştığınızda bu ileti kutusu görünür. Örneğin, bir tamsayı değişkenin değeri olarak bir karakter dizesi ayarlamaya çalışırsanız, bu ileti kutusu görünür.  
  
## <a name="solution"></a>Çözüm  
 Hata ayıklayıcı penceresine yazın giriş emin olun veya QuickWatch iletişim kutusu ayarlamaya çalıştığınız değişkeni için geçerli bir değeri temsil eder.
