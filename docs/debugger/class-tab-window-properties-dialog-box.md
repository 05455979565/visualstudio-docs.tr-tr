# <a name="class-tab-window-properties-dialog-box"></a>Sınıf Sekmesi, Pencere Özellikleri İletişim Kutusu
Kullanım **sınıfı** sekmesi seçili pencere sınıfı hakkında bilgi gösterir. Görüntülenecek [pencere Özellikleri iletişim kutusu](../debugger/window-properties-dialog-box.md), odağı taşımak [Windows görünümü](../debugger/windows-view.md) penceresi. Herhangi bir pencere düğüm ağaçta seçin, sonra seçin **özellikleri** gelen **Görünüm** menüsü.  
  
 Aşağıdaki ayarlar kullanılabilir **sınıfı** sekmesi:  
  
|Giriş|Açıklama|  
|-----------|-----------------|  
|**Sınıf adı**|Bu pencere sınıfı adı (veya sıra numarası).|  
|**Sınıf stilleri**|Sınıf stili kodlarını birleşimi.|  
|**Sınıf bayt**|Bu pencere sınıfı ile ilgili uygulamaya özgü veriler.|  
|**Sınıf Atom**|Atom tarafından döndürülen sınıfı için **RegisterClass** çağırın.|  
|**Örnek tanıtıcısı**|Sınıf kayıtlı modül örneği tanıtıcısı. Örnek tanıtıcısı benzersiz değil.|  
|**Pencere bayt**|Bu sınıfın her pencere ilişkili ek bayt sayısı. Bu bayt anlamını uygulama tarafından belirlenir. DWORD biçiminde bayt değerleri görmek için liste kutusunu genişletin.|  
|**Pencere Proc**|Geçerli adresini **WndProc** Windows, bu sınıfın işlevi. Bu farklıdır **penceresi Proc** üzerinde **genel** penceresi sınıflandırma sekmesini.|  
|**Menü adı**|Bu sınıf ("hiçbiri" hiçbir menüsünü ise) windows ile ilişkili ana menü adı.|  
|**Simge işleci**|Bu sınıf ("hiçbiri" hiçbir simge ise) windows ile ilişkili simgesiyle işleci.|  
|**İmleç işleci**|Bu sınıf ("hiçbiri" Hiçbir imleç ise) windows ile ilişkili imleci için işleci.|  
|**Arka plan Fırçası**|Bu sınıf veya pencere arka plan ("hiçbiri" hiçbir fırça ise) boyama için önceden tanımlanmış COLOR_ * renkleri birinin windows ile ilişkili arka planı fırçasını için işleci.|
