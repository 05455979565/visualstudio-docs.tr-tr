# <a name="help-menu-commands"></a>Yardım Menü Komutları
**Yardımcı** menü komutları Yardım dosyaları açmak için sağlar.

 **Dizin** Spy ++ Yardımı için içeriği ekranı açılır.

 **Yardım'ı kullanarak** Yardımın nasıl kullanılacağını açıklayan bir Yardım dosyasını açar.

 **Spy ++ hakkında** Spy ++ için yazılım sürüm ve telif hakkı bilgileri gösterir.
