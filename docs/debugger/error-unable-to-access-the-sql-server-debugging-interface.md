# <a name="error-unable-to-access-the-sql-server-debugging-interface"></a>Hata: SQL Server hata ayıklama arabirimine erişilemiyor

### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Bkz. [SQL hata ayıklamayı ayarlama](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2010/s4sszxst(v=vs.100)).
