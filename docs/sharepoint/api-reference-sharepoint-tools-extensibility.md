# <a name="api-reference-sharepoint-tools-extensibility"></a>API Başvurusu (SharePoint araçları genişletilebilirliği)
  Bu bölümde, Visual Studio'da SharePoint araçlarını genişletmek için API başvuru belgelerini içerir.

## <a name="in-this-section"></a>Bu bölümde
 <xref:Microsoft.VisualStudio.SharePoint>

 SharePoint Proje sistemini genişletmek için kullandığınız türleri içerir. Örneğin, yerleşik SharePoint projelerini ve proje öğelerini genişletebilir veya kendi Proje öğelerinizi oluşturabilirsiniz.

 <xref:Microsoft.VisualStudio.SharePoint.Commands>

 Özel oluşturmak için kullandığınız türleri içerir *SharePoint komutları*. Bir SharePoint komutu bir SharePoint Araçlar uzantısından SharePoint sunucusu nesne modeline çağıran bir yöntemdir.

 <xref:Microsoft.VisualStudio.SharePoint.Deployment>

 SharePoint projeleri için dağıtım işlemi genişletmek için kullandığınız türleri içerir.

 <xref:Microsoft.VisualStudio.SharePoint.Explorer>

 SharePoint düğümleri genişletmek için kullandığınız türleri içerir **Sunucu Gezgini** veya kendi düğüm türlerinizi tanımlamak için.

 <xref:Microsoft.VisualStudio.SharePoint.Explorer.Extensions>

 Yerleşik hakkında bilgi almak için kullandığınız türleri içerir **Sunucu Gezgini** bir liste, alan veya içerik türünü temsil eden bir düğüm gibi bir SharePoint sitesindeki tek tek bileşenleri temsil eden düğümler.

 <xref:Microsoft.VisualStudio.SharePoint.Features>

 SharePoint projesindeki bir özellik tanımına erişmek için kullandığınız türleri içerir.

 <xref:Microsoft.VisualStudio.SharePoint.Packages>

 SharePoint projesindeki paket tanımına erişmek için kullandığınız türleri içerir.

 <xref:Microsoft.VisualStudio.SharePoint.Remote.Authentication>

 Uzak SharePoint sitelerine dağıtılan kimlik doğrulaması ve SharePoint için uygulamalar ile iletişim kurmak için kullandığınız türleri içerir.

 <xref:Microsoft.VisualStudio.SharePoint.Remote.Commands>

 Uzak SharePoint sitelerine dağıtılan SharePoint uygulamaları ile kullanılan SharePoint uzak komutlarını oluşturmak için kullandığınız türleri içerir.

 <xref:Microsoft.VisualStudio.SharePoint.Tasks>

 Visual Studio paketleme ve SharePoint projeleri, Office uygulamaları ve SharePoint uygulamaları için hata ayıklama görevleri oluştururken kullandığı türleri içerir. Bu API Office ve SharePoint altyapısını destekler ve doğrudan kodunuzdan kullanılmaya yönelik değildir.

 <xref:Microsoft.VisualStudio.SharePoint.Validation>

 Bir SharePoint projesi için özellik ve paket doğrulama davranışını özelleştirmek için kullandığınız türleri içerir.
