# <a name="view-recent-job-performance-and-details"></a>En son iş performansını ve ayrıntıları görüntüleyin

İşler gönderildikten sonra, durumlarını, sürelerini ve daha fazlasını görmek için işlerin listesini görüntüleyebilirsiniz.

1. Sunucu **Gezgini'nde,** belirli bilgi işlem bağlamını genişletin.
2. **İşler**’e çift tıklayın.
3. Bu bilgi işlem bağlamına gönderilen işlerin listesini görürsünüz.
4. Ayrıntıları görüntülemek için listede belirli bir **İş'i** seçin.

![işleri izleme](media/job-details/monitor-jobs.png)

> Linux VM'lerine gönderilen iş geçmişi /tmp dizininde VM'de depolanır. Bu nedenle, yeniden başlatıldığında iş geçmişi temizlenir. İş geçmişinizin kalıcı bir kaydı için, lütfen Azure Machine öğreniminde VM'inizi bilgi işlem bağlamı olarak yapılandırın, ardından İşi Azure Machine Learning'e Gönderin (Bilgi işlem bağlamı olarak VM'nizi seçin).
