> [!NOTE]
> C dil standardı, kitaplıklar gibi uygulamalar için alt`_`() karakteriyle başlayan tanımlayıcıların kullanımını saklı tutar. Kodunuzda alt bir alt noktayla başlayan adları kullanmayın.
