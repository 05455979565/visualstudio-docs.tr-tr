# <a name="collect-additional-performance-data"></a>Ek performans verileri toplama

Bir profil oluşturma yöntemi kullanarak topladığınız verilere ek olarak, Visual Studio Profil Oluşturma Araçları ek verileri kaydetmenizi ve görüntülemenizi sağlar.

## <a name="common-tasks"></a>Genel görevler

|Görev|İlgili içerik|
|----------|---------------------|
|**Uygulamanızdaki ADO.NET aramaları için performans verileri toplayın.** Uygulamanızın veritabanlarına yaptığı eşzamanlı çağrılar hakkında veri ekleyin.|- [Katman etkileşim verilerini toplama](../profiling/collecting-tier-interaction-data.md)|
|**Windows performans sayacı verilerini toplayın.** Profil oluşturma verilerinize profil oluşturma işaretleri olarak sistem performans sayaçları ekleyin. İşaretler raporları filtrelemek için kullanılabilir.|- [Nasıl kullanılır: Windows karşı veri toplama](../profiling/how-to-collect-windows-counter-data.md)|
|**Windows verileri için Olay İzleme'yi toplayın.** Profil oluşturma verilerinize ek olarak uygulama ve sistem olaylarını kaydetmek için Windows (ETW) verileri için Olay İzleme'yi toplayın.|- [Nasıl yapılır: Windows (ETW) verileri için Olay İzlemesini Toplama](../profiling/how-to-collect-event-tracing-for-windows-etw-data.md)|
