# <a name="how-to-view-performance-warnings"></a>Nasıl yapılır: Performans uyarılarını görüntüleme
Profil Oluşturma Araçları [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] Profil Oluşturma Araçlarının performans uyarıları, profilli bir uygulamada program yürütmesini yavaşlatabilecek sorunları gösterir. Uyarılar, daha yararlı veriler toplamak için toplama yöntemlerini değiştirmek isteyebileceğinizi de gösterebilir. Performans uyarıları bir profil oluşturma oturumunda otomatik olarak **Error List** oluşturulur ve [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] IDE'de bir profil oluşturma veri dosyası açıldığında Hata Listesi penceresinde görünür.

 Profil oluşturma veri dosyasını açtığınızda **Hata Listesi** penceresi görünmüyorsa, **Görünüm** menüsünü tıklatın ve ardından **Hata Listesi'ni**tıklatın.
