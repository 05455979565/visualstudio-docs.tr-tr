# <a name="functions-view"></a>İşlevler Görünümü
İşlevler görünümü profil oluşturma sırasında çağrılan işlevleri listeler.

 Varsayılan olarak, işlevler ada göre listelenir. Modüle göre işlevleri görüntülemek için, İşlevler Görünümü penceresinde sağ tıklatın ve ardından **Modül Görünümünde Göster'i**seçin. Bu, Modül görünümünü görüntüler.

 Sütun eklemek veya kaldırmak için rapor penceresine sağ tıklayın ve ardından **Sütun Ekle/Kaldır'ı**seçin. Ayrıca, bir sütun adını tıklatarak verileri sıralayabilirsiniz. Daha fazla bilgi için [bkz: Rapor Görünümü sütunlarını özelleştirin.](../profiling/how-to-customize-report-view-columns.md)

 İşlevler görünümünde kullanılabilen sütunlar, verileri toplamak için kullanılan profil oluşturma yöntemine (örnekleme veya enstrümantasyon) ve .NET bellek verilerinin profil oluşturma çalışmasında toplanıp toplanmadığına bağlıdır.
