# <a name="command-line-profiling-tools-reference"></a>Komut satırı profil oluşturma araçları başvurusu
Komut satırı performans araçları, programcıların uygulamalarını komut isteminde profillemelerine olanak sağlar. Aşağıdaki komut satırı profil oluşturma araçları komut isteminde profil kodu oluşturmanızı sağlar.

## <a name="in-this-section"></a>Bu bölümde
- [VSInstr](../profiling/vsinstr.md)

 VSInstr.exe enstrümantasyon seçenekleri listeler.

- [VSInstr Uyarıları](../profiling/vsinstr-warnings.md)

 VSInstr aracı tarafından verilen uyarıları listeler.

- [VSPerfCLREnv](../profiling/vsperfclrenv.md)

 Listeler VSPerfCLREnv.exe seçenekleri profil oluşturma yönetilen kod için ortamı yapılandırmak için.

- [VSPerfCmd](../profiling/vsperfcmd.md)

 Performans verilerini toplamak için VSPerfCmd.exe seçeneklerini listeler.

- [VSPerfMon](../profiling/vsperfmon.md)

 Listeler VSPerfMon.exe veri toplama için seçenekler.

- [VSPerfReport](../profiling/vsperfreport.md)

 Raporlar oluşturmak için VSPerfReport.exe komut satırı seçeneklerini listeler.
