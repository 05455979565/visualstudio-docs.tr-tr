# <a name="report-based-on-visible-time-range"></a>Görünür zaman aralığına göre rapor
Profil görünümü, şu anda görünür olan zaman aralığını ve kanalları temel alan raporları görüntüler. Verilerin farklı alt kümelerinin ayrıntılarını görmek için göstergedeki öğeleri tıklatın.

 Tablolardaki veriler hakkında daha fazla bilgiyi burada bulabilirsiniz.
