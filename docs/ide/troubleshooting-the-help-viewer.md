# <a name="troubleshooting-the-help-viewer"></a>Yardım Görüntüleyici'de Sorun Giderme
Bu konuda Yardım Görüntüleyici ile karşılaşabileceğiniz sorunlar açıklanmaktadır.  
  
## <a name="audio-doesnt-work"></a>Ses çalışmıyor.  
 Yardım Görüntüleyicisi'ni bir ses yürütücüsü içermez. Ses içeren içerik indirmek ve seçtiğinizde hiçbir şey olmaz **Yürüt**, bir ses yürütücüsü yükleyin.  
  
## <a name="search-doesnt-work-in-windows-server-2008-windows-server-2008-with-sp1-or-windows-server-2008-r2"></a>Arama, Windows Server 2008, Windows Server 2008 SP1 veya Windows Server 2008 R2 ile çalışmaz.  
 Yüklenecek Windows Search hizmeti Yardım Görüntüleyici'de arama ve filtreleme özellikleri gerektirir ve. Varsayılan olarak, bu hizmet Windows Server 2008, Windows Server 2008 Service Pack 1 (SP1) ve Windows Server 2008 R2 ile kapalıdır.  
  
#### <a name="to-activate-windows-search-service"></a>Windows Arama hizmeti etkinleştirmek için  
  
1.  Sunucu Yöneticisi'ni başlatın.  
  
2.  Sol gezinti bölmesinde seçin **rolleri**.  
  
3.  Rol Özeti bölmesinde seçin **Rol Ekle**.  
  
4.  Dosya Hizmetleri rolünü seçin ve ardından **sonraki** düğmesi.  
  
5.  Windows Search rol hizmeti seçin.
