# <a name="localizing-applications"></a>Uygulamaları Yerelleştirme
*Yerelleştirme* uygulamanızı belirli bir kültür ya yerel ayar için özelleştirme işlemidir. Yerelleştirme, öncelikle kullanıcı arabirimini çevirmeyi içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Yerelleştirme için Kaynakların Hiyerarşik Organizasyonu](../ide/hierarchical-organization-of-resources-for-localization.md)  
 Nasıl yerelleştirilmiş kaynaklar açıklanır saklanır ve Visual Studio'da erişilir.  
  
 [Güvenlik ve Yerelleştirilmiş Yardımcı Derlemeler](../ide/security-and-localized-satellite-assemblies.md)  
 Genel-özel anahtar çiftleri ile imzalama uydu derlemelerini açıklanır.  
  
 [Ana ve Yerelleştirilmiş Yardımcı Derlemeler İçin Sürüm Numaraları](../ide/version-numbers-for-main-and-localized-satellite-assemblies.md)  
 Tanıtır <xref:System.Resources.SatelliteContractVersionAttribute> belirleyen sınıfı uydu derlemelerinin çalıştığını uygulamanın ana derleme ile.  
  
 [Yerelleştirme İçin Bağımsız Kaynak Dilleri](../ide/neutral-resources-languages-for-localization.md)  
 Tanıtır <xref:System.Resources.NeutralResourcesLanguageAttribute> uygulamanın ana derlemesinde bulunan kaynakların kültürünü belirtir sınıfı.
