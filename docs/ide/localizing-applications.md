# <a name="localizing-applications"></a>Uygulamaları Yerelleştirme

*Yerelleştirme* uygulamanızı belirli bir kültür ya yerel ayar için özelleştirme işlemidir. Yerelleştirme, öncelikle kullanıcı arabirimini çevirmeyi içerir.

## <a name="in-this-section"></a>Bu Bölümde
 [Yerelleştirme, hiyerarşik kuruluş kaynakları](../ide/hierarchical-organization-of-resources-for-localization.md) açıklar nasıl yerelleştirilmiş kaynaklar saklanır ve Visual Studio'da erişilir.

 [Güvenlik ve yerelleştirilmiş yardımcı derlemeler](../ide/security-and-localized-satellite-assemblies.md) genel-özel anahtar çiftleri ile imzalama uydu derlemelerini açıklanır.

 [Ana ve yerelleştirilmiş yardımcı derlemeler için sürüm numaraları](../ide/version-numbers-for-main-and-localized-satellite-assemblies.md) Introduces <xref:System.Resources.SatelliteContractVersionAttribute> belirleyen sınıfı uydu derlemelerinin çalıştığını uygulamanın ana derleme ile.

 [Yerelleştirme için bağımsız kaynak dilleri](../ide/neutral-resources-languages-for-localization.md) Introduces <xref:System.Resources.NeutralResourcesLanguageAttribute> uygulamanın ana derlemesinde bulunan kaynakların kültürünü belirtir sınıfı.
