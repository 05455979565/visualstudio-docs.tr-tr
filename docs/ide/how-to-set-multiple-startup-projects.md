# <a name="how-to-set-multiple-startup-projects"></a>Nasıl yapılır: Birden çok başlangıç projesi ayarlama

Visual Studio, hata ayıklamayı başlattığınızda birden fazla projenin nasıl çalıştırıldığını belirtmenize olanak tanır.

## <a name="to-set-multiple-startup-projects"></a>Birden çok başlangıç projesi ayarlamak için

1. **Çözüm Gezgini'nde,** çözümü (üst düğüm) seçin.

2. Çözüm düğümünün bağlamını (sağ tıklatma) menüsünü seçin ve ardından **Özellikler'i**seçin. **Çözüm Özelliği Sayfaları** iletişim kutusu görüntülenir.

3. Ortak **Özellikler** düğümini genişletin ve **Başlangıç Projesi'ni**seçin.

4. Çoklu **Başlangıç Projeleri** seçeneğini seçin ve uygun eylemleri ayarlayın.
