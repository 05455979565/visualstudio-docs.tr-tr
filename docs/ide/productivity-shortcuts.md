# <a name="shortcut-tips-for-visual-studio"></a>Visual Studio için kısayol ipuçları

Bu makalede kısayolları kullanarak, Visual Studio'da daha kolay gezinebilirsiniz. Bu kısayollar, klavye ve Fare kısayolları ve bunun yanı sıra daha kolay bir görevi gerçekleştirmenize yardımcı olacak girdiğiniz metin ekleyin.

Komut kısayolların tam listesi için bkz. [varsayılan klavye kısayolları](../ide/default-keyboard-shortcuts-in-visual-studio.md).

> [!NOTE]
> Bu konu, Windows üzerinde Visual Studio için geçerlidir. Mac için Visual Studio için bkz: [Mac için Visual Studio'da ortak klavye kısayollarının](/visualstudio/mac/keyboard-shortcuts).
