# <a name="how-to-build-to-a-common-output-directory"></a>Nasıl yapılır: ortak çıktı dizinine oluşturma
Varsayılan olarak, [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] kendi çözüm klasöründe çözümde her projede oluşturur. Yapı çıktı yolları projeleriniz tüm çıkışları aynı klasörde yerleştirilecek zorlamak için değiştirebilirsiniz.  
  
### <a name="to-place-all-solution-outputs-in-a-common-directory"></a>Tüm çözüm yerleştirmek için ortak bir dizini çıkarır.  
  
1.  Çözümdeki bir proje tıklayın.  
  
2.  Üzerinde **proje** menüsünde tıklatın **özellikleri**.  
  
3.  Proje türüne bağlı olarak, üzerinde tıklayın **derleme** sekmesini veya **yapı** sekmesini tıklatın ve ayarlama **çıkış yolu** Çözümdeki tüm projeleri için kullanılacak bir klasöre.  
  
4.  Çözümdeki tüm projeleri için 1-3 adımları yineleyin.
