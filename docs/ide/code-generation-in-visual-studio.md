# <a name="code-generation-in-visual-studio"></a>Visual Studio'da kod oluşturma
Visual Studio sizin için kaynak kodu bölümlerini oluşturmak yardımcı gerçek çalışma odaklanmanıza olanak tanır.  Bazı ortak kod oluşturma özellikleri içerir:  

* Sınıfları, yöntemleri, türleri, vb. oluşturuluyor.
* XML belgeleri açıklamaları oluşturma
* Soyut sınıflar veya arabirimler uygulama
* Yerel değişkene bir karmaşık ifadesine Tanıtımı

Kod oluşturma mantığı yakından dili sözdizimine bağlı olduğundan, her dil hizmeti Visual Studio'da kendi kod oluşturma yetenekleri sağlar. Daha fazla bilgi için aşağıdaki sayfalarına bakın:  
  
* [Kod oluşturma (C#)](../csharp-ide/code-generation-csharp.md)
