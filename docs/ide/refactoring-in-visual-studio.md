# <a name="refactoring-in-visual-studio"></a>Visual Studio'da yeniden düzenleme
Yeniden düzenleme kod korumak, anlamak ve genişletmek kolaylaştırmak için değiştirme ancak davranışını değiştirmeden işlemidir. Bazı ortak *yapan yeniden düzenlemeler* içerir:  
  
1.  küçük parçalara uzun işlevleri bölme  
  
2.  değişkenleri, türleri, işlevleri veya parametreleri yeniden adlandırma  
  
3.  parametreleri yeniden sıralama  
  
4.  kaldırma veya yönergeleri kullanarak yeniden sıralama  
  
 Yeniden düzenleme mantığı yakından dili sözdizimine bağlı olduğundan, her dil hizmeti Visual Studio'da yeniden düzenleme özelliklerini sağlar. Daha fazla bilgi için aşağıdaki sayfalarına bakın:
