# <a name="refactoring-code"></a>Kodu yeniden düzenleme

Yeniden düzenleme kod korumak, anlamak ve genişletmek kolaylaştırmak için değiştirme ancak davranışını değiştirmeden işlemidir.

Farklı yeniden düzenleme işlemi Visual Studio'da farklı programlama dillerinde kullanılabilir. Bu bölümdeki konular, C# ve Visual Basic için yapan yeniden düzenlemeler kullanılabilir kapsar. C++, kod, yeniden düzenleme hakkında daha fazla bilgi için bkz: [yazma ve kod (C++) yeniden düzenlemesi](/cpp/ide/writing-and-refactoring-code-cpp). F # tarafından sağlanan için destek yeniden düzenleme [Visual F # güç araçları](https://marketplace.visualstudio.com/items?itemName=FSharpSoftwareFoundation.VisualFPowerTools), üçüncü taraf Visual Studio uzantısı.
