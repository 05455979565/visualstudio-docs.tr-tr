# <a name="microsoft-help-viewer-installation"></a>Microsoft Yardım Görüntüleyicisi'ni yükleme
Birkaç ürünleri, Microsoft Yardım Visual Studio ve SQL Server dahil olmak üzere Görüntüleyici'de, Yardım içeriği görüntüleyebilirsiniz.

Yardım Görüntüleyicisi, Visual Studio'nun isteğe bağlı yükleme bileşendir. Visual Studio yükleyicisi yüklemek için aşağıdaki adımları izleyin:

1. Açık **Visual Studio yükleyicisi** Başlat menüsünden veya Visual Studio açmak varsa, seçebilirsiniz **Araçları**, **alma araçları ve özelliklerinin...**  Visual Studio yükleyicisi açın.

2. Seçin **bileşenleri tek tek** sekmesini ve ardından **Yardım Görüntüleyici** altında **kod Araçları** bölümü.

   ![VS yükleyici Yardım Görüntüleyici bileşeni](media/help_viewer_vs_installer.png)

3. Seçin **Değiştir** düğmesi Microsoft Yardım Görüntüleyici yüklemesini başlatın.

Aracılığıyla kolayca Microsoft Yardım Görüntüleyicisi'ni yüklemek için başka bir yol olduğu **hızlı başlatma** kutusunda:

1. Yazın veya girin **Yardım Görüntüleyicisi** içinde **hızlı başlatma** Visual Studio başlık çubuğunda kutusu.

   ![Hızlı Başlatma kutusu](media/help_viewer_quick_launch.png)

2. Adlı yükleme sonucu seçin **Yardım Görüntüleyicisi'ni (Indiviual bileşeni)**.

3. Açılan iletişim kutusunda seçin **yükleme** düğmesi.

   ![Yükle düğmesi](media/help_viewer_install.png)
