# <a name="installvstemplates-devenvexe"></a>/InstallVSTemplates (devenv.exe)
Bulunan öğe veya proje şablonları kaydeder  *\<Visual Studio yükleme yolu >*\Common7\IDE\ProjectTemplates\ veya  *\<Visual Studio yükleme yolu >*\Common7\IDE\ItemTemplates\ aracılığıyla erişilebilir böylece **yeni proje** ve **Yeni Öğe Ekle** iletişim kutuları.  
  
> [!WARNING]
>  Bu anahtar, yalnızca Visual Studio iş ortağı geliştirme için desteklenen ve Express sürümlerinde kullanılabilir değildir. Kullanmak için yönetici olarak devenv çalıştırmalısınız [/Setup (devenv.exe)](../../ide/reference/setup-devenv-exe.md) ve [/InstallVSTemplates (devenv.exe)](../../ide/reference/installvstemplates-devenv-exe.md) anahtarları. Daha fazla bilgi için bkz: [kullanıcı izinleri](../../ide/user-permissions-and-visual-studio.md).  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
devenv.exe /InstallVSTemplates  
```
