# <a name="convert-dialog-box"></a>Dönüştür İletişim Kutusu
**Dönüştür** iletişim kutusu oluşturma gibi eski bir, yeni bir proje oluşturmak için bir dönüştürme aracı seçmenize olanak sağlayan bir [!INCLUDE[vbprvblong](../../ide/reference/includes/vbprvblong_md.md)] proje bir Visual Basic 6 projeden. Aynı zamanda [Visual Studio Proje dönüştürücü](https://github.com/ssvaidya/VSProjectConverter) Visual Studio'nun önceki sürümleri oluşturulan projeleri yükseltme.   
  
## <a name="opening-the-convert-dialog-box"></a>Dönüştür iletişim kutusunu açma  
  
#### <a name="to-access-the-convert-dialog-box"></a>Dönüştür iletişim kutusuna erişmek için  
  
1.  Menü çubuğunda seçin **dosya**, **açık**, **dönüştürme**.  
  
     Şu anda kullanılabilir kod dönüştürme araçları listesini görürsünüz.  
  
2.  Uygulamanız için uygun olan aracı seçin.
