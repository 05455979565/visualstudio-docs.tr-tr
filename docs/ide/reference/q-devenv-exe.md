# <a name="-devenvexe"></a>/? (devenv.exe)

Her anahtarın kısa `devenv` bir açıklamasını içeren tüm anahtarları listeleyen bir ileti kutusu görüntüler.

## <a name="syntax"></a>Sözdizimi

```shell
devenv /?
```
