# <a name="-devenvexe"></a>/? (devenv.exe)

Tüm listeleyen bir ileti kutusu görüntüler `devenv` anahtarlarla her anahtarın kısa bir açıklaması.

## <a name="syntax"></a>Sözdizimi

```shell
devenv /?
```
