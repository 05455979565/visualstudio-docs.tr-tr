# <a name="options-dialog-box-environment--web-browser"></a>Seçenekler iletişim kutusu: \> Çevre Web Tarayıcısı

Hem dahili Web tarayıcısı hem de Internet Explorer için seçenekler ayarlar. Bu iletişim kutusuna erişmek için **Araçlar** menüsünde **Seçenekler'i** tıklatın, **Ortam** klasörünü genişletin ve **Web Tarayıcısı'nı**seçin.

> [!NOTE]
> Gördüğünüz iletişim kutuları ve menü komutları, etkin ayarlarınıza ve ürün sürümüne bağlı olarak Yardım menüsünde açıklanana göre farklılık gösterebilir. Ayarlarınızı değiştirmek için **Araçlar** menüsünde **Ayarlar İçe Ve Dışa Aktar'ı** seçin. Daha fazla bilgi [için, ayarları Sıfırla'ya](../environment-settings.md#reset-settings)bakın.

> [!IMPORTANT]
> Web'den belirli dosyaları veya bileşenleri açmak bilgisayarınızda kod yürütebilir.

## <a name="home-page"></a>Giriş sayfası

IDE Web Tarayıcısını açtığınızda görüntülenen sayfayı ayarlar.

## <a name="search-page"></a>Arama sayfası

Dahili Web tarayıcısı için bir Arama sayfası belirlemenize olanak tanır. Bu konum, tümleşik geliştirme ortamı (IDE) dışında başlatılan Internet Explorer örnekleri tarafından kullanılan arama sayfasından farklı olabilir.

## <a name="view-source-in"></a>Kaynağı görüntüle

İç Web tarayıcısından sayfada **Kaynak Görüntüle'yi** seçtiğinizde, web sayfasını açmak için kullanılan düzenleyiciyi ayarlar.

- **Kaynak editör** [Düzenleyicide](../../ide/writing-code-in-the-code-and-text-editor.md)kaynağı görüntülemek için seçin.

- **HTML düzenleyicisi** [HTML tasarımcısında](https://msdn.microsoft.com/Library/640043cc-3657-4677-a091-bc315e636477)kaynağı görüntülemek için seçin. Web sayfasını iki görünümden birinde düzenlemek için bu seçimi kullanın: Tasarım görünümü veya standart metin tabanlı Kaynak görünümü.

- **Dış editör** Kaynağı başka bir düzenleyicide görüntülemek için seçin. Seçtiğiniz herhangi bir düzenleyicinin (örneğin Notepad.exe) yolunu belirtin.

## <a name="internet-explorer-options"></a>Internet Explorer Seçenekleri

**Internet Properties** iletişim kutusunda Internet Explorer seçeneklerini değiştirmek için tıklatın. Bu iletişim kutusunda yapılan değişiklikler hem dahili Web tarayıcısını hem de Visual Studio IDE dışında başlatılan Internet Explorer örneklerini (örneğin, Başlat menüsünden) etkiler.

> [!NOTE]
> Visual Studio dahili Web tarayıcısını seçtiğiniz bir tarayıcıyla değiştirmek için **Gözat ile** iletişim kutusunu kullanın. Projenizdeki bir HTML dosyasının sağ tıklama veya bağlam menüsünden Gözat ile iletişim kutusuna erişebilirsiniz.
