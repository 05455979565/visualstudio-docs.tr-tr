# <a name="web-browser-environment-options-dialog-box"></a>Web Tarayıcısı, Ortam, Seçenekler İletişim Kutusu
İç Web tarayıcısı ve Internet Explorer seçeneklerini ayarlar. Bu iletişim kutusunu erişmek için tıklatın **seçenekleri** üzerinde **Araçları** menüsünde genişletin **ortam** klasörü ve select **Web tarayıcısı**.  
  
> [!NOTE]
>  Gördüğünüz iletişim kutuları ve menü komutları, etkin ayarlarınıza ve ürün sürümüne bağlı olarak Yardım menüsünde açıklanana göre farklılık gösterebilir. Ayarlarınızı değiştirmek için Araçlar menüsünden içeri ve dışarı aktarma ayarları seçin. Daha fazla bilgi için bkz: [Visual Studio IDE'yi kişiselleştirme](../../ide/personalizing-the-visual-studio-ide.md).  
  
> [!IMPORTANT]
>  Belirli dosyaları veya bileşenleri Web'den açma bilgisayarınızda kod yürütebilir.  
  
## <a name="home-page"></a>Giriş sayfası
 IDE Web tarayıcısı açtığınızda görüntülenen sayfa ayarlar.  
  
## <a name="search-page"></a>Arama sayfası
 İç Web tarayıcısı için arama sayfası belirlemenizi sağlar. Bu konum, Internet Explorer'ın tümleşik geliştirme ortamı (IDE) dışında başlatılan örnekleri tarafından kullanılan arama sayfası türünden farklı olabilir.  
  
## <a name="view-source-in"></a>İçinde kaynağı görüntüle
 Öğesini seçtiğinizde, bir Web sayfasını açmak için kullanılan Düzenleyicisi'nin ayarlar **kaynağı görüntüle** iç Web tarayıcısından sayfasında.  
  
-   **Kaynak Düzenleyici** kaynağında görüntülemeyi seçin [Düzenleyicisi](../../ide/writing-code-in-the-code-and-text-editor.md).  
  
-   **HTML Düzenleyicisi** kaynağında görüntülemeyi seçin [HTML Tasarımcısı](http://msdn.microsoft.com/Library/640043cc-3657-4677-a091-bc315e636477). Web sayfasında iki görünümlerden birini düzenlemek için bu seçimi kullanın: Tasarım görünümü veya standart metin tabanlı kaynak görünümü.  
  
-   **Dış Düzenleyicisi** başka bir düzenleyicide kaynağını görüntülemek için seçin. Seçtiğiniz herhangi bir Düzenleyicisi Örneğin, Notepad.exe yolunu belirtin.  
  
## <a name="internet-explorer-options"></a>Internet Explorer Seçenekleri  
Internet Explorer'da seçeneklerini değiştirmek için tıklatın **Internet Özellikleri** iletişim kutusu. Bu iletişim kutusunda yapılan değişiklikler iç Web tarayıcısı ve Internet Explorer (örneğin, Başlat menüsünden) Visual Studio IDE dışında başlatılan örneklerini etkiler.  
  
> [!NOTE]
>  Kullanım **Gözat ile** tercih ettiğiniz bir tarayıcı ile Visual Studio iç Web tarayıcısı değiştirmek için iletişim kutusu. Projenize sağ tıklayın veya bağlam menüsünden, örneğin, bir HTML dosyası Gözat ile iletişim kutusuna erişebilirsiniz.
