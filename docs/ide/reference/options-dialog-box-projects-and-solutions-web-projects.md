# <a name="options-dialog-box-projects-and-solutions-web-projects"></a>Seçenekler İletişim Kutusu, Projeler ve Çözümler, Web Projeleri

Web projelerinin içinde geliştirme için kullanacağı [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]web sunucusunu ayarlar. Bu seçeneklere erişmek için Araçlar > **Seçenekleri'ni**seçin ve **Web Projeleri'ni**seçin. **Tools > Options**

Varsayılan olarak, Visual Studio'da bir web projesi çalıştırmak Visual Studio Development Server'ı kullanır. Daha fazla bilgi için, [ASP.NET Web Projeleri için Visual Studio'daki Web Sunucuları'na](/previous-versions/aspnet/58wxa9w5\(v\=vs.120\))bakın.

## <a name="settings"></a>Ayarlar

**Web siteleri ve projeler için IIS Express'in 64 bit sürümünü kullanma**

Visual Studio Development Server yerine IIS Express'i kullanmak için bu seçeneği belirleyin. Daha fazla bilgi için IIS Express ve [IIS Express Genel](/iis/extensions/introduction-to-iis-express/iis-express-overview) [Bakış'a](https://weblogs.asp.net/scottgu/introducing-iis-express) bakın.

**Hata listesinde hatalar olduğunda web uygulamalarını çalıştırmadan önce uyar**

Bu seçenek ayarlanırsa, web uygulamanızı hatasız derlemediğinde çalıştırmayı denerseniz uyarılılır.
