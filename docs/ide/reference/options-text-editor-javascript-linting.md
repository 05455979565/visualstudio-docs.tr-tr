# <a name="options-text-editor-javascript-linting"></a>Seçenekler, metin düzenleyici, JavaScript, Linting

Kullanım **Linting** sayfasının **seçenekleri** Kod Düzenleyicisi'nde kod çözümleme seçeneklerini ayarlamak için iletişim kutusu. Menü çubuğunda, bu sayfaya erişmek için seçin **Araçları** > **seçenekleri**ve ardından **metin düzenleyici** > **JavaScript**  >  **Linting**.

## <a name="eslint-settings"></a>ESLint ayarları

Bu seçenekler, statik JavaScript kod analizini etkinleştirme ve hangi dosyaların analiz seçin olanak tanır. ESLint hakkında daha fazla bilgi için bkz: [ESLint.org](https://eslint.org/).

### <a name="uielement-list"></a>UIElement listesi

|Seçenek|Açıklama|
|------------|-----------------|
|**Eslint'i etkinleştir**|Bu seçenek belirlendiğinde, statik kod çözümleme için Kod Düzenleyicisi sağlar.|
|**Eklenen tüm dosyalar projesinde, hatta lint kapalı dosyaları**|Bu seçenek belirlendiğinde, tanılama için açık dosyaların yalnızca bildirilen sürece kapalı dosyalar incelenir.|

## <a name="global-eslint-config-options"></a>Genel Eslint'i yapılandırma seçenekleri

Bu seçenek genel Eslint'i yapılandırma dosyasının konumunu kopyalama sağlar. Ayrıca, daha önce konumunu değiştirdiyseniz, dosyanın varsayılan konumuna sıfırlayabilirsiniz.
