# <a name="options-dialog-box-text-editor--javascripttypescript--linting"></a>Seçenekler iletişim kutusu: \> Metin DüzenleyicijavaScript/TypeScript \> Linting

Kod Düzenleyicisi'ndeki kodu çözümlemek için seçenekleri ayarlamak için **Seçenekler** iletişim kutusunun **Linting** sayfasını kullanın. Bu sayfaya erişmek için menü çubuğunda **Araçlar** > **Seçenekleri'ni**seçin ve ardından **Text Editor** > **JavaScript/TypeScript** > **Linting'i genişletin.**

## <a name="eslint-settings"></a>ESLint Ayarları

Bu seçenekler statik JavaScript ve TypeSCript kod çözümlemesi etkinleştirmek ve hangi dosyaların analiz edilir seçmenize olanak sağlar. ESLint hakkında daha fazla bilgi için [ESLint.org.](https://eslint.org/)

### <a name="uielement-list"></a>UIElement listesi

|Seçenek|Açıklama|
|------------|-----------------|
|**ESLint'i etkinleştirme**|Bu seçenek seçildiğinde, Kod Düzenleyicisi kod üzerinde statik çözümleme sağlar.|
|**Projeye dahil olan tüm dosyaları, hatta kapalı dosyaları**|Bu seçenek seçildiğinde, tanılama yalnızca açık dosyalar için bildirilmediği sürece kapalı dosyalar çözümlenir.|

## <a name="global-eslint-config-options"></a>Global ESLint Config Seçenekleri

Bu seçenek, genel ESLint yapılandırma dosyasının konumunu kopyalamanızı sağlar. Ayrıca, konumu daha önce değiştirdiyseniz, dosyayı varsayılan konumuna sıfırlayabilirsiniz.
