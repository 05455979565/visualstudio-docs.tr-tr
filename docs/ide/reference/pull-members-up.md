# <a name="pull-members-up"></a>Üyeleri çekme

Bu yeniden düzenleme için geçerlidir:

- C#

- Visual Basic

**Ne:** Temel türü kadar üyeleri çekme sağlar.

**ne zaman:** Bir arabirim uygulanmıştır ve temel türün üyesi taşımak istediğiniz.

**Neden:** Üyeleri çekme Arabiriminizin bu üyeler de devralmak için diğer uygulamaları sağlar.

## <a name="how-to"></a>Nasıl Yapılır Konuları

1. İmlecinizi, herhangi bir üyesi uygulanan bir arabirim yerleştirin.
2. Tuşuna **Ctrl**+**.** Tetikleyici için **hızlı Eylemler ve yeniden düzenlemeler** menüsü.

   ![Üyeleri çekme](media/pull-members-up.png)

2. Seçin **temel türü kadar üyeleri çekme**.

3. İletişim kutusunda, seçili arabirimi eklemek istediğiniz üyeleri seçin.

   ![Üye çekme](media/pull-members-up-dialog.png)

4. **Tamam**’ı seçin. Seçilen üyeleri arabirimine alınır.

   ![Üye tamamlanmış çekme](media/pull-members-up-completed.png)
