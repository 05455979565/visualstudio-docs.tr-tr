# <a name="pull-members-up"></a>Üyeleri yukarı çekme

Bu yeniden düzenleme aşağıdakiler için geçerlidir:

- C#

- Visual Basic

**Ne:** Üyeleri taban türüne çekmenizi sağlar.

**Ne zaman:** Bir arabirim uyguladınız ve bir üyeyi temel türe taşımak istiyorsunuz.

**Neden:** Üyeleri yukarı çekmek, arabiriminizin diğer uygulamalarını da bu üyeleri devralmaya olanak tanır.

## <a name="how-to"></a>Nasıl yapılır

1. İmlecinizi uygulanan arabirimin herhangi bir üyesine yerleştirin.
2. **Ctrl**+tuşuna**basın.** **Hızlı Eylemler ve Refactorings** menüsünü tetiklemek için.

   ![Üyeleri Yukarı Çek](media/pull-members-up.png)

2. **Üyeleri taban türüne kadar çek'i**seçin.

3. İletişim kutusunda, seçili arabirime hangi üyeleri eklemek istediğinizi seçin.

   ![Üyeyi Yukarı Çek](media/pull-members-up-dialog.png)

4. **Tamam'ı**seçin. Seçilen üyeler arayüze çekilir.

   ![Üyeyi Yukarı Çek tamamlandı](media/pull-members-up-completed.png)
