# <a name="useenv-devenvexe"></a>/UseEnv (devenv.exe)
Başlatır [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] ve ortam değişkenlerini içine yükler **VC ++ dizinleri** iletişim kutusu.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
Devenv /useenv  
```  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek başlatır [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] ve ortam değişkenleri içine yükler **VC ++ dizinleri** iletişim kutusu.  
  
```  
Devenv.exe /useenv  
```
