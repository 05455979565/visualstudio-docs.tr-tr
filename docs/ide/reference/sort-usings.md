# <a name="sort-usings"></a>Kullanımları sıralama

Bu yeniden düzenleme için geçerlidir:

- C#

**Ne:** Kullanımları sıralama.

**ne zaman:** Sıralamak istediğinizi `using` dosyanızın üst kısmındaki yönergeleri alfabetik sırada olmasını sağlamak. 

**Neden:** Kullanarak bir bulmayı kolaylaştırır yönergesi.

## <a name="how-to"></a>Nasıl Yapılır Konuları

1. Seçin **Düzenle** menü çubuğundan.
2. Seçin **IntelliSense** > **kullanımları sıralama**.

   ![Kullanımları sıralama](media/sort-usings.png)
