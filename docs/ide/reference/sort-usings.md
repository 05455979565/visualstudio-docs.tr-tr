# <a name="sort-usings"></a>Kullanımları sıralama

Bu yeniden düzenleme aşağıdakiler için geçerlidir:

- C#

- Visual Basic

**Ne:** Kullanımı sıralayın.

**Ne zaman:** Yönergeleri dosyanızın `using` en üstünde, alfabetik sıraya göre sırayla sıralamak istiyorsunuz. 

**Neden:** Kullanarak bir yönerge bulmayı kolaylaştırır.

## <a name="how-to"></a>Nasıl yapılır

1. Menü çubuğundan **Edit'i** seçin.
2. **Intellisense** > **Sıralama Kullanımı'nı**seçin.

   ![Kullanımları sıralama](media/sort-usings.png)
