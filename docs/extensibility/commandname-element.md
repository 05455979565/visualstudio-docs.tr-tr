# <a name="commandname-element"></a>CommandName öğesi
`CommandName` Öğesi klavye kategorisinde görüntülenen metni belirtir **seçenekleri** iletişim kutusu hem de **komutları** listesinde **Özelleştir** iletişim bir kutu.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
<CommandName>MyCommand</CommandName>  
```  
  
## <a name="attributes-and-elements"></a>Öznitelikler ve Öğeler  
 Öznitelikler, alt ve üst öğeler aşağıdaki bölümlerde açıklanmaktadır.  
  
### <a name="attributes"></a>Öznitelikler  
 Yok.  
  
### <a name="child-elements"></a>Alt Öğeler  
 Yok.  
  
### <a name="parent-elements"></a>Üst Öğeler  
  
|Öğe|Açıklama|  
|-------------|-----------------|  
|[Dizeleri öğesi](../extensibility/strings-element.md)|Metin öğeleri gibi gruplar `ButtonText` ve `CommandName`.|
