# <a name="extensibility-in-visual-studio"></a>Visual Studio genişletilebilirlik
Visual Studio genişletilebilirlik seçenekleri büyük bir sağlar. Kendi SDK'ları oluşturmak, Windows, Windows Phone ve Azure (hangi Visual Studio bir parçası yüklenir) SDK'ları kullanın ve Visual Studio kendisini genişletir.  
  
## <a name="extend-visual-studio"></a>Visual Studio’yu Genişletme  
 Visual Studio neredeyse her bölümünü genişletmek için Visual Studio SDK'yı kullanabilirsiniz: komutlar, menüler, windows, düzenleyiciler ve projeleri. Daha fazla bilgi bulmak için bkz: [Visual Studio SDK](../extensibility/visual-studio-sdk.md).  
  
## <a name="create-your-own-sdks"></a>Kendi SDK'ları oluşturma  
 Oluşturma, paket ve kendi platform ve uzantı SDK'ları dağıtma öğrenin: [bir yazılım geliştirme seti oluşturma](../extensibility/creating-a-software-development-kit.md).  
  
## <a name="roslyn-extensibility"></a>Roslyn genişletilebilirliği  
 Genişletilebilirlik için Roslyn kullanmayı öğrenin: [.NET derleyici Platformu ("Roslyn") genişletilebilirlik](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).
