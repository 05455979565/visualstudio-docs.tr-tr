# <a name="idebugmodopt"></a>IDebugModOpt
Hata ayıklama isteğe bağlı değiştiriyi temsil eder.

## <a name="syntax"></a>Sözdizimi

```
IDebugModOpt : IUnknown
```

## <a name="notes-for-callers"></a>Arayanlar İçin Notlar
 Bir sınıfı veya yöntemi temsil eden bir [IDebugField](../../../extensibility/debugger/reference/idebugfield.md) nesnesinden elde edilir.

## <a name="methods"></a>Yöntemler
 Bu arabirim aşağıdaki yöntemi uygular:

|Yöntem|Açıklama|
|------------|-----------------|
|[GetModOpts](../../../extensibility/debugger/reference/idebugmodopt-getmodopts.md)|İsteğe bağlı değiştiriciler listesini alır.|
