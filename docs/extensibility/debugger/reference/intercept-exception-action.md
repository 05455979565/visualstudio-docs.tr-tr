# <a name="intercept_exception_action"></a>INTERCEPT_EXCEPTION_ACTION
Özel durumları keserken hangi eylemlerin gerçekleşip gerçekleşılacağını belirtir.

## <a name="syntax"></a>Sözdizimi

```cpp
enum enum_INTERCEPT_EXCEPTION_ACTION
{
    IEA_INTERCEPT = 0x0001
}
typedef DWORD INTERCEPT_EXCEPTION_ACTION;
```

```csharp
public enum enum_INTERCEPT_EXCEPTION_ACTION
{
    IEA_INTERCEPT = 0x0001
}
```

## <a name="parameters"></a>Parametreler

`IEA_INTERCEPT`\
Geçerli özel durumu engellemeyi sağlar. Bu, şu anda desteklenen tek değerdir ve belirtilmelidir.

## <a name="remarks"></a>Açıklamalar
Bu değerler [InterceptCurrentException](../../../extensibility/debugger/reference/idebugstackframe3-interceptcurrentexception.md) yöntemine aktarılır.
