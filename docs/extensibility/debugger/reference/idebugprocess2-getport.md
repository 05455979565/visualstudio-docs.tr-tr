# <a name="idebugprocess2getport"></a>IDebugProcess2::GetPort
İşlemin devam ettiği bağlantı noktasını alır.

## <a name="syntax"></a>Sözdizimi

```cpp
HRESULT GetPort( 
   IDebugPort2** ppPort
);
```

```csharp
int GetPort( 
   out IDebugPort2 ppPort
);
```

## <a name="parameters"></a>Parametreler
`ppPort`\
[çıkış] İşlemin başlatıldığı bağlantı noktasını temsil eden bir [IDebugPort2](../../../extensibility/debugger/reference/idebugport2.md) nesnesi döndürür.

## <a name="return-value"></a>Dönüş Değeri
 Başarılı olursa, `S_OK`döner; aksi takdirde, bir hata kodu döndürür.
