# <a name="idebugobjectsetreferencevalue"></a>IDebugObject::SetReferenceValue
Bu nesnenin başvuru değerini ayarlar.

## <a name="syntax"></a>Sözdizimi

```cpp
HRESULT SetReferenceValue( 
   IDebugObject* pObject
);
```

```csharp
int SetReferenceValue(
   [In] IDebugObject pObject
);
```

## <a name="parameters"></a>Parametreler
`pObject`\
[içinde] Yeni başvuru değerini temsil eden bir [IDebugObject](../../../extensibility/debugger/reference/idebugobject.md) nesnesi.

## <a name="return-value"></a>Dönüş Değeri
 Başarılı olursa, S_OK döndürür; aksi takdirde, bir hata kodu döndürür.

## <a name="remarks"></a>Açıklamalar
 Bu yöntem, bu [IDebugObject](../../../extensibility/debugger/reference/idebugobject.md) nesnesi `pObject` parametre verilen nesnenin değerine bir başvuru yapar, önceki herhangi bir başvuru atarak. Bu `IDebugObject` nesnenin zaten bir başvuru türü olması gerektiğini unutmayın.
