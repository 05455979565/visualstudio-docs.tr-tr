# <a name="idebugproperty2getparent"></a>IDebugProperty2::GetParent
Bir mülkün ana özelliğini alır.

## <a name="syntax"></a>Sözdizimi

```cpp
HRESULT GetParent ( 
   IDebugProperty2** ppParent
);
```

```csharp
int GetParent ( 
   out IDebugProperty2 ppParent
);
```

## <a name="parameters"></a>Parametreler
`ppParent`\
[çıkış] Özelliğin üst temsil eden bir [IDebugProperty2](../../../extensibility/debugger/reference/idebugproperty2.md) nesnesi döndürür.

## <a name="return-value"></a>Dönüş Değeri
 Başarılı olursa, `S_OK`döner; aksi takdirde hata kodu döndürür. Ebeveyn `S_GETPARENT_NO_PARENT` yoksa döndürür.
