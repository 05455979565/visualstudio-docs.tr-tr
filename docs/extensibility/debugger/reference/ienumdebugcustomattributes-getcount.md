# <a name="ienumdebugcustomattributesgetcount"></a>IEnumDebugCustomAttributes::GetCount
Bir numaralandırmada özel özniteliklerin sayısını alır.

## <a name="syntax"></a>Sözdizimi

```cpp
HRESULT GetCount( 
   ULONG* pcelt
);
```

```csharp
int GetCount(
   out uint pcelt
);
```

## <a name="parameters"></a>Parametreler
`pcelt`\
[çıkış] Numaralandırmadaki eleman sayısını verir.

## <a name="return-value"></a>Dönüş Değeri
 Başarılı olursa, `S_OK`döner; aksi takdirde, bir hata kodu döndürür.

## <a name="remarks"></a>Açıklamalar
 Bu yöntem, `Next`yalnızca , , `Clone`, `Skip`ve `Reset` uygulanması gerektiğini belirten alışılmış COM numaralandırma arabiriminin bir parçası değildir.
