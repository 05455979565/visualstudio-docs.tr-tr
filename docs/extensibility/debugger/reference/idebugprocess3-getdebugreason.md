# <a name="idebugprocess3getdebugreason"></a>IDebugProcess3::GetDebugReason
Bu yöntem, işlemin hata ayıklama için başlatılmasının nedenini döndürür.

## <a name="syntax"></a>Sözdizimi

```cpp
HRESULT GetDebugReason(
   DEBUG_REASON* pReason
);
```

```csharp
int GetDebugReason(
   out enum_DEBUG_REASON pReason
);
```

## <a name="parameters"></a>Parametreler
`pReason`\
[çıkış] [Numaralandırma DEBUG_REASON](../../../extensibility/debugger/reference/debug-reason.md) bir değer verir.

## <a name="return-value"></a>Dönüş Değeri
 Başarılı olursa, `S_OK`döner; aksi takdirde, hata kodu döndürür.
