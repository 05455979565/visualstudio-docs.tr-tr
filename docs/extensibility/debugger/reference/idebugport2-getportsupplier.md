# <a name="idebugport2getportsupplier"></a>IDebugPort2::GetPortSupplier
Bu bağlantı noktası için bağlantı noktası tedarikçisi alır.

## <a name="syntax"></a>Sözdizimi

```cpp
HRESULT GetPortSupplier( 
   IDebugPortSupplier2** ppSupplier
);
```

```csharp
int GetPortSupplier( 
   out IDebugPortSupplier2 ppSupplier
);
```

## <a name="parameters"></a>Parametreler
`ppSupplier`\
[çıkış] Bir [IDebugPortSupplier2](../../../extensibility/debugger/reference/idebugportsupplier2.md) nesnesi döndürür bağlantı noktası için bağlantı noktası tedarikçisini temsil eder.

## <a name="return-value"></a>Dönüş Değeri
 Başarılı olursa, `S_OK`döner; aksi takdirde, bir hata kodu döndürür.
