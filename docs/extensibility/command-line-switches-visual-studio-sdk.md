# <a name="command-line-switches-visual-studio-sdk"></a>Komut hattı anahtarları (Visual Studio SDK)
Entegre geliştirme ortamını [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] (IDE) başlatan çalıştırılabilir *Devenv.exe,* özellikle VSPackages'ın geliştirilmesi için komut satırı anahtarlarını destekler. VSPackages ayrıca Devenv komut satırında kendi anahtarlarını kullanılabilir hale getirebilir. Aşağıdaki konular IDE'de komut satırı anahtarlarının nasıl kullanılacağı hakkında bilgi sağlar.

## <a name="in-this-section"></a>Bu bölümde
- [VSPackage Ddevelopment için Devenv komut satırı anahtarları](../extensibility/devenv-command-line-switches-for-vspackage-development.md) VSPackage geliştirme, hata ayıklama ve dağıtımı kolaylaştırmak için kullanabileceğiniz Devenv komut satırı anahtarlarını açıklar.

- [Komut satırı anahtarları ekleme](../extensibility/adding-command-line-switches.md) Paketiniz için kullanıcı tarafından girilen komut satırı anahtarlarını nasıl uygulayacağınızı, arayacağınızı ve okuyacağınızı açıklar.

## <a name="related-sections"></a>İlgili bölümler
- [Devenv komut satırı anahtarları](../ide/reference/devenv-command-line-switches.md) Devenv'in IDE için çeşitli seçenekler ayarlamanıza nasıl izin verir ve komut satırından projeleri nasıl derleyebilir, oluşturup ayıklaabileceğinizi açıklar. Ayrıca, Devenv anahtarları sözdizimi ve anahtarları bir tablo hakkında bilgi vardır.
