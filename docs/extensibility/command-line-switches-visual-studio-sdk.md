# <a name="command-line-switches-visual-studio-sdk"></a>Komut satırı anahtarları (Visual Studio SDK)
*Devenv.exe*, başlayan yürütülebilir [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] tümleşik geliştirme ortamı (IDE), özellikle VSPackage geliştirme için komut satırı anahtarları destekler. VSPackage de kendi anahtarlar kullanılabilir Devenv komut satırında yapabilirsiniz. Aşağıdaki konular IDE içinde komut satırı anahtarları kullanma hakkında bilgi sağlar.

## <a name="in-this-section"></a>Bu bölümde
- [VSPackage geliştirme için Devenv komut satırı anahtarları](../extensibility/devenv-command-line-switches-for-vspackage-development.md) VSPackage geliştirme, hata ayıklama ve dağıtım sürecini basitleştirmek için kullanabileceğiniz Devenv komut satırı anahtarları açıklar.

- [Komut satırı anahtarları ekleme](../extensibility/adding-command-line-switches.md) uygulamak için arayın ve kullanıcı tarafından girilen komut satırı anahtarları paketiniz için okuma işlemini açıklamaktadır.

## <a name="related-sections"></a>İlgili bölümler
- [Devenv komut satırı anahtarları](../ide/reference/devenv-command-line-switches.md) nasıl IDE ve nasıl, derleyebilir, çeşitli seçenekleri ayarladığınız Devenv sağlar derleme ve hata ayıklama, komut satırından projeleri açıklar. Ayrıca, söz dizimi Devenv anahtarları ve anahtarlar bir tablo hakkında bilgi yok.
