# <a name="net-compiler-platform-package-version-reference"></a>.NET derleyici platformu paket sürüm başvurusu

Aşağıdaki tabloda Visual Studio'nun farklı sürümleri için [.NET derleyici platformu (Roslyn) paket](https://www.nuget.org/packages/Microsoft.Net.Compilers/) sürümlerinin desteklendiği gösterilmektedir.

Örnek olarak, özel çözümleyicinizin Visual Studio 2017'nin tüm sürümlerinde çalıştığından emin olmak için Microsoft.Net.Compilers'ın 2.0 sürümünü hedeflemelidir.

| Roslyn paket sürümü | Minimum desteklenen Visual Studio sürümü |
| - | - |
| 3.x | Visual Studio 2019 |
| 2.10.0 | Visual Studio 2017 sürümü 15.9 |
| 2.9.0 | Visual Studio 2017 sürümü 15.8 |
| 2.8.2 | Visual Studio 2017 sürüm 15.7 Sürüm Notları |
| 2.7.0 | Visual Studio 2017 sürüm 15.6 |
| 2.6.1 | Visual Studio 2017 sürüm 15.5 |
| 2.4.0 | Visual Studio 2017 sürüm 15.4 |
| 2.3.2 | Visual Studio 2017 sürüm 15.3 |
| 2.2.0 | Visual Studio 2017 sürüm 15.2 |
| 2.1.0 | Visual Studio 2017 sürüm 15.1 |
| 2.0.0 | Visual Studio 2017 RTM |
| 1.3.2 | Visual Studio 2015 güncelleme 3 |
| 1.2.2 | Visual Studio 2015 güncelleme 2 |
| 1.1.1 | Visual Studio 2015 güncelleme 1 |
| 1.0.1 | Visual Studio 2015 RTM |

> [!TIP]
> Minimum desteklenen Visual Studio sürümünün Visual Studio 2017 sürümü olduğu Roslyn paketleri için Visual Studio 2019'un tüm sürümleri de daha sonra geldikleri için desteklenir.
