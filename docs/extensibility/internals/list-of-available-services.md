# <a name="list-of-available-services"></a>Kullanılabilir Hizmetlerin Listesi

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]ve Visual Studio SDK aşağıdaki hizmetleri destekler. Bazı paketler burada listelenmemiş kendi hizmetlerini sunar—örneğin, dil hizmetlerinin tek bir hizmet GUID'i yoktur. Kayıt defterinde dil hizmetinin GUID'ini bulmak için dilin adını kullanmanız gerekir.

Her hizmette gösterilen birincil arabirimi veya arabirimleri elde etmek için burada listelenen veya başka bir kaynaktan (örneğin, dil hizmetleri) alınan hizmet GUID'lerini kullanın.
