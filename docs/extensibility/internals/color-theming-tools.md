# <a name="color-theme-tools"></a>Renk tema araçları
Visual Studio SDK, renk temaları oluşturmanıza ve yönetmenize yardımcı olacak çeşitli araçlar sağlar.

- [VSIX Color Editor](../../extensibility/internals/vsix-color-editor.md) aracı Visual Studio için özel renkler oluşturabilir ve düzenleme yapabilir.

- [VSIX Color Compiler](../../extensibility/internals/vsix-color-compiler.md) aracı, mevcut Visual Studio tema *.xml* dosyalarını uzantınız için *.pkgdef* dosyalarına dönüştürür.
