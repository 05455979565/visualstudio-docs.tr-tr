# <a name="get-started-with-source-control-plug-ins"></a>Kaynak kontrol eklentileri ile başlayın
Kaynak denetimi eklentisi oluşturmak için, Kaynak Denetimi Eklentisi API'sinde tanımlanan işlevleri uygulayan bir DLL oluşturmanız [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] ve ardından Kaynak Kodu sürüm denetiminde kullanıma sunulması için DLL'yi kaydetmeniz gerekir.

 Kaynak Denetimi Eklentisi API'sinin üç sürümü (sürüm 1.1, 1.2 ve 1.3) kaynak denetim eklentileri için kullanılabilir. Burada belgelenen Kaynak Denetim Eklentisi API sürüm 1.3'dür. 1.1 ve 1.2 sürümlerini destekleyen kaynak kontrol eklentileri ile tam uyumlu olacak şekilde tasarlanmıştır. [Kaynak Denetimi Eklentisi API Sürüm 1.3 bölümündeki yenilikler,](../../extensibility/internals/what-s-new-in-the-source-control-plug-in-api-version-1-3.md) Kaynak Denetimi Eklentisi API'sinin en son sürümünde desteklenen yeni özellikleri ayrıntılarıyla anlatır.
