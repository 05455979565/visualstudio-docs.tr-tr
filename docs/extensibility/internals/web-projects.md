# <a name="web-projects"></a>Web projeleri
Bir Web projesi sistemi Web projeleri oluşturan bir proje sistemidir. Web projeleri, Web uygulamaları sırayla oluşturun. Geleneksel programlama dilleri gibi kullanarak bir Web sayfası program [!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)] veya [!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)]toplamak ve kullanıcıdan bilgi işlem, bir veritabanında depolamak ve benzeri için. Web sayfaları ile ilgili kod akıllı Web sayfaları denir.
