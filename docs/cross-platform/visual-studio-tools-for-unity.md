# <a name="visual-studio-tools-for-unity"></a>Unity için Visual Studio Araçları

![Unity için Visual Studio Araçları](media/vstu_header.png)

Visual Studio Tools for Unity, Visual Studio'yı Unity ile platformlar arası oyunlar ve uygulamalar geliştirmek için güçlü bir araca dönüştüren ücretsiz bir Visual Studio uzantısıdır.

Unity editörü oyun dünyanızı bir araya getirmek için harika olsa da, kodunuzu yazamazsınız. Visual Studio Tools for Unity ile Visual Studio'nun tanıdık kod düzenleme, hata ayıklama ve üretkenlik özelliklerini kullanarak C#'ı kullanarak Unity projeniz için editör ve oyun komut dosyaları oluşturabilir ve Visual Studio'nun güçlü hata ayıklama özelliklerini kullanarak bunları hata ayıklayabilirsiniz.

Ama Birlik için Visual Studio Tools bundan daha fazlasıdır; ayrıca, basit görevleri yapmak için ileri geri geçiş yapmak için daha az zaman harcayacak, Birliğe özel üretkenlik geliştirmeleri sağlar ve Unity belgelerini parmaklarınızın ucuna getirir.

## <a name="overview"></a>Genel Bakış

Visual [Studio geliştirme](https://visualstudio.microsoft.com/vs/unity-tools/) geliştirme için Visual Studio'yu kullanmanın avantajlarına genel bir bakış için Visual Studio ile Birlik Oyunları Oluştur sayfasını ziyaret edin veya Visual Studio Tools for Unity'nin temel özelliklerini daha derinlemesine araştırmak için aşağıda devam edin:

### <a name="compatible-with-visual-studio-community-and-bundled-with-unity"></a>Visual Studio Topluluğu ile uyumlu ve Unity ile birlikte

[Visual Studio Community](https://visualstudio.microsoft.com/) ücretsiz olarak mevcuttur ve Unity 2018.1 ile başlayan Unity yüklemeleri ile birlikte sunulmaktadır. Daha fazla bilgi için [belgelere başlarken](getting-started-with-visual-studio-tools-for-unity.md) Unity için Visual Studio Tools'a bakın.

### <a name="intellisense-for-unity-messages"></a>Birlik mesajları için IntelliSense

IntelliSense kod tamamlama, parametreleri de dahil olmak üzere `OnCollisionEnter` [Unity API iletilerinin](using-visual-studio-tools-for-unity.md#intellisense-for-unity-api-messages) hızlı ve kolay bir şekilde uygulanmasını sağlar.

### <a name="superior-debugging"></a>Üstün hata ayıklama

Visual Studio Tools for Unity, Visual Studio'dan beklediğiniz sağlam [hata ayıklama](using-visual-studio-tools-for-unity.md#unity-debugging) özelliklerini destekler:

* Koşullu kesme noktaları da dahil olmak üzere kesme noktaları ayarlayın.
* İzleme penceresindekarmaşık ifadeleri değerlendirin.
* Değişkenlerin ve bağımsız değişkenlerin değerini inceleyin ve değiştirin.
* Karmaşık nesnelere ve veri yapılarına ayrıntılı bilgi verin.

### <a name="roslyn-diagnostics-and-quick-fixes-for-unity"></a>Birlik için Roslyn tanılama ve hızlı düzeltmeler

Visual Studio'nun Unity projelerini derinlemesine anlaması yla en iyi uygulamaları yakalayan daha iyi kodyazın.
