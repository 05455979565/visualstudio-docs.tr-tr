# <a name="visual-studio-tools-for-apache-cordova"></a>Apache Cordova için Visual Studio Araçları
İOS, Android ve Windows çalıştıran cihazlar ve yaklaşık %100 kodunuzun paylaşan uygulamalar oluşturun. Bunları, sık kullanılan web teknolojileri ile oluşturun. Apache Cordova bir telefon kamerayı gibi cihaz özellikleri için yerel erişim sağlar ve en önemlisi, ücretsizdir. Visual Studio uygulamanızı oluşturmaya daha kolay hale getirmek için bu ek özellikler ekler:  
  
-   Kolay yükleme ve Kurulum.  
  
-   Bul ve sorunları gidermenize yardımcı olması için Gelişmiş hata ayıklayıcı'ı seçin.  
  
-   Aygıt benzetmeleri kolay erişim.  
  
-   Cihaz eklentilerinizi yönetmek için bir tasarımcı.  
  
-   Komut satırı arabirimi ile tamamen uyumlu olan bir proje.  
  
 Bu araçları düşünüyorsanız, ancak bunları deneyin henüz ziyaret etmek için hazır değil bizim [özellikleri](https://www.visualstudio.com/explore/cordova-vs) sayfası, belirleriz, bunları hakkında daha fazla.  
  
 Başlatmaya hazırsınız, ziyaret bizim [Web sitesi](http://taco.visualstudio.com/en-us/docs/get-started-vs-tools-apache-cordova/).
