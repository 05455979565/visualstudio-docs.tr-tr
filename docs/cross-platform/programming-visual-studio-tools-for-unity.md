# <a name="program-visual-studio-tools-for-unity"></a>Unity için Visual Studio Araçları’nı Programlama
Bu bölümde, Unity API için Visual Studio Araçlarını kullanmak için örnekler bulacaksınız.

## <a name="examples"></a>Örnekler
 Burada, Unity API'leri için Visual Studio Araçlarını nasıl kullanabileceğinizi gösteren bazı örnekler verilmiştir.

### <a name="customize-project-files-created-by-vstu"></a>VSTU tarafından oluşturulan proje dosyalarını özelleştirme
 Visual Studio Tools for Unity, proje dosyası oluşturma sırasında Unity tarzı bir geri arama sağlar. Proje dosyasını yeniden oluşturulduğunda nasıl değiştirebileceğinizi öğrenmek için [bkz.](../cross-platform/customize-project-files-created-by-vstu.md)

### <a name="share-the-unity-log-callback-with-vstu"></a>Unity günlük geri aramasını VSTU ile paylaşın
 Visual Studio Tools for Unity konsolunu Visual Studio'ya aktarabilmek için Unity ile günlük geri arama kaydeder. Düzenleyici komut dosyalarınız da Unity ile günlük geri arama sıyrık kaydederse, VSTU geri araması bu komut dosyasına engel olabilir. Unity günlük geri aramasını VSTU ile nasıl paylaşabileceğinizi öğrenmek için [bkz.](../cross-platform/share-the-unity-log-callback-with-vstu.md)
