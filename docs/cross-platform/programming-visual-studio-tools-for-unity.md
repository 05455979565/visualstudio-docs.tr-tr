# <a name="programming-visual-studio-tools-for-unity"></a>Unity için Visual Studio Araçlarını Programlama
Bu bölümde, Unity API için Visual Studio araçları kullanarak örnekleri bulabilirsiniz.

## <a name="examples"></a>Örnekler
 Aşağıda, Unity API'leri için Visual Studio Araçları nasıl kullanacağınızı gösteren bazı örnekler verilmiştir.

### <a name="customize-project-files-created-by-vstu"></a>VSTU tarafından oluşturulan proje dosyalarını özelleştirme
 Unity için Visual Studio Araçları Unity stili geri çağırma proje dosyası oluşturma sırasında sağlar. Nasıl yeniden zaman, proje dosyası değiştirebilirsiniz öğrenmek için bkz: [örnek: Proje dosyası oluşturma](../cross-platform/customize-project-files-created-by-vstu.md).

### <a name="share-the-unity-log-callback-with-vstu"></a>VSTU ile Unity günlük geri paylaşma
 Unity için Visual Studio Araçları ile Visual Studio, konsola akış yapabilmek için Unity günlük geri çağırma kaydeder. VSTU geri çağırma Düzenleyicisi komut dosyalarınızı ayrıca ile Unity günlük geri çağırma kaydeder, kendisiyle etkileyebilir. VSTU ile Unity günlük geri nasıl paylaşabilirsiniz bilgi edinmek için [örnek: günlük geri çağırma](../cross-platform/share-the-unity-log-callback-with-vstu.md).
