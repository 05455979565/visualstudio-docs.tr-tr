# <a name="xslt-profiler"></a>XSLT Profil Oluşturucusu
XSLT profil oluşturucu geliştirmek ve XSLT belgeleri hata ayıklama yardımcı olan bir performans çözümleme profil oluşturucu aracıdır. XSLT profil oluşturucu ölçmek ve performans ile ilgili sorunlar XSLT kodda ayrıntılı XSLT performans raporu oluşturarak değerlendirmek geliştiricilerin sağlar. XSLT profil oluşturucu yararlı ipuçları için en yüksek performans talep XSLT tabanlı uygulamalar için gerekli olan XSL ve XSLT stil sayfası iyileştirmeleri içerir.  
  
XSLT profil oluşturucu Visual Studio bir parçasıdır ve kullanılabilir **XML** menüsü.  
  
![XSLT profil oluşturucu](../xml-tools/media/xsltprofilermenu.gif "XSLTProfilerMenu")
  
> [!NOTE]
>  Bu konu başlıklarında göründüklerinden bu ekran görüntüleri yakalanan bu yana bazı görsel öğeler değiştiğinden bazı ekranlar, Visual Studio sürümünde farklı şekilde görüntülenir.
