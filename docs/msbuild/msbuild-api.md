# <a name="use-the-msbuild-api"></a>MSBuild API'sını kullanma

MSBuild, programınızın yapılar gerçekleştirebilmeleri ve projeleri inceleyebilmeleri için genel bir API yüzeyi sağlar.

MSBuild API için belgeler [Microsoft.Build ad alanlarında](https://msdn.microsoft.com/library/mt476050(v=vs.110).aspx)bulunabilir.
