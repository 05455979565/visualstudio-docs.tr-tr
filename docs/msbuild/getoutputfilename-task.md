# <a name="getoutputfilename-task"></a>GetOutputFileName görevi

CI için çıkış dosyasının adını ve diğer araçları yalnızca belirtmeye izin almak için görev Yardımcısı, dizin veya tam dosya adı veya hiçbir şey çıktı.

## <a name="parameters"></a>Parametreler

Parametreleri aşağıdaki tabloda açıklanmıştır **GetOutputFileName** görev.

|Parametre|Açıklama|
|---------------|-----------------|
|**OutputExtension**|Gerekli **dize** parametresi.|
|**OutputFile**|İsteğe bağlı **dize** çıkış parametresi.|
|**OutputPath**|İsteğe bağlı **dize** parametresi.|
|**Kaynakdosya**|Gerekli **dize** parametresi.|
