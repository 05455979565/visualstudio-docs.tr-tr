# <a name="initialize-correlation-dialog-box"></a>Bağıntıyı Başlat İletişim Kutusu

**Başlatma bağıntı** iletişim kutusunda iş akışı Tasarımcısı'nda düzenlemek için kullanılan <xref:System.ServiceModel.Activities.InitializeCorrelation.CorrelationData%2A> özelliği bir <xref:System.ServiceModel.Activities.InitializeCorrelation> etkinlik. Daha fazla bilgi için bkz: [InitializeCorrelation](../workflow-designer/initializecorrelation-activity-designer.md).

Kullanıcı Arabirimi (UI) öğelerini aşağıdaki tabloda açıklanmaktadır **başlatma bağıntı** iletişim kutusunda:

|Arabirim Öğesi|Açıklama|
|----------------|-----------------|
|**Bağıntı**|<xref:System.ServiceModel.Activities.CorrelationHandle> Başlatmak için bağıntı biri.|
|**Üzerinde başlatılamadı**|Başlatmak için gerekli verileri içeren bir anahtar/değer çifti. Bu değer karşılık gelen <xref:System.ServiceModel.Activities.InitializeCorrelation.CorrelationData%2A> özelliği. "OrderID adlı bir değişkende ile eşleştirilmiş OrderID" adlı bir anahtar geçerli anahtar/değer çifti örnektir.|

## <a name="to-launch-the-initialize-correlation-dialog-box"></a>Bağıntı başlatma iletişim kutusunu başlatmak için

Tıklatın **Görünüm** üzerinde **InitializeCorrelation** etkinlik Tasarımcısı veya select bir <xref:System.ServiceModel.Activities.InitializeCorrelation> etkinliği iş akışı Tasarımcısı'nda. Ardından yanındaki üç nokta düğmesini tıklayın <xref:System.ServiceModel.Activities.InitializeCorrelation.CorrelationData%2A> özellik kılavuzunda özelliği.
