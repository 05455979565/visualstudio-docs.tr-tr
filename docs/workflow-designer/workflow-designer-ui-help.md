# <a name="workflow-designer-ui-help"></a>İş Akışı Tasarımcısı kullanıcı Arabirimi Yardım

Bu bölüm Windows iş akışı Tasarımcısı için bağlama duyarlı Yardım konularını içerir. Visual Studio 2010'da çalışırken, bu konularda kullanıcı arabiriminden tuşlarına basarak erişebilirsiniz **F1** ne zaman bir iş akışı Tasarımcısı iletişim kutusunu açın.

Bu bölümde ayrıca gezinti ve hata iletileri ile ilgili çeşitli konuları içerir.
