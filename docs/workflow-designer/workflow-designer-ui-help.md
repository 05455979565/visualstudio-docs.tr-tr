Çalışırken [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)], basarak bu konularda kullanıcı arabiriminden erişebilirsiniz **F1** zaman bir [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)] iletişim kutusunu açın.
