# <a name="debugging-workflows-with-the-workflow-designer"></a>İş Akışı Tasarımcısı ile hata ayıklama

Windows iş akışı Tasarımcısı iş akışları ve özel etkinlikler hata ayıklama olanağı sağlar. İşlem ve davranışı, varsayılan Visual Studio 2010 hata ayıklayıcısı benzerdir. Bu bölümdeki konular, geleneksel yordamları hata ayıklama yanı sıra XAML ve ifadeler hata ayıklama gerçekleştirmek nasıl açıklar.
