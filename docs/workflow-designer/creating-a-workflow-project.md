# <a name="creating-a-workflow-project"></a>Bir iş akışı projesi oluşturma

İş akışları, Windows Communication Foundation (WCF) iş akışı Hizmetleri, özel etkinlikler ve özel etkinlik tasarımcıları Visual Studio Proje şablonları kullanılarak oluşturulabilir. Bu bölümde, Visual Studio Proje şablonları ile kitaplıkları ve uygulamaları oluşturmayı açıklar.

İş akışı bulunan şablonları kullanma aşağıdaki ilk dört konular açıklanmaktadır **yeni proje** penceresi. Visual Studio 2010'u açıp seçerek bu şablonları erişilebilir **dosya** > **yeni** > **proje**, sonra  **İş akışı** kategoriden **yüklü şablonlar** ya da altında sol bölümü **Visual Basic** veya **Visual C#** düğümü.

Son konu iş akışı etkinlikleri, tasarımcıları ve Hizmetleri veya diğer Visual Studio bir var olan iş akışı (veya diğer) öğelerine projesi nasıl ekleneceğini açıklar.
