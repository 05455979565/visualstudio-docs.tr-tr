# <a name="develop-apps-with-the-workflow-designer"></a>İş Akışı Tasarımcısı ile uygulamaları geliştirme

İş Akışı Tasarımcısı Visual Studio'daki [Windows İş Akışı Vakfı](/dotnet/framework/windows-workflow-foundation/index) (WF) uygulamalarının grafiksel yapısı ve hata ayıklanması için görsel bir tasarımcı ve hata ayıklayıcıdır. Şablonlar ve etkinlik tasarımcıları aracılığıyla bileşik iş akışı uygulaması, etkinlik kitaplığı veya Windows Communication Foundation (WCF) hizmeti oluşturmanıza olanak tanır.

## <a name="install-windows-workflow-foundation"></a>Windows İş Akışı Temelini Yükleme

Visual Studio'da İş Akışı proje şablonlarını kullanmak için önce **Windows İş Akışı Temeli** bileşenini yükleyin.

1. Açık Visual Studio Yükleyici. Açmak için hızlı bir yolu **Araçlar** > Al**Araçları ve Visual** Studio Özellikleri seçerek gereğidir.

1. Visual Studio Installer'da, **Bireysel bileşenler** sekmesini seçin.

1. **Geliştirme etkinlikleri** kategorisine gidin ve ardından Windows İş **Akışı Temeli** bileşenini seçin.

   ![Visual Studio için Windows İş Akışı Temel bileşeni](media/windows-workflow-foundation-component.png)

1. **Değiştir'i**seçin.

   Visual **Studio, Windows İş Akışı Temeli** bileşenini yükler.
