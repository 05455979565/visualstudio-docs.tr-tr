# <a name="using-the-workflow-designer"></a>İş Akışı Tasarımcısı'nı kullanarak
Bu bölümde yeni etkinlikler ve yerleşik tasarımcıları ve ifade kullanarak iş akışı projeleri birkaç etkinlik tasarımcıları tarafından barındırılan Düzenleyicisi'ni nasıl oluşturulacağını ayrıntılarını verir.

 [Nasıl yapılır: etkinlikler araç kutusuna ekleme](../workflow-designer/how-to-add-activities-to-the-toolbox.md)

 [Nasıl yapılır: içeri aktarmalar Tasarımcısı'nı kullanın](../workflow-designer/how-to-use-the-imports-designer.md)

 [Nasıl yapılır: içerik haritası Gezinti kullanın](../workflow-designer/how-to-use-breadcrumb-navigation.md)
