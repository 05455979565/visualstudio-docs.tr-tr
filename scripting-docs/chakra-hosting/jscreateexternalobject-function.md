# <a name="jscreateexternalobject-function"></a>JsCreateExternalObject İşlevi
Bazı dış veri depolayan yeni bir nesne oluşturur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
STDAPI_(JsErrorCode) JsCreateExternalObject(  
   _In_opt_ void *data,  
   _In_opt_ JsFinalizeCallback finalizeCallback,  
   _Out_ JsValueRef *object  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `data`  
 Nesneyi temsil edecek dış veri. Null olabilir.  
  
 `finalizeCallback`  
 Nesnenin ne zaman sonlandırılır için bir geri çağırma. Null olabilir.  
  
 `object`  
 Yeni nesne.  
  
## <a name="return-value"></a>Dönüş Değeri  
 İşlem başarılı ise `JsNoError` kodu, değilse hata kodu.  
  
## <a name="remarks"></a>Açıklamalar  
 Etkin komut dosyası bağlamı gerektiriyor.  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jsrt.h
