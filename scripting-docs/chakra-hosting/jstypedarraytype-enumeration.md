# <a name="jstypedarraytype-enumeration"></a>JsTypedArrayType Numaralandırması
Türü belirtilmiş bir JavaScript dizi türü.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
enum JsTypedArrayType;  
```  
  
## <a name="members"></a>Üyeler  
  
### <a name="values"></a>Değerler  
  
|Ad|Açıklama|  
|----------|-----------------|  
|`JsArrayTypeInt8`|İnt8 dizisi.|  
|`JsArrayTypeUint8`|Uint8 dizisi.|  
|`JsArrayTypeUint8Clamped`|Uint8 clamped dizisi.|  
|`JsArrayTypeInt16`|İnt16 dizisi.|  
|`JsArrayTypeUint16`|Uint16 dizisi.|  
|`JsArrayTypeInt32`|Int32 dizisi.|  
|`JsArrayTypeUint32`|Uint32 dizisi.|  
|`JsArrayTypeFloat32`|Float32 dizisi.|  
|`JsArrayTypeFloat64`|Bir float64 dizisi.|  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jsrt.h
