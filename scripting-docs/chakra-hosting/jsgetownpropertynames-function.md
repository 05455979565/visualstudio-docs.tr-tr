# <a name="jsgetownpropertynames-function"></a>JsGetOwnPropertyNames İşlevi
Bir nesne üzerinde tüm özelliklerinin listesini alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
STDAPI_(JsErrorCode) JsGetOwnPropertyNames(  
   _In_ JsValueRef object,  
   _Out_ JsValueRef *propertyNames  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `object`  
 Özellik adlarının alınacağı nesnesi.  
  
 `propertyNames`  
 Özellik adları dizisi.  
  
## <a name="return-value"></a>Dönüş Değeri  
 İşlem başarılı ise `JsNoError` kodu, değilse hata kodu.  
  
## <a name="remarks"></a>Açıklamalar  
 Etkin komut dosyası bağlamı gerektiriyor.  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jsrt.h
