# <a name="jsgetruntimememoryusage-function"></a>JsGetRuntimeMemoryUsage İşlevi
Geçerli bellek kullanımı için bir çalışma zamanı alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
STDAPI_(JsErrorCode) JsGetRuntimeMemoryUsage(  
   _In_ JsRuntimeHandle runtime,  
   _Out_ size_t *memoryUsage  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `runtime`  
 Alınacak, bellek kullanımı olduğu çalışma zamanı.  
  
 `memoryUsage`  
 Çalışma zamanı'nın geçerli bellek kullanımı, bayt.  
  
## <a name="return-value"></a>Dönüş Değeri  
 İşlem başarılı ise `JsNoError` kodu, değilse hata kodu.  
  
## <a name="remarks"></a>Açıklamalar  
 Bellek kullanımı her zaman hesaplanması alınan, çalışma zamanı başka bir iş parçacığı üzerinde etkin olup olmadığına bakılmaksızın.  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jsrt.h
