# <a name="jsvalueref-typedef"></a>JsValueRef Tür Tanımı
JavaScript değerine bir başvuru.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bir JavaScript değeri, şu tür değerlerden biridir: Tanımsız, Null, Boolean, Dize, Sayı veya Nesne.  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jsrt.h
