# <a name="jsconstructobject-function"></a>JsConstructObject İşlevi
Bir işlev Oluşturucusu çağırır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
STDAPI_(JsErrorCode) JsConstructObject(  
   _In_ JsValueRef function,  
   _In_reads_(argumentCount) JsValueRef *arguments,  
   _In_ unsigned short argumentCount,  
   _Out_opt_ JsValueRef *result  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `function`  
 Bir oluşturucu çağrılacak işlev.  
  
 `arguments`  
 Arama için bağımsız değişkenler.  
  
 `argumentCount`  
 İşlev için geçirilen bağımsız değişken sayısı.  
  
 `result`  
 İşlev çağrısından döndürülen değer.  
  
## <a name="return-value"></a>Dönüş Değeri  
 İşlem başarılı ise `JsNoError` kodu, değilse hata kodu.  
  
## <a name="remarks"></a>Açıklamalar  
 Etkin komut dosyası bağlamı gerektiriyor.

 **Başlık:** jsrt.h
