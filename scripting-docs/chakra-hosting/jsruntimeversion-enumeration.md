# <a name="jsruntimeversion-enumeration"></a>JsRuntimeVersion Numaralandırması
Çalışma zamanı sürümü.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
enum JsRuntimeVersion;  
```  
  
## <a name="members"></a>Üyeler  
  
### <a name="values"></a>Değerler  
  
|Ad|Açıklama|  
|----------|-----------------|  
|`JsRuntimeVersion10`|Çalışma zamanı ıe10 sürümüyle oluşturun.|  
|`JsRuntimeVersion11`|Çalışma zamanı IE11 sürümüyle oluşturun.|  
|`JsRuntimeVersionEdge`|Çalışma zamanı en yüksek sürümü çalışma zamanında makinedeki mevcut oluşturun.|  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jsrt.h
