# <a name="jscreatedataview-function"></a>JsCreateDataView İşlevi
Bir Javascript oluşturur `DataView` nesnesi.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
JsErrorCode  JsCreateDataView(  
   _In_ JsValueRef arrayBuffer,  
   _In_ unsigned int byteOffset,  
   _In_ unsigned int byteLength,  
   _Out_ JsValueRef *result  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `arrayBuffer`  
 Var olan `ArrayBuffer` sonuç için depolama alanı olarak kullanılacak nesne `DataView` nesne.  
  
 `byteOffset`  
 Bayt başından uzaklık `arrayBuffer` sonucu için `DataView` başvuru.  
  
 `byteLength`  
 DataView başvuru sonucu ArrayBuffer bayt sayısı.  
  
 `result`  
 Yeni DataView nesnesi.  
  
## <a name="return-value"></a>Dönüş Değeri  
 İşlem başarılı ise `JsNoError` kodu, değilse hata kodu.  
  
## <a name="remarks"></a>Açıklamalar  
 Etkin komut dosyası bağlamı gerektiriyor.  
  
 Bu API yalnızca kenar modunda desteklenir.
