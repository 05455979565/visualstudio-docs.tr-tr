# <a name="array-or-arguments-object-expected"></a>Dizi ya da bağımsız değişkenler nesnesi bekleniyor
Bağımsız değişken olarak bir dizi sağlamadınız. Bu hata yalnızca **function. prototype. Apply** yöntemi için geçerlidir. Belirtilmişse, bu işlevin ikinci bağımsız değişkeni bir `Array` nesnesi ya da bir **bağımsız değişken** nesnesi olmalıdır.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İkinci bağımsız değişken olarak bir `Array` ya da bir **bağımsız değişken** nesnesi belirtin.
