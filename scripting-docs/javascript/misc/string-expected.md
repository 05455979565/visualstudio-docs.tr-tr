# <a name="string-expected"></a>Dize bekleniyor
Çağırmaya çalıştığınız **String.prototype.toString** veya **String.prototype.valueOf** yöntemi dışında bir türde bir nesne üzerinde `String`. Bu tür çağrısının nesne türünde olmalıdır `String`.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yalnızca çağırma **String.prototype.toString** veya **String.prototype.valueOf** türünden nesnelerin yöntemleri `String`.
