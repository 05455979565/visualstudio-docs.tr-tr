# <a name="expected-3939-javascript"></a>Beklenen &#39; (&#39; (JavaScript)
Bir dizi parantez içindeki ifadenin içine denedi, ancak parantez içermiyordu. Bazı ifadeleri açma ve kapama parantez kümesi içinde alınmalıdır. Aşağıdaki örnekte parantez kullanımına dikkat edin.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sol parantez değerlendirme ifadesine ekleyin.
