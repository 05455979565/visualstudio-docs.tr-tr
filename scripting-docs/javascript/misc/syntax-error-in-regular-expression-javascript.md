### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Normal ifade arama dizenizin yapısını aynılarını için olun [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] normal ifade söz dizimi.
