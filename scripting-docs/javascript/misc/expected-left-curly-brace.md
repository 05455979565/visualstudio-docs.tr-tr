# <a name="expected-3939"></a>Beklenen &#39; {&#39;
İşlev gövdesi başlangıcını işaretleyen sol ayraç içermiyordu. Tek bir satır, olsa bile, bir işlev gövdesi yapan kod sol ve sağ köşeli parantez içinde yer almalıdır. İşlev gövdesi kaşlı kullanımını daha katı Döngülerde olduğuna dikkat edin.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İşlev gövdesi başlangıcını işaretleyen sol ayraç ekleyin.
