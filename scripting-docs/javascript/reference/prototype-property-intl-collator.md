# <a name="prototype-property-intlcollator"></a>prototype Özelliği (Intl.Collator)
Prototip başvuru için bir harmanlama döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
collator.prototype  
```  
  
## <a name="remarks"></a>Açıklamalar  
 `collator` Bağımsız değişkeni bir harmanlama adıdır.  
  
 Kullanım `prototype` temel nesne sınıfı için işlevselliği sağlamak için özellik. Bir nesnenin "Bu nesneye atanmış prototip davranışını devral" yeni örnekleri.  
  
 Örneğin, bir yöntem eklemek için `Intl.Collator` kümesinin en büyük öğenin değerini döndürür nesne, işlev bildirme, ekleyin `Intl.Collator.prototype`ve ardından onu kullanabilirsiniz.
