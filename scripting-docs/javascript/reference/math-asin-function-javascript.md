# <a name="mathasin-function-javascript"></a>Math.asin İşlevi (JavaScript)
Bir sayının ark sinüsünü döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli `number` bağımsız değişkeni olan sayısal ifadenin arksinüsünü gereklidir.  
  
 Dönüş değeri arksinüsünü mi `number` radyan cinsinden bağımsız değişkeni.
