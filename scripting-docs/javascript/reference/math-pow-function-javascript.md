# <a name="mathpow-function-javascript"></a>Math.pow İşlevi (JavaScript)
Belirtilen güç için gerçekleştirilecek temel bir ifadenin değerini döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>Parametreler  
 *temel*  
 Gerekli. İfade taban değeri.  
  
 *Üs*  
 Gerekli. İfade üs değeri.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnekte, 1000 baseexponent için eşit bir sayısal ifade döndürür.  
  
```JavaScript  
Math.pow(10,3);  
```
