# <a name="mathatanh-function-javascript"></a>Math.atanh İşlevi (JavaScript)
Bir sayının ters hiperbolik tanjantını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Varsa `x` olan `NaN`, sonuç `NaN`.
