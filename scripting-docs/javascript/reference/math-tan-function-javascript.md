# <a name="mathtan-function-javascript"></a>Math.tan İşlevi (JavaScript)
Bir sayının tanjantını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli `radians` radyan cinsinden ölçülen açı içeren sayısal bir ifade bağımsız değişkeni aşağıdaki gibidir.  
  
 Dönüş değeri sayısal bağımsız değişkeni tanjantını mi `radians`.
