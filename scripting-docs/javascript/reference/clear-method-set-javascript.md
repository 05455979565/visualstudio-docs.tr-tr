# <a name="clear-method-set-javascript"></a>clear Yöntemi (Küme) (JavaScript)
Tüm öğeleri kümesinden kaldırır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
setObj.clear()  
```  
  
#### <a name="parameters"></a>Parametreler  
 `setObj`  
 Gerekli. Clear kümesi.
