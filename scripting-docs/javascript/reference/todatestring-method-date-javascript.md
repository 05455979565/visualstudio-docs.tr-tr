# <a name="todatestring-method-date-javascript"></a>toDateString Yöntemi (Tarih) (JavaScript)
Bir tarih olarak bir string değeri döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli *objDate* başvurusu olan bir `Date` nesnesi.  
  
 `toDateString` Kolay okuma biçimi yöntemi geçerli saat diliminizde, uygun, tarih içeren bir string değeri döndürür.
