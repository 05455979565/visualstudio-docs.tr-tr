# <a name="infinity-constant-javascript"></a>Sonsuzluk Sabiti (JavaScript)
En büyük kayan noktası sayısından daha büyük olan bir sayıdır. Başlangıç değeri döndürür `Number.POSITIVE_INFINITY`. Negatif sonsuz (-sonsuz) en küçük kayan daha küçük nokta sayısı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>Açıklamalar  
 `Infinity` Üyesi sabittir `Global` nesne ve komut dosyası altyapısı başlatıldığında kullanılabilir hale getirilir.
