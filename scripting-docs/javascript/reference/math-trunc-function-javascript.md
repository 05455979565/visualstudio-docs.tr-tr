# <a name="mathtrunc-function-javascript"></a>Math.trunc İşlevi (JavaScript)
Herhangi kesir basamakları kaldırma sayının tamsayı kısmını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```vb  
Math.trunc(number)  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli `number` bağımsız değişkeni olan sayısal ifadenin kesilmiş tamsayı gereklidir.  
  
 Varsa `number` olan `NaN`, döndürür `NaN`.
