# <a name="modulus-assignment-operator--javascript"></a>Modulus Atama İşleci (JavaScript)
Bir ifadenin değerini bir değişken değerle böler ve kalanı değişkenine atar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
result %= expression  
```  
  
## <a name="arguments"></a>Arguments  
 `result`  
 Herhangi bir değişken.  
  
 `expression`  
 Herhangi bir sayısal ifade.  
  
## <a name="remarks"></a>Açıklamalar  
 Kullanarak `%=` işleci tam olarak aynı olup belirtme:  
  
```  
result = result % expression  
```
