# <a name="global-object-javascript"></a>Genel Nesne (JavaScript)
Genel işlevler ve sabitleri bir nesnesine toplamak için amacı olan bir iç nesne.  
  
## <a name="remarks"></a>Açıklamalar  
 `Global` Nesne hiçbir zaman doğrudan kullanılır ve kullanılarak oluşturulamaz `new` işleci. Komut dosyası altyapısı başlatıldığında, bu nedenle, İşlevler ve sabitleri hemen kullanılabilmesini oluşturulur.
