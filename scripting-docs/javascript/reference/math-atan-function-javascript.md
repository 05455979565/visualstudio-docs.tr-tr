# <a name="mathatan-function-javascript"></a>Math.atan İşlevi (JavaScript)
Bir sayının ark tanjantını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli `number` bağımsız değişkeni olan sayısal ifadenin arktanjantını gereklidir.  
  
 Dönüş değeri arktanjantını mi `number` radyan cinsinden bağımsız değişkeni.
