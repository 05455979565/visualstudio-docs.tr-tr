# <a name="prototype-property-intldatetimeformat"></a>prototype Özelliği (Intl.DateTimeFormat)
Prototip başvuru için bir biçimlendirici döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
dateTimeFormat.prototype  
```  
  
## <a name="remarks"></a>Açıklamalar  
 `dateTimeFormat` Bağımsız değişkeni bir biçimlendirici adıdır.  
  
 Kullanım `prototype` temel nesne sınıfı için işlevselliği sağlamak için özellik. Bir nesnenin "Bu nesneye atanmış prototip davranışını devral" yeni örnekleri.  
  
 Örneğin, bir yöntem eklemek için `Intl.DateTimeFormat` kümesinin en büyük öğenin değerini döndürür nesne, işlev bildirme, ekleyin `Intl.DateTimeFormat.prototype`ve ardından onu kullanabilirsiniz.
