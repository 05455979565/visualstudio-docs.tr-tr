# <a name="tostring-method-string-1"></a>toString yöntemi (dize) 1
Bir dizeyi bir dize gösterimini döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
string.toString()  
```  
  
## <a name="parameters"></a>Parametreler  
 `string`  
 Gerekli. Bir dize olarak göstermek için dizisi.  
  
## <a name="return-value"></a>Dönüş Değeri  
 Dize dize gösterimi.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek kullanımını göstermektedir **toString** yöntemi ile bir dize.  
  
```JavaScript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
