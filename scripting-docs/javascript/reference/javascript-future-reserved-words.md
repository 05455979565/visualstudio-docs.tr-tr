# <a name="javascript-future-reserved-words"></a>JavaScript Gelecekte Kullanmak Üzere Ayrılmış Sözcükler
Bunlar geçerli sürümde özel bir anlamı olsa bile JavaScript, gelecekteki ayrılmış anahtar sözcükler tanımlayıcıları kullanılmamalıdır.  
  
 JavaScript ayrılmış sözcükler listesi için bkz: [JavaScript ayrılmış sözcükler](../../javascript/reference/javascript-reserved-words.md).  
  
## <a name="future-reserved-words"></a>Gelecekteki ayrılmış sözcükler  
 Gelecekteki ayrılmış sözcükler şunlardır:  
  
1.  sınıf  
  
2.  enum  
  
3.  dışarı aktar  
  
4.  genişletir  
  
5.  içeri aktar  
  
6.  Süper  
  
## <a name="future-reserved-words-in-strict-mode"></a>Katı modda gelecekteki ayrılmış sözcükler  
 Gelecekteki ayrılmış sözcükler yalnızca katı modda verilmiştir. Katı mod hakkında daha fazla bilgi için bkz: [katı yönerge kullanma](../../javascript/reference/use-strict-directive.md).  
  
1.  uygulamalar  
  
2.  arabirim  
  
3.  Paket  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
