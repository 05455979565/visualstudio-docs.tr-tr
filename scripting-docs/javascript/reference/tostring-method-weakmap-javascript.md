# <a name="tostring-method-weakmap-javascript"></a>toString Yöntemi (WeakMap) (JavaScript)
Bir dize gösterimini döndürür bir `WeakMap` nesnesi.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>Parametreler  
 `weakmapObj`  
 Gerekli. A `WeakMap` nesnesi.
