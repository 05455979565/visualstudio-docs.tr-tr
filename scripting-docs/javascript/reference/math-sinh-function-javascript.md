# <a name="mathsinh-function-javascript"></a>Math.sinh İşlevi (JavaScript)
Bir sayının ters hiperbolik sinüsünü döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Dönüş değeri olarak aynıdır `(exp(x) - exp(-x))/2`. X ise `NaN`, sonuç `NaN`.
