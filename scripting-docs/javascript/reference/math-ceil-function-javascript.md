# <a name="mathceil-function-javascript"></a>Math.ceil İşlevi (JavaScript)
En küçük tamsayı, büyük veya eşit sayısal bağımsız değişkeni döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli `number` bağımsız değişken sayısal bir ifadedir.  
  
 Dönüş değeri sıfırdan büyük veya eşit sayısal bağımsız değişkeni için en küçük tamsayı eşit bir tamsayı değeri ' dir.
