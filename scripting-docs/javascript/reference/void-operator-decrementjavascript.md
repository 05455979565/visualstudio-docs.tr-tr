# <a name="void-operator-javascript"></a>void İşleç (JavaScript)
Gelen bir değer döndüren bir ifade engeller.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
void expression   
```  
  
## <a name="remarks"></a>Açıklamalar  
 `expression` Bağımsız değişkeni herhangi geçerli [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] ifade.  
  
 `void` İşleci ifadesini değerlendirir ve döndürür `undefined`. Burada bir ifadenin değerlendirilmesi ancak sonuçları betik geri kalanı için görünür istemediğiniz durumlarda yararlıdır.
