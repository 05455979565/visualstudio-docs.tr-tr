# <a name="prototype-property-set"></a>prototype Özelliği (Küme)
Prototip kümesine ilişkin bir başvuru döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
set.prototype  
```  
  
## <a name="remarks"></a>Açıklamalar  
 `set` Bağımsız değişkeni bir küme adıdır.  
  
 Kullanım `prototype` temel nesne sınıfı için işlevselliği sağlamak için özellik. Bir nesnenin "Bu nesneye atanmış prototip davranışını devral" yeni örnekleri.  
  
 Örneğin, bir yöntem eklemek için `Set` kümesinin en büyük öğenin değerini döndürür nesne, işlev bildirme, ekleyin `Set.prototype`ve ardından onu kullanabilirsiniz.
