# <a name="mathatan2-function-javascript"></a>Math.atan2 İşlevi (JavaScript)
(Radyan cinsinden açı) X ekseni bir noktasına döndürür (*y*,*x*).  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>Parametreler  
 *x*  
 Gerekli. Kartezyen x koordinatını temsil eden bir sayısal ifade.  
  
 *y*  
 Gerekli. Kartezyen y koordinatını temsil eden bir sayısal ifade.  
  
## <a name="remarks"></a>Açıklamalar  
 Dönüş değeri arasındadır *-pi* ve *PI*. Sağlanan açısını temsil eder (*y*,*x*), radyan cinsinden gelin.
