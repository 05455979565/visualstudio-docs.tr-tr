# <a name="addition-operator--javascript"></a>Toplama İşleci (+) (JavaScript)
Başka bir sayısal ifadenin değerini ekler veya iki dizeyi art arda ekler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
result = expression1 + expression2  
```  
  
## <a name="parameters"></a>Parametreler  
 `result`  
 Herhangi bir değişken.  
  
 `expression1`  
 Herhangi bir ifade.  
  
 `expression2`  
 Herhangi bir ifade.  
  
## <a name="remarks"></a>Açıklamalar  
 İki ifade türleri davranışını belirlemek  **+**  işleci.  
  
|Eğer|Ardından|  
|--------|----------|  
|Her iki ifade sayısal veya Boolean|Ekle|  
|Her iki ifade dizeler şunlardır:|Birleştirme|  
|Bir ifade sayısal ve diğer bir dizedir|Birleştirme|
