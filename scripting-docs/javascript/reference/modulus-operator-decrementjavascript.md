# <a name="modulus-operator--javascript"></a>Modulus işleci (JavaScript)
Başka bir sayısal ifadenin değerini sayısal ifadenin değerini böler ve kalanı üretir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>Arguments  
 `result`  
 Herhangi bir değişken.  
  
 `expression1`  
 Herhangi bir sayısal ifade.  
  
 `expression2`  
 Herhangi bir sayısal ifade.
