# <a name="mathfloor-function-javascript"></a>Math.floor İşlevi (JavaScript)
En büyük tamsayı sayısal bağımsız değişkeni küçük veya buna eşit döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli *numarası* bağımsız değişken sayısal bir ifadedir.  
  
 Dönüş değeri en büyük tamsayıya eşit bir tamsayı değeri veya sayısal bağımsız değişkeni eşittir küçük.
