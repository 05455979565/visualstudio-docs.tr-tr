# <a name="size-property-map-javascript"></a>size Özelliği (Eşleme) (JavaScript)
Bir harita öğe sayısını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
sizeVar = mapObj.size  
```  
  
#### <a name="parameters"></a>Parametreler  
 `sizeVar`  
 Gerekli. Herhangi bir sayı.  
  
 `mapObj`  
 Gerekli. Tüm `Map` nesnesi.
