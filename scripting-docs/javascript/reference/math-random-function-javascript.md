# <a name="mathrandom-function-javascript"></a>Math.random İşlevi (JavaScript)
Geçici rastgele bir sayı 0 ile 1 arasında döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Oluşturulan geçici rastgele sayı 0 (dahil) 1 için (özel), diğer bir deyişle, döndürülen sayı sıfır olabilir, ancak her zaman birden az olacak. Rastgele sayı üreticisinin otomatik olarak zaman sağlanmış [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] önce yüklenir.
