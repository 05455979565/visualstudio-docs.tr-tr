# <a name="resolvedoptions-method-intlnumberformat"></a>resolvedOptions Yöntemi (Intl.NumberFormat)
Özellikleri ve değerlerini içeren bir nesne döndürür `NumberFormat` biçimlendirici nesnesi.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametreler  
 `numberFormatObj`  
 `NumberFormat` Bilgi almaya nesnesi.  
  
## <a name="remarks"></a>Açıklamalar  
 Döndürülen nesne özelliklerini hesaplanan özelliklerine karşılık gelen `NumberFormat` nesnesi.
