# <a name="mathexpm1-function-javascript"></a>Math.expm1 İşlevi (JavaScript)
Üssü 1'e (Doğal logaritmanın tabanı)'den çıkarılmasıyla sonucunu döndürür).  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.expm1(number)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli `number` bağımsız değişkeni e gücünü temsil eden bir sayısal ifadedir.  
  
 Dönüş değeri olan bir sayıdır. E sabiti Euler'ın 2.71828 için yaklaşık olarak eşit numarasıdır ve `number` sağlanan bağımsız değişken.
