# <a name="javascript-reserved-words"></a>JavaScript Ayrılmış Sözcükler
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]tanımlayıcı olarak kullanamazsınız ayrılmış sözcükler sayısına sahip. Ayrılmış sözcükler sahip özel bir anlamı [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] dil dili sözdizimi bir parçası olarak. Ayrılmış bir sözcük kullanarak bir derleme hatası komut dosyanız yüklenirken neden olur.  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]aynı zamanda gelecekteki ayrılmış sözcükler listesine sahiptir. Bu sözcükleri olmayan şu anda parçası [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] dil, gelecekte kullanılmak üzere ayrılmış olsa. JavaScript gelecekte ayrılmış anahtar sözcükler hakkında daha fazla bilgi için bkz: [JavaScript gelecekteki ayrılmış sözcükler](../../javascript/reference/javascript-future-reserved-words.md).  
  
 Tanımlayıcıları seçerken da zaten iç adları olan herhangi bir sözcük önlemek önemlidir [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] nesneleri veya işlevleri gibi `String` veya `parseInt`.
