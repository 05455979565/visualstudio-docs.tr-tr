# <a name="mathlog2-function-javascript"></a>Math.log2 İşlevi (JavaScript)
Bir sayının 2 logaritmasını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
Math.log2(x)   
```  
  
## <a name="return-value"></a>Dönüş Değeri  
 Varsa `x` olan `NaN`, bu işlevi döndürür `NaN`. Varsa `x` değerinden sıfır (0), bu işlevi döndürür olduğu `NaN`.
