# <a name="prototype-property-intlnumberformat"></a>prototype Özelliği (Intl.NumberFormat)
Prototip başvuru için bir biçimlendirici döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
numberFormat.prototype  
```  
  
## <a name="remarks"></a>Açıklamalar  
 `numberFormat` Bağımsız değişkeni bir biçimlendirici adıdır.  
  
 Kullanım `prototype` temel nesne sınıfı için işlevselliği sağlamak için özellik. Bir nesnenin "Bu nesneye atanmış prototip davranışını devral" yeni örnekleri.  
  
 Örneğin, bir yöntem eklemek için `Intl.NumberFormat` kümesinin en büyük öğenin değerini döndürür nesne, işlev bildirme, ekleyin `Intl.NumberFormat.prototype`ve ardından onu kullanabilirsiniz.
