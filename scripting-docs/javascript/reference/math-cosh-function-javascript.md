# <a name="mathcosh-function-javascript"></a>Math.cosh İşlevi (JavaScript)
Bir sayının hiperbolik kosinüsünü döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Dönüş değeri olarak aynıdır `(exp(x) + exp(-x))/2`. Varsa `x` olan `NaN`, sonuç `NaN`.
