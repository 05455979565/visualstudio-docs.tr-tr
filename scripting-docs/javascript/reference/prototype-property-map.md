# <a name="prototype-property-map"></a>prototype Özelliği (Eşleme)
Bir harita prototipi bir başvuru döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
map.prototype  
```  
  
## <a name="remarks"></a>Açıklamalar  
 `map` Bağımsız değişkeni bir harita adıdır.  
  
 Kullanım `prototype` temel nesne sınıfı için işlevselliği sağlamak için özellik. Bir nesnenin "Bu nesneye atanmış prototip davranışını devral" yeni örnekleri.  
  
 Örneğin, bir yöntem eklemek için `Map` kümesinin en büyük öğenin değerini döndürür nesne, işlev bildirme, ekleyin `Map.prototype`ve ardından onu kullanabilirsiniz.
