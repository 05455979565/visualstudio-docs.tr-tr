# <a name="resolvedoptions-method-intldatetimeformat"></a>resolvedOptions Yöntemi (Intl.DateTimeFormat)
Özellikleri ve tarih biçimlendirici nesnenin değerlerini içeren bir nesne döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametreler  
 `dateTimeFormatObj`  
 `DateTimeFormat` Bilgi almaya nesnesi.  
  
## <a name="remarks"></a>Açıklamalar  
 Döndürülen nesne özelliklerini hesaplanan özelliklerine karşılık gelen `DateTimeFormat` nesnesi.
