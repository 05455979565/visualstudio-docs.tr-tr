# <a name="mathclz32-function-javascript"></a>Math.clz32 İşlevi (JavaScript)
32-bit ikili temsili bir sayı sıfır BITS baştaki sayısını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.clz32(  
number  
)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Varsa `number` 0 ise, sonuç 32 olacaktır. Varsa, en çok 32-bit ikili kodlamasını, önemli bit `number` 1, sonuç 0 olacaktır.
