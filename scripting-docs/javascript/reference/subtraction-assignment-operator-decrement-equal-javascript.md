# <a name="subtraction-assignment-operator---javascript"></a>Çıkarma Atama İşleci (-=) (JavaScript)
Değerinden bir ifadenin bir değişkenin değerini çıkarır ve sonuç değişkenine atar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>Parametreler  
 *Sonuç*  
 Herhangi bir sayısal değişkeni.  
  
 *ifade*  
 Herhangi bir sayısal ifade.  
  
## <a name="remarks"></a>Açıklamalar  
 Kullanarak `-=` işleci tam olarak aynı olup aşağıdakileri yapabilirsiniz:  
  
```JavaScript  
result = result - expression  
```
