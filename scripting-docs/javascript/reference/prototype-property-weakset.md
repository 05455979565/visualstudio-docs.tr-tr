# <a name="prototype-property-weakset"></a>prototype Özelliği (WeakSet)
İçin prototip bir başvuru döndürür bir `WeakSet`.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
weakset.prototype  
```  
  
## <a name="remarks"></a>Açıklamalar  
 `weakset` Bağımsız değişkeni bir küme adıdır.  
  
 Kullanım `prototype` temel nesne sınıfı için işlevselliği sağlamak için özellik. Bir nesnenin "Bu nesneye atanmış prototip davranışını devral" yeni örnekleri.  
  
 Örneğin, bir yöntem eklemek için `WeakSet` kümesinin en büyük öğenin değerini döndürür nesne, işlev bildirme, ekleyin `WeakSet.prototype`ve ardından onu kullanabilirsiniz.
