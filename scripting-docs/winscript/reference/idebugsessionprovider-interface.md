# <a name="idebugsessionprovider-interface"></a>IDebugSessionProvider Arabirimi
Hata ayıklayıcı IDE konak ve dil etkinleştirmek için sağlanan birincil arabirimi hata ayıklama başlatıldı. Bu, bir hata ayıklama oturumu için çalışan bir uygulama oluşturur. Bu arabirim Makine Hata Ayıklama Yöneticisi tarafından uygulanır.  
  
 Devralınan yöntemleri yanı sıra `IUnknown`, `IDebugSessionProvider` arabirimi aşağıdaki yöntemleri sunar.  
  
## <a name="methods-in-vtable-order"></a>Vtable sırayla yöntemleri  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[IDebugSessionProvider::StartDebugSession](../../winscript/reference/idebugsessionprovider-startdebugsession.md)|Belirtilen uygulama ile bir hata ayıklama oturumu başlatır.|
