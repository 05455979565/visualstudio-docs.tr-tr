# <a name="iapplicationdebuggerui-interface"></a>IApplicationDebuggerUI Arabirimi
Hata ayıklayıcı tümleşik geliştirme ortamı (IDE) uygulanan (ek olarak `IApplicationDebugger`) bir dış bileşen hata ayıklayıcının kullanıcı arabirimini (UI) üzerinde daha fazla denetim sağlamak için.  
  
 Devralınan yöntemleri yanı sıra `IUnknown`, `IApplicationDebuggerUI` arabirimi aşağıdaki yöntemleri sunar.  
  
## <a name="methods-in-vtable-order"></a>Vtable sırayla yöntemleri  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[IApplicationDebuggerUI::BringDocumentToTop](../../winscript/reference/iapplicationdebuggerui-bringdocumenttotop.md)|Pencerenin üst hata ayıklayıcı belirtilen hata ayıklama belge içeren kullanıcı arabirimi sunar.|  
|[IApplicationDebuggerUI::BringDocumentContextToTop](../../winscript/reference/iapplicationdebuggerui-bringdocumentcontexttotop.md)|Üst hata ayıklayıcı kullanıcı arabirimi için belirtilen belge bağlamını içeren bir pencere getirir ve bağlam pencereye kaydırır.|
