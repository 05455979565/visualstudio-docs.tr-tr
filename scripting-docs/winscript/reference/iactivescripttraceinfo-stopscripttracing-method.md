# <a name="iactivescripttraceinfostopscripttracing-method"></a>IActiveScriptTraceInfo::StopScriptTracing Yöntemi
Betik izlemeyi durdurur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp
HRESULT StopScriptTracing(void); }  
```  
  
#### <a name="parameters"></a>Parametreler  
 Bu yöntem, hiç parametre yok.  
  
## <a name="return-value"></a>Dönüş Değeri  
 Bu yöntemin dönüş olası değerler aşağıdaki gibidir:  
  
1. S_OK: Başarılı.  
  
2. E_NOTIMPL: Henüz uygulanmadı.
