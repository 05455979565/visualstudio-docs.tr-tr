# <a name="idebugasyncoperationcallback-interface"></a>IDebugAsyncOperationCallBack Arabirimi
İlerleme durumunu ilgili durum olayları sağlayan bir `IDebugAsyncOperation` arabirim değerlendirme.  
  
## <a name="methods"></a>Yöntemler  
 Devralınan yöntemleri yanı sıra `IUnknown`, `IDebugAsyncOperationCallBack` arabirimi aşağıdaki yöntemleri sunar.  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|Bir zaman uyumsuz hata ayıklama işleminin bir sonucu olduğunu bildirir.|
