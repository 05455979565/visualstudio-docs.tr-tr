# <a name="icanhandleexception-interface"></a>ICanHandleException Arabirimi
Çağıranın hangi özel durumları belirtmek için bir komut dosyası altyapısı çağıran sağlayan işler.  
  
## <a name="methods"></a>Yöntemler  
 Devralınan yöntemleri yanı sıra `IUnknown`, `ICanHandleException` arabirimi aşağıdaki yöntemleri sunar.  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|Komut dosyası altyapısı çağıran bir belirtilen bir özel durumu işleyebilir belirler.|
