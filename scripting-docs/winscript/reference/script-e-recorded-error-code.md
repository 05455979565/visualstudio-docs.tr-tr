# <a name="script_e_recorded-error-code"></a>SCRIPT_E_RECORDED Hata Kodu
Betik altyapısı ve konak arasında bir hata geçirildi. Konağın hata kodunu çağırana geçmesi gerekir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
