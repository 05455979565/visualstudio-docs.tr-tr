# <a name="ijsenumdebugpropertygetcount-method"></a>IJsEnumDebugProperty::GetCount Metodu
Nesnenin özelliklerini sayısını alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `pCount`  
 [out] Özellik sayısı.  
  
## <a name="return-value"></a>Dönüş Değeri  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jscript9diag.h
