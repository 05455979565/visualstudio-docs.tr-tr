# <a name="ijsdebugbreakpoint-interface"></a>IJsDebugBreakPoint Arabirimi
Bir kesme noktasını temsil eder.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp
IJsDebugBreakPoint : public IUnknown;  
```  
  
## <a name="members"></a>Üyeler  
  
### <a name="public-methods"></a>Ortak Yöntemler  
  
|Ad|Açıklama|  
|----------|-----------------|  
|[IJsDebugBreakPoint::Delete Metodu](../../winscript/reference/ijsdebugbreakpoint-delete-method.md)|Kesme noktasını siler.|  
|[IJsDebugBreakPoint::Disable Metodu](../../winscript/reference/ijsdebugbreakpoint-disable-method.md)|Kesme noktasını devre dışı bırakır.|  
|[IJsDebugBreakPoint::Enable Metodu](../../winscript/reference/ijsdebugbreakpoint-enable-method.md)|Kesme noktasını etkinleştirir.|  
|[IJsDebugBreakPoint::GetDocumentPosition Metodu](../../winscript/reference/ijsdebugbreakpoint-getdocumentposition-method.md)|Kesme noktasının nerede bağlı olduğu deyimin konumunu döndürür.|  
|[IJsDebugBreakPoint::IsEnabled Metodu](../../winscript/reference/ijsdebugbreakpoint-isenabled-method.md)|Kesme noktasının etkinleştirilip etkinleştirilmeyeceğini belirler.|
