# <a name="contribute-to-visual-studio-documentation"></a>Visual Studio belgelerine katkıda bulunun

Visual Studio dokümanlarına katkıda bulunmak için zaman ayırdığınız için teşekkür ederiz.

Bu kılavuz, katkıda bulunma yla ilgili bazı genel konuları kapsar ve daha ayrıntılı açıklamalar için docs.microsoft.com [katkıda bulunanlar kılavuzuna](https://docs.microsoft.com/contribute) atıfta bulunur.

## <a name="code-of-conduct"></a>Kullanım kuralları

Bu proje [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) (Microsoft Açık Kaynak Kullanım Kuralları) belgesinde listelenen kurallara uygundur. Daha fazla bilgi için, [Davranış Kuralları SSS'ye](https://opensource.microsoft.com/codeofconduct/faq/)bakın veya ek soru veya yorumlarla iletişim kurun. [opencode@microsoft.com](mailto:opencode@microsoft.com)

## <a name="how-can-i-contribute"></a>Nasıl katkıda bulunabilirim?

Belgelere katkıda bulunmanın çeşitli yolları vardır. Hangisinin sizin için uygun olduğunu öğrenmek için aşağıdaki bölümleri inceleyin.

### <a name="report-bugs-or-suggest-enhancements"></a>Hataları bildirme veya geliştirmeler önerme

Hataları ve önerileri göndermek için lütfen herhangi bir makalenin altındaki geri bildirim aracını kullanın.

![Geri bildirim aracı](media/feedback-tool.png)

### <a name="quick-edit-in-github"></a>GitHub'da hızlı bir şekilde edin

Katkıda bulunanlar kılavuzundaki [varolan belgelerde hızlı bir şekilde yapılan tadiller](https://docs.microsoft.com/contribute/#quick-edits-to-existing-documents) için kılavuzu izleyin.

### <a name="larger-edits"></a>Daha büyük deyiş

Katkıda bulunanlar kılavuzundaki [çekme istekleri](https://docs.microsoft.com/contribute/how-to-write-workflows-major#pull-request-processing) kılavuzunu gözden geçirin.

**Not**

Ürünümüzün eski bir sürümü olan `docs/vs-2015` Visual Studio 2015 için olduğu için klasördeki dosyalar üzerinde aktif olarak çalışmıyoruz. Bu dosyalarda önemsiz değişiklikler için çekme isteklerini kabul etmiyoruz.
