# <a name="activation"></a>Etkinleştirme

Mac için Visual Studio, bir Merkezi'nden erişilebilen seçmek için üç farklı bir abonelik düzeyi sağlar:

* Mac için Visual Studio Enterprise
* Mac için Visual Studio Professional
* Mac için Visual Studio Community

Bir abonelik kullanmaya başlamak için oturum açmak için Visual Studio için Mac içindeki adımları izleyerek [Mac için Visual Studio'da oturum açma](~/signing-in.md) Kılavuzu. Bu otomatik olarak lisansınızı etkinleştirmeniz ve hesapları iletişim kutusunda görüntüleyin:

![Kullanıcı Lisansı iletişim kutusunu göster](media/user-accounts-login.png)

## <a name="update-expired-licenses"></a>Süresi dolan lisansları güncelleştirin 

Aboneliğinizin süresi sona erdi, iki seçeneğiniz vardır:

1. Aboneliğinizi yenileyin. Göz atarak aboneliğinizi yenileyebilirsiniz [https://www.visualstudio.com/vs/pricing/](https://www.visualstudio.com/vs/pricing/).

2. Başka bir hesapla etkin bir aboneliğiniz varsa, bunu Visual Studio'ya Mac için ayrıntılı biçimde açıklandığı gibi ekleyebilirsiniz [birden çok kullanıcı hesapları ekleme](~/signing-in.md) konu. Mac için Visual Studio eklediğiniz tüm hesapları arasından en iyi lisans algılar. 

## <a name="product-key-usage"></a>Ürün anahtarı kullanımını

Şu anda, Visual Studio Professional veya Enterprise yetkilendirmeler Mac için etkinleştirmek üzere bir ürün anahtarı kullanmak üzere bir yolu yoktur Visual Studio Enterprise Mac veya Visual Studio Professional Mac için kullanmak üzere ilgili bilmeniz gereken [abonelik](https://www.visualstudio.com/subscriptions/) ve [oturum](~/signing-in.md) IDE'ye.

## <a name="offline-activation"></a>Çevrimdışı etkinleştirme

> [!NOTE]
> Şu anda çevrimdışı etkinleştirme yok Visual Studio için Mac için yok Aboneliğinizi etkinleştirmek için oturum açarken Internet'e bağlı gerekir.

Mac Professional veya Enterprise abonelik için Visual Studio 2017 etkinleştirmek için oturum açarken Internet'e bağlı gerekir.  

**Not**: şu anda hiçbir çevrimdışı etkinleştirme için Visual Studio Professional Mac veya Visual Studio Enterprise Mac için yok Ancak, Mac için Visual Studio 2017 lisans uygun olarak bir internet bağlantısı olmayan ortamlarda kullanılabilir.

İnternet bağlantısı olmadan makineler için Mac için Visual Studio Enterprise 2017 aldıysanız etkinleştirme, ancak Xamarin denetçisi, Xamarin profil oluşturucu Mac için Visual Studio uygulamasını kullanmaya devam edebilir ve katıştırılmış derlemeleri özellikleri değişmeyecek kullanılabilir.
