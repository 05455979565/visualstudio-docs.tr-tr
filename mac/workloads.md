# <a name="visual-studio-for-mac-workloads"></a>Mac İş Yükleri için Visual Studio

Mac için Visual Studio, aşağıdaki uygulama türlerini geliştirmek için iş yükleriyle birlikte önceden yüklenmiş olarak gelir:

* Android, iOS, tvOS ve watchOS için Xamarin kullanarak .NET'e sahip çapraz platform mobil uygulamalar
* Mac masaüstü uygulamaları
* .NET Core uygulamaları
* ASP.NET Çekirdek Web uygulamaları
* Platform ötesi Unity oyunları

## <a name="mobile-applications-with-xamarin"></a>Xamarin ile mobil uygulamalar

[Xamarin,](xamarin.md) Android, iOS, tvOS, watchOS ve Mac masaüstü uygulamaları için .NET ile platformlar arası mobil uygulamalar oluşturmanıza olanak tanır.

Mac için Visual Studio'da Xamarin'i kullanma hakkında daha fazla bilgi için [Xamarin Geliştirici Merkezi'ne](https://developer.xamarin.com/) bakın ve Mobil uygulamalar [uygulamalı laboratuvarına](https://github.com/Microsoft/vs4mac-labs/tree/master/Mobile/Getting-Started)göz atın.

## <a name="mac-desktop-applications"></a>Mac masaüstü uygulamaları

Cocoa kullanıcı arabirimi çerçevesini kullanarak [Xamarin.Mac](/xamarin/mac/) ile Mac uygulamaları ve diğer platformlarla kod ve kullanıcı arabirimi paylaşan uygulamalar oluşturmak için [Mac'teki Xamarin.Forms'u](/xamarin/xamarin-forms/platform/other/mac) oluşturabilirsiniz.

## <a name="net-core-applications"></a>.NET Çekirdek uygulamaları

.NET Core'u kullanma hakkında daha fazla bilgi için [belgelere](/dotnet/core/)bakın.

## <a name="aspnet-core-web-applications"></a>ASP.NET Core Web Uygulamaları

Mac için Visual Studio'da ASP.NET Core'u kullanmaya başlamak hakkında daha fazla bilgi için, [ASP.NET Core kılavuzuyla Başlarken'e](asp-net-core.md) bakın veya ASP.NET Core web uygulamaları [uygulamalı laboratuvarına](https://github.com/Microsoft/vs4mac-labs/tree/master/Web/Getting-Started)göz atın.

## <a name="unity-game-development"></a>Unity oyun geliştirme

Unity ile oyun oluşturmaya başlamak için [Kurulum sayfasına](setup-vsmac-tools-unity.md) ve [Unity Tools'u kullanma kılavuzuna](using-vsmac-tools-unity.md)bakın veya Unity [uygulamalı laboratuvarına](https://github.com/Microsoft/vs4mac-labs/tree/master/Unity/Getting-Started)göz atın.
