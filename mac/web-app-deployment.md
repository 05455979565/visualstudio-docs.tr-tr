# <a name="aspnet-core-web-app-deployment"></a>ASP.NET Core Web uygulaması dağıtımı

Web uygulamaları geliştirme önemli bir adım (dağıtım) yayımlama. Başkalarının uygulamayı kullanabilmesi için veya test amacıyla uzak bir ortama yayımlayabilirsiniz. Mac için Visual Studio uygulamanızı yayımlamak için iki seçenek sunar. Azure App Service sağlar, Azure App Service, web uygulamanızı yayımlamak için yayımlayın. Klasör sağlar, bir klasöre uygulamanızı yayımlamak için yayımlayın.

## <a name="publish-to-azure-app-service"></a>Azure App Service’e yayımlama

Mac için Visual Studio kullanarak web uygulamanızı Azure App Service'e doğrudan yayımlayabilirsiniz. ASP.NET Core web uygulamasını Azure App Service'e başvurmak için yayımlamayı [Azure App Service'e yayımlama](publish-app-svc.md) makalesi.

## <a name="publish-to-folder"></a>Klasöre Yayımlama

ASP.NET Core web uygulamalarınızı bir klasöre de yayımlayabilirsiniz. Bilgi edinmek için web uygulamanızı bir klasöre nasıl yayımlayabilirsiniz hakkında daha fazla başvurmak [yayımlama klasörüne](publish-folder.md).
