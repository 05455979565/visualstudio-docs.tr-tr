# <a name="razor-support"></a>Razor desteği

Mac için Visual Studio, IntelliSense ve *.cshtml* dosyalarında sözdizimi vurgulama dahil olmak üzere Razor düzenlemeyi destekler.

![mac için görsel stüdyoda jilet düzenleme](media/razor-image1.png)

## <a name="getting-started-with-razor-in-visual-studio-for-mac"></a>Mac için Visual Studio'da Razor ile Başlarken

Mac için Visual Studio Razor ile başlarken göz önünde bulundurulması gereken iki seçenek vardır: ASP.NET Core razor Pages ve ASP.NET Core MVC. Öğreticiler ve her iki seçenek hakkında daha fazla bilgi için aşağıdaki kılavuzlardan birini ziyaret edin:

- [Mac için Visual Studio ile macOS'ta ASP.NET Core'daki Razor Pages ile başlayın](/aspnet/core/tutorials/razor-pages-mac/razor-pages-start?view=aspnetcore-2.1)
- [Mac için ASP.NET Core MVC ve Visual Studio ile başlayın](/aspnet/core/tutorials/first-mvc-app-mac/start-mvc?view=aspnetcore-2.1)
