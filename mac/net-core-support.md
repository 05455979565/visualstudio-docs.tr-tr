# <a name="net-core-support"></a>.NET Core Desteği

Aşağıdaki tabloda, Mac için Visual Studio kararlı ve önizleme sürümleri tarafından desteklenen .NET Core sürümlerini açıklar:

.NET core SDK sürümü  |8\.1 Mac için Visual Studio (Stable)  |8\.2 (Önizleme) Mac için Visual Studio  |
|---------|---------|---------|
|v2.1.0 - v2.1.5xx    |✗|✗|
|v2.1.600 +     |✔︎|✔︎|
|v2.2.1 - v2.2.1xx|✗|✗|
|v2.2.200 + |✔︎| ✔︎ |
|V3.0 < 3 Önizleme |✗|✗|
|V3.0 Preview 3, 4, 5 |✔︎|✔︎ |
|V3.0 Önizleme 6 |✗|✔︎ |

> [!IMPORTANT]
> Daha önce .NET Core v2.2.1xx Visual Studio ile Mac 8.0 için kullanmakta olduğunuz ise, el ile .NET Core, desteklenen bir sürüme güncelleştirmek yukarıdaki tabloda listelenen gerekir. Ya da öneririz [2.1.700](https://dotnet.microsoft.com/download/dotnet-core/2.1) veya [2.2.300](https://dotnet.microsoft.com/download/dotnet-core/2.2)

* .NET core v2.1.700 Yükleyici ile varsayılan olarak yüklenir.
* Herhangi bir .NET Core sürümünü indirmek için şurayı ziyaret edin [dotnet sayfa](https://dotnet.microsoft.com/download/dotnet-core).
* Mac için Visual Studio'nun Önizleme sürümü yükleme hakkında daha fazla bilgi için bkz: [bir önizleme sürümünü yüklemek](https://docs.microsoft.com/visualstudio/mac/install-preview) Kılavuzu.
