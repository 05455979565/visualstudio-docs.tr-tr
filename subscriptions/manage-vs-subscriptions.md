# <a name="managing-subscriptions"></a>Abonelikleri yönetme

Visual Studio [Abone Portalı,](https://my.visualstudio.com) tüm aboneliklerinizi ve program üyeliklerinizi tek bir yerde yönetmenize izin verir. Profilinizde değişiklik yapabilir, aboneliklerinizde yer alan tüm avantajları kullanabilir ve abonelikler arasında hızlı ve kolay bir şekilde geçiş yapabilirsiniz.

## <a name="managing-multiple-subscriptions"></a>Birden çok aboneliği yönetme

Visual Studio abonelikleri portalı, tüm Visual Studio aboneliklerinizi ve programlarınızı tek bir geliştirilmiş portal deneyiminde bir araya getirir. Örneğin, iş yoluyla Visual Studio Enterprise gibi standart bir aboneliğiniz ve evde kullandığınız aylık Visual Studio Professional aboneliğiniz veya Visual Studio Dev Essentials üyeliğine sahip olabilirsiniz.

### <a name="which-subscription-am-i-using"></a>Hangi aboneliği kullanıyorum?

Avantajlar sayfasında, kullanılan abonelik mavi bayrakta belirgin bir şekilde görüntülenir. Ayrıca sayfanın sağ üst kısmında görüntülenir.
> [!div class="mx-imgBorder"]
> ![Etkin abonelik](_img/manage-vs-subscriptions/current-subscription-cropped.png)

### <a name="to-switch-between-subscriptions-you-can"></a>Abonelikler arasında geçiş yapmak için şunları yapabilirsiniz:

- Sağ üstteki açılır yere tıklayın ve listeden aboneliği seçin
- Abonelikler sekmesine tıklayın ve ardından istediğiniz abonelmeye tıklayın
  > [!div class="mx-imgBorder"]
  > ![Abonelikleri değiştirme](_img/manage-vs-subscriptions/change-subscription-resized.png)

### <a name="if-your-subscription-is-not-visible"></a>Aboneliğiniz görünmüyorsa...

- Süresi dolmuş olabilir. Abone Portalında yalnızca etkin abonelikler görüntülenir.
- Farklı bir e-posta adresiyle ilişkili olabilir. Örneğin, iş/okul e-postanızla Visual Studio Professional aboneliğinde oturum açarsanız ve Microsoft Hesabınızla ilişkili bir Visual Studio Dev Essentials üyeliğiniz varsa, aboneliği yalnızca Abone Portalı'nda oturum açmak için kullandığınız e-posta adresine karşılık gelen abonelik görürsünüz. Diğer aboneliği görmek için oturum açmanız ve ardından diğer e-posta adresiyle oturum açmanız gerekir.
- Şirketiniz Toplu Lisans aracılığıyla abonelik satın alıyorsa, yöneticiniz size bir abonelik atamamış veya aboneliğinizi başka bir kullanıcıya yeniden atamış olabilir.

## <a name="renewing-my-subscriptions"></a>Aboneliklerimi yenileme

**Standart abonelikler**
- Microsoft Mağazası: Aboneliğinizi Microsoft [Mağazası'ndan](https://www.microsoft.com/store)satın aldıysanız, aboneliğinizi orada da yenileyebilirsiniz.
- Toplu Lisans: Şirketiniz Toplu Lisans aracılığıyla abonelik satın alıyorsa, yöneticiniz aboneliklerinizi yenilemek için bir satıcıya başvurmalıdır.

**Bulut abonelikleri:**  Bulut aboneliklerinin yenilenmesi gerekmez. Aboneliğiniz ödendiği sürece etkin kalır.

**Programlar aracılığıyla edinilen abonelikler:**  Aboneliğinizi bir program aracılığıyla alırsanız, aboneliğiniz programa katılımınıza bağlıdır. Daha fazla bilgi için lütfen program kişilerinize ulaşın.

- [Hayal](https://imagine.microsoft.com/about)
- [Microsoft İş Ortağı Ağı](https://partner.microsoft.com)
- [Microsoft Sertifikalı Eğitmen](https://www.microsoft.com/learning/mct-certification.aspx)

## <a name="transferring-subscriptions"></a>Abonelikaktarma

Aboneliğiniz Microsoft Toplu Lisanslama aracılığıyla satın alındıysa ve bu aboneliği kuruluş taki başka birine yeniden atamanız gerekiyorsa, yöneticinizin bunu sizin için yapması gerekir.
Perakende abonelikleri, Müşteri Hizmetleri ile iletişime geçilerek başka bir kullanıcıya aktarılabilir. Ücretsiz veya "Yeniden Satış Için Değil" abonelikleri aktarılamaz.

## <a name="managing-my-profile"></a>Profilimi yönetme

Abonelikportalı, bir profil oluşturmanıza ve yönetmenize olanak sağlayarak abonelik yönetimini kolaylaştırır. Kimlik bilgilerinizde değişiklik yapabilir ve aboneliklerinizi mevcut Azure hesaplarındaki veya Azure DevOps kuruluşlarından alternatif kimlikler kullanarak diğer oturum açma kimlikleriyle ilişkilendirebilirsiniz.

### <a name="changing-profile-information"></a>Profil bilgilerini değiştirme

Profilinizdeki kişi bilgilerini ve tercihlerini kolayca değiştirebilirsiniz.

1. Abone Portalı'nın sağ üst köşesindeki adınıza tıklayın.
2. Profili **Edit** düğmesini tıklatın.
3. **Hakkında** sekmesinde adınızı, e-postanızı, ülkenizi/bölgenizi, profil resminizi ve kişi tercihlerinizi değiştirebilirsiniz. **Tercihler** sekmesinde, tercih ettiğiniz dil, tarih ve saat desenleri, saat dilimi ve kullanıcı arabirimi tesini seçebilirsiniz.
4. Değişikliklerinizi tamamladıktan sonra **Kaydet'i**tıklatın.
5. Avantajlar sayfasına dönmek için tarayıcınızın **geri** düğmesini tıklatın.

### <a name="setting-communications-preferences"></a>İletişim tercihlerini ayarlama
Aylık Visual Studio Abonelikleri bülteni de dahil olmak üzere aboneliğiniz ve avantajlarınız hakkında periyodik güncellemeler almak istiyorsanız, Visual Studio profilinizdeki Visual Studio Abonelikleri kutusunu işaretleyerek [tercihlerinizi güncelleyin.](https://app.vsaex.visualstudio.com/me?workflowID=devprogram&tab=edit) Bu tercihleri değiştirebilir ve istediğiniz zaman aboneliğinizi iptal edebilirsiniz. 

   > [!div class="mx-imgBorder"]
   > ![Tercihleri Ayarlama](_img/manage-vs-subscriptions/change-prefs.png)
   
### <a name="linking-my-subscription-to-existing-azure-devops-services-or-azure-subscriptions"></a>Aboneliğimi mevcut Azure DevOps Hizmetlerine veya Azure aboneliklerine bağlama
Aboneliğinizde kişisel bir Microsoft hesabı (MSA) kullanarak oturum açtıysanız ve Azure tarafından yönetilen bir iş veya okul hesabını kullanarak Azure DevOps Hizmetleri veya Microsoft Azure içeren bir aboneliğiniz varsa, bu iki aboneliği bağlayabilirsiniz. Buna "alternatif hesap" adı verilir ve okul veya iş hesaplarınızla ilişkili mevcut Azure, Azure DevOps Hizmetleri veya IDE kaynaklarına erişirken aboneliğinizde oturum açmak için MSA'nızı kullanmaya devam etmenizi sağlar.

Her iki hesap da aynı e-posta adresini kullansa bile, her iki hesaptaki avantajlarınızı kullanmak için iş/okul hesabınızı alternatif olarak eklemeniz gerekir.

Alternatif bir hesap eklemek için:

1. Microsoft Hesabınızla (MSA) [Abone Portalında](https://my.visualstudio.com?wt.mc_id=o~msft~docs) oturum açın.
2. **Abonelikler** sekmesine tıklayın.
3. **İlgili Bağlantılar**altında, alternatif **hesap ekle'yi**seçin.
4. İş/okul hesabı e-posta adresinizi girin ve **Ekle'yi**tıklatın.

Alternatif bir hesap ekledikten sonra, **Abonelikler** sekmesini tıklayarak ve **İlgili Bağlantılar** bölümünde uygun seçeneği seçerek istediğiniz zaman bu hesabı güncelleyebilir veya kaldırabilirsiniz.

## <a name="frequently-asked-questions"></a>Sık sorulan sorular

### <a name="q-if-i-do-not-renew-my-subscription-how-long-will-microsoft-keep-my-data"></a>S: Aboneliğimi yenilemezsem, Microsoft verilerimi ne kadar süreyle saklar?
C: Satın alma kanalına bağlı olarak, Microsoft müşteri verilerini farklı uzunluklarda tutar.

| Kanal                                                | Süre (gün) |
|--------------------------------------------------------|-----------------|
|    Perakende satış (örn. Microsoft Mağazası aracılığıyla)               |    360          |
|    Toplu LisansLama Hizmet Merkezi (VLSC)              |    180          |
|    Microsoft Ürün ve Hizmet Sözleşmesi (MPSA)    |    180          |
|    Görsel Stüdyo Bulut Abonelikleri                   |    180          |
|    Program (örneğin MCT, MPN, Imagine)          |    180          |
|    Satış Için Değil (örneğin MVP, MSP)                      |    180          |
|    DevEssentials                                       |    30           |

> [!NOTE]
> - Microsoft Hesabınızı veya Azure REKLAM'ınızı kapatmayı seçerseniz, kişisel verileriniz derhal anonim hale gelecektir.
> - Etkin aboneliğiniz varsa tüm kişisel verilerinizi tam olarak kaldırmayız.
