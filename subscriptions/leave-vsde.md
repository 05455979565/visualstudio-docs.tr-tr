# <a name="how-do-i-leave-the-visual-studio-dev-essentials-program"></a>Visual Studio geliştirme Essentials programı nasıl bırakılsın mı? 
Visual Studio geliştirme Essentials program bırakmak isterseniz, lütfen şu adımları izleyin:
1. Git [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs)
2. Aboneliğiniz için oturum açın.  (Yardım [oturum açma](/visualstudio/subscriptions/signing-in).)
3. Başarılı oturum açma geçerli abonelik listesi görüntülenir.
4. Visual Studio geliştirme Essentials ile ilişkili satırındaki tıklatın **bırakın program** en sağdaki bağlantılar sütun.
5. Bir pencere görüntülenir. Seçin **onaylayın**. 

> [!NOTE]  
> Visual Studio geliştirme Essentials programın gelmeye kolaydır.  Yalnızca 1-5 adımlarını izleyin ve tıklayın **katılma Visual Studio geliştirme Essentials** Bağlantılar sütunun üstündeki düğmesi. Ardından **Onayla**.
