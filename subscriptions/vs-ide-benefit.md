# <a name="using-the-visual-studio-2017-benefit"></a>Visual Studio 2017 avantajı kullanma
Visual Studio abone olarak, Visual Studio IDE erişebilirsiniz.  Sürüm aboneliğiniz düzeyini bağlıdır.  Avantajı araçları kategorideki bir kutucuk olarak görünür [avantajları](https://my.visualstudio.com/benefits) abone portal sayfası.  

(Visual Studio Enterprise Bu örnek için kullanılır.). 

IDE karşıdan yüklemek için:
1. Tıklatın **karşıdan** Visual Studio avantajı döşemenin sol alt bağlantıda. Ayrıca buradan indirebilirsiniz [indirmeleri](https://my.visualstudio.com) sayfası. 

![Visual Studio Enterprise döşeme](_img\vs-ide-experience\vs-ide-tile.png)

2.  Avantajı döşemenin altındaki "İndir" bağlantısını tıklatın.  Visual Studio IDE karşıdan yüklemek ve etkinleştirmek için bir ürün anahtarı edinmek için Fırsat olduğu kuruluş için indirme sonuçlarını sayfasına yönlendirilir. Bir ürün anahtarı üzerinde mavi tıklayarak da talep **alma anahtar** düğmesini veya bir anahtarı daha sonra talep [ürün anahtarları](https://my.visualstudio.com/productkeys) sayfası.
3.  Karşıdan yükleme sonuçları sayfasının Ayrıntılar sekmesinde:
- İstenen Mimarinizi (x86, x64) seçin
- Dil tercihiniz seçin. 
4.  Yeşil İndir düğmesini tıklayın.
![Visual Studio Enterprise Yükleme ayrıntıları](_img\vs-ide-experience\vs-ide-download-details-cropped.png)
5.  Şimdi yüklemek için "Çalıştır" öğesini tıklatın.
6.  Kullanıcı Hesabı Denetimi iletişim kutusunda, "Evet" i tıklatın.
7.  Tıklatın **devam** lisans koşullarını kabul etmek için.
8.  Yükleyici başladığında, karşılık gelen sekmelerinde seçerek istenen iş yükleri, ayrı ayrı bileşenlerin ve dil paketlerini seçin.  
![Visual Studio Enterprise özelleştirme](_img\vs-ide-experience\vs-ide-customize-install-cropped.png)
9.  **Yükle**'ye tıklatın. 
10. Yükleme tamamlandığında yüklemeyi değiştirmek veya Visual Studio'yu başlatın seçeneği gerekir.  Tıklatın **başlatma** başlamak için.  
![Visual Studio Enterprise başlatma](_img\vs-ide-experience\vs-ide-launch-cropped.png)
11. Visual Studio aboneliğinizle ilişkili hesabınızla oturum açın. 
12. Tercih edilen tema seçin ve tıklatın **Start Visual Studio**.
13. Başlangıç sayfası görüntülenir.  Visual Studio kullanmaya başlamak hazırsınız!
![Visual Studio Enterprise kullanmaya başlama](_img\vs-ide-experience\vs-ide-start-cropped.png)
