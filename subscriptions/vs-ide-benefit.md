# <a name="using-the-visual-studio-2017-benefit"></a>Visual Studio 2017 avantajı kullanma

Visual Studio abone olarak, Visual Studio IDE erişebilirsiniz.  Belirli sürümü aboneliğinizi düzeyini bağlıdır.  Avantajı araçları kategorideki bir kutucuk olarak görünür [avantajları](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) abone portal sayfası.  


## <a name="activation-steps"></a>Etkinleştirme adımları

*(Visual Studio Enterprise Bu örnek için kullanılır.)*

IDE karşıdan yüklemek için:
1. Oturum [ https://my.visualstudio.com/benefits ](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs). 
2. Visual Studio IDE kutucuğu Araçlar bölümünde bulun ve tıklayın **karşıdan** avantajı döşemenin altındaki bağlantıyı.  Ayrıca buradan indirebilirsiniz [indirmeleri](https://my.visualstudio.com/downloads?wt.mc_id=o~msft~docs) sayfası. 

    ![Visual Studio Enterprise döşeme](_img\vs-ide-experience\vs-ide-tile.png)

3.  ' I tıklatın, Visual Studio IDE karşıdan yüklemek ve etkinleştirmek için bir ürün anahtarı edinmek için Fırsat olduğu kuruluş için indirme sonuçlarını sayfasına yeniden yönlendirilen. Bir ürün anahtarı üzerinde mavi tıklayarak da talep **alma anahtar** bağlantı veya bir anahtarı daha sonra talep [ürün anahtarları](https://my.visualstudio.com/productkeys) sayfa.
4.  Karşıdan yükleme sonuçları sayfasının Ayrıntılar sekmesinde:
    - İstenen Mimarinizi (x86, x64) seçin
    - Dil tercihiniz seçin. 
5.  Mavi İndir düğmesini tıklayın.
    ![Visual Studio Enterprise Yükleme ayrıntıları](_img\vs-ide-experience\vs-ide-download-details.png)
6.  Şimdi yüklemek için tıklatın **çalıştırmak**.
7.  Kullanıcı Hesabı Denetimi iletişim kutusunda tıklatın **Evet**.
8.  Tıklatın **devam** lisans koşullarını kabul etmek için.
9.  Yükleyici başladığında, karşılık gelen sekmelerinde seçerek istenen iş yükleri, ayrı ayrı bileşenlerin ve dil paketlerini seçin.  
    ![Visual Studio Enterprise özelleştirme](_img\vs-ide-experience\vs-ide-customize-install-cropped.png)

10.  **Yükle**'ye tıklatın. 

11. Yükleme tamamlandığında yüklemeyi değiştirmek veya Visual Studio'yu başlatın seçeneği gerekir.  Tıklatın **başlatma** başlamak için.  

    ![Visual Studio Enterprise başlatma](_img\vs-ide-experience\vs-ide-launch-cropped.png)

12. Visual Studio aboneliğinizle ilişkili hesabınızla oturum açın. 

13. Tercih edilen tema seçin ve tıklatın **Start Visual Studio**.

14. Başlangıç sayfası görüntülenir.  Visual Studio kullanmaya başlamak hazırsınız!

    ![Visual Studio Enterprise kullanmaya başlama](_img\vs-ide-experience\vs-ide-start-cropped.png)
