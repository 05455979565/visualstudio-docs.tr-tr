# <a name="use-microsoft-azure-in-visual-studio-subscriptions"></a>Visual Studio aboneliklerinde Microsoft Azure kullanma
Bir Visual Studio abonesi olarak, ek ücret ödemeden Microsoft Azure kullanabilirsiniz.  [Aylık Azure krediniz](https://azure.microsoft.com/pricing/member-offers/msdn-benefits-details/)sayesinde Azure, geliştirme ve test için kişisel korumalı alanınız.  Sanal makineleri, bulut hizmetlerini ve diğer Azure kaynaklarını sağlayabilirsiniz.  Kredi tutarları abonelik düzeyine göre farklılık gösterir.

## <a name="activation-steps"></a>Etkinleştirme adımları
1. [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) adresinde oturum açın.

2. Avantajlar sayfasında Araçlar bölümünde yer alan Azure kutucuğunu bulun ve avantaj kutucuğunun altındaki bağlantıyı **Etkinleştir** ' e tıklayın.
   > [!div class="mx-imgBorder"]
   > ![Azure kutucuğu](_img/vs-azure/vs-azure-tile.png)

3. Mevcut bir Azure aboneliğiniz yoksa, Azure aboneliğinizi oluşturmak için gerekli bilgileri doldurmanız, koşulları & koşulları kabul etmek için onay kutusuna tıklamanız ve **Kaydol**' a tıklamanız istenir.
   > [!div class="mx-imgBorder"]
   > ![Azure kaydolma](_img/vs-azure/vs-azure-sign-up-cropped.png) mevcut bir aboneliğiniz varsa, sözleşmeyi kabul etmek için onay kutusuna tıklayın ve **Kaydol**' a tıklayın.

4. Hesabınız hazırsanız, bir Azure destek planı seçme (Ayrıca satılan) veya Azure aboneliğinizi kullanmaya başlama seçeneğiniz olacaktır.
   > [!div class="mx-imgBorder"]
   > ![Azure hazırlanıyor](_img/vs-azure/vs-azure-getting-ready-cropped.png)

5. Azure panosu hızlı başlangıç merkezi yüklenir.  
   > [!div class="mx-imgBorder"]
   > ![Azure panosu](_img/vs-azure/vs-azure-dashboard-quickstart.png) 

6. Gelecekte kolay erişim için [Azure portalına](https://portal.azure.com) yer işareti ekleyin.

## <a name="frequently-asked-questions"></a>Sık sorulan sorular
### <a name="q-how-do-i-submit-a-technical-support-incident-from-within-the-azure-portal"></a>Ç Nasıl yaparım? Azure portal içinden bir teknik destek olayı mı gönder?
Y:  Azure portal bir destek olayı göndermek, üç adımlı bir işlemdir.
1. Teknik destek avantajınızı etkinleştirin ve sözleşme KIMLIĞI erişim KIMLIĞINIZI alın.
2. Destek sözleşmenizi Azure aboneliğinize bağlayın.
3. Destek olayı gönderme.

Tüm ayrıntılar için lütfen [Teknik destek](vs-tech-support.md) belgelerini ziyaret edin.

### <a name="q-ive-received-a-message-stating-that-my-access-to-azure-will-end-soon--what-can-i-do-to-continue-using-azure"></a>Ç Azure 'a erişimimin yakında bitediğini belirten bir ileti aldım.  Azure 'ı kullanmaya devam etmek için ne yapabilirim?
Y:  Bu e-postayı, Visual Studio aboneliğinizle ilişkili Azure avantajını kullandığınız ve aboneliğin süresi sona erdiği veya kaldırıldığı için alıyorsunuz.  Azure 'ı kullanmaya devam etmek için çeşitli seçenekleriniz vardır:

**Önerilen eylem:**

Visual Studio Azure kredilerinizi ve indirimlerinizi kullanmaya devam etmek için aşağıdaki adımlardan birini uygulayın:

Takımınızda veya kuruluşunuzdaki bir kişi sizin adınıza abonelik satın alıyorsa, kuruluşunuzun Visual Studio yöneticisiyle iletişim kurun ve ihtiyacınız olan Azure aylık kredi miktarını sahibine bir Visual Studio aboneliği isteyin.   Lütfen, Azure aboneliğinizi aktarma konusunda endişelenmenize gerek kalmaması için, önceki aboneliğinizin atandığı kullanıcı KIMLIĞINE yeni aboneliği atamak üzere yönetici isteyin.

Aboneliği doğrudan satın alabilirsiniz, lütfen aşağıdakilerden birini yapın:
1. Visual Studio Market aracılığıyla yıllık bir Visual Studio [Enterprise](https://marketplace.visualstudio.com/items?itemName=ms.vs-enterprise-annual) veya [Professional](https://marketplace.visualstudio.com/items?itemName=ms.vs-professional-annual) bulut aboneliği satın alın.
2. Geçerli aboneliğinizi [Microsoft Store](https://www.microsoft.com/en-us/search/result.aspx?q=visual+studio+subscription)aracılığıyla yenileyerek% 50 veya daha yüksek bir indirimden yararlanın.

Tüm [abonelikleri ve bunların avantajlarını](https://visualstudio.microsoft.com/vs/pricing/)karşılaştırın.

Bir Microsoft hesabı kapsamında **aynı** abonelik düzeyinde başka bir etkin Visual Studio aboneliğiniz varsa, Visual Studio abonelik portalına [alternatif bir hesap](manage-vs-subscriptions.md#linking-my-subscription-to-existing-azure-devops-services-or-azure-subscriptions) ekleyerek bu Azure avantajına bağlayabilirsiniz. Bu işe yaramazsa, [Azure avantajınızı](/azure/billing/billing-subscription-transfer/) etkin Visual Studio aboneliğine aktarmayı da deneyebilirsiniz.

**Farklı** bir abonelik düzeyinde başka bir etkin Visual Studio aboneliğiniz varsa ve bir Microsoft hesabı altında yardım için [Azure desteği](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) 'ne başvurun.

Aboneliğinizin harcama limiti varsa ve süresi dolmadan önce hiçbir işlem yapmadıysanız, Azure aboneliğiniz devre dışı bırakılacak. Bunu bir Kullandıkça Öde teklifine dönüştürmek ve kullanmaya devam etmek için bir ödeme aracı ekleyin ve [harcama limitinizi kaldırın](/azure/billing/billing-spending-limit/).

Bu Visual Studio Azure aboneliğine devam etmeyi tercih etmemeyi tercih ediyorsanız, sanal makineler ve veritabanları gibi kaynaklarınızı başka bir Azure aboneliğine aktarabilir ve bu aboneliği dilediğiniz zaman [iptal](https://azure.microsoft.com/support/cancel-account/) edebilirsiniz.

Bu e-postayı hatada aldığınızı düşünüyorsanız lütfen [Azure müşteri desteği](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)'ne başvurun.
