# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Visual Studio aboneliklerinde Enterprise Mobility + Security (EMS)

Visual Studio Enterprise yıllık bulut ve standart abonelikler Enterprise Mobility + Security (EMS) içerir.  Kimlik, cihaz, uygulama ve verilerinizi EMS-Azure Active Directory, Microsoft Intune, Azure Information Protection, Microsoft Advanced Threat Analytics ve Microsoft Cloud App Security ile güvenli hale getirin ve yönetin. 

## <a name="who-should-install-ems"></a>EMS 'yi kimler yüklemelidir?
Zaten bir Office 365 Geliştirici (E3) kiracınız varsa EMS 'yi yüklemelisiniz.  Aksi takdirde, hem EMS hem de Office 365 Developer (E5) ve aşağıdakiler de dahil olmak üzere [Microsoft 365](vs-m365.md)yüklemelisiniz:
- Power BI ile gelişmiş analiz
- Office 365 Gelişmiş tehdit koruması
- Gelişmiş kimlik ve erişim yönetimi çözümleri oluşturmak için Azure Active Directory
