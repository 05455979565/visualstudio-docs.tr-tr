# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Visual Studio Aboneliklerinde Kurumsal Mobilite + Güvenlik (EMS)

Visual Studio Enterprise yıllık bulut ve standart abonelikleri Enterprise Mobility + Security (EMS) içerir.  EMS - Azure Active Directory, Microsoft Intune, Azure Bilgi Koruması, Microsoft Advanced Threat Analytics ve Microsoft Cloud App Security ile kimliklerinizi, cihazlarınızı, uygulamalarınızı ve verilerinizi güvenli hale verin ve yönetin. 

## <a name="who-should-install-ems"></a>EMS'yi kimler yüklemeli?
Zaten bir Office 365 Geliştirici (E3) kiracınız varsa, EMS yüklemeniz gerekir.  Aksi takdirde, hem EMS hem de Office 365 Geliştiricisi (E5) içeren [Microsoft 365'i](vs-m365.md)ve aşağıdakileri yüklemeniz gerekir:
- Power BI ile gelişmiş analitik
- Office 365 Gelişmiş Tehdit Koruması
- Gelişmiş kimlik ve erişim yönetimi çözümleri oluşturmak için Azure Active Directory
