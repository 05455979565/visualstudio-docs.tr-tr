# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Visual Studio aboneliklerinde Enterprise Mobility + Security (EMS)

Daha önce seçilen Visual Studio aboneliklerinde bağımsız olarak bulunan Enterprise Mobility + Security, artık seçili Visual Studio aboneliklerine dahil olan Microsoft 365 Geliştirici aboneliğine eklenmiştir.  

Sonuç olarak, Enterprise Mobility + Security artık ayrı olarak sunulmayacaktır.  

Microsoft 365 Geliştirici aboneliğinizi etkinleştirmek için:
1. [Visual Studio abonelikleri portalında](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) oturum açın
1. Microsoft 365 Geliştirici aboneliği kutucuğunu bulun ve **Etkinleştir**' e tıklayın.  
1. Hesabınızı oluşturmayı tamamlamaya yönelik yönergeleri izleyin.  

[Geliştirici aboneliğini Microsoft 365](vs-m365.md)etkinleştirme hakkında daha fazla bilgi edinin. 

## <a name="support-resources"></a>Destek kaynakları
- EMS ile ilgili yardıma mı ihtiyacınız var?  Çevrimiçi [Enterprise Mobility + Security belgelerine](/enterprise-mobility-security/)göz atın.
- EMS 'nin herhangi bir bileşeniyle teknik destek için lütfen sorunuz için doğru desteği bulmak üzere [Enterprise Mobility + Security destek SSS](https://docs.microsoft.com/enterprise-mobility-security/solutions/ems-support-faq) ' yi ziyaret edin.
- Visual Studio abonelikleri için Sales, abonelikler, hesaplar ve faturalandırma konusunda yardım için Visual Studio [abonelikleri desteğiyle](https://visualstudio.microsoft.com/subscriptions/support/)görüşün.
- Visual Studio IDE, Azure DevOps Services veya diğer Visual Studio ürünleri veya hizmetleri hakkında sorularınız mı var?  [Visual Studio desteği](https://visualstudio.microsoft.com/support/)' ni ziyaret edin.
