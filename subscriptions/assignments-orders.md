# <a name="track-user-assignment-and-process-orders"></a>Kullanıcı atamave işlem emirlerini izleme
Visual Studio abonelik yöneticilerinin, Toplu Lisanslama sözleşmelerinde veya Microsoft Ürün ve Hizmet Sözleşmesi'nde belirtilen zamanlamadaki kullanım artışları için Visual Studio kullanımını ve işlem emirlerini izlemeleri beklenir. Yeni Visual Studio Abonelikleri Yönetimi portalı, mevcut ve kullanılmış lisanslarınızı gösteren yararlı bir izleyici ile bu basit hale getirmiştir.

## <a name="maximum-usage"></a>En fazla kullanım
**Şirketinizin Visual Studio aboneliklerini satın alma yükümlülüğü aşağıdaki anda hemen yürürlüğe girer:**
- Bir lisans bir **kullanıcıya** atanır VEYA
- Bir kullanıcı Visual Studio yazılımı ile etkileşime girer.

Tam satın alma yükümlülüğünüz, kullanıcılara atadığınız maksimum abonelik sayısına göre belirlenir. Bu abonelik atama düzeyi, günlük kullanıcı atamalarında veya Visual Studio yazılımıyla etkileşimde bulunan kullanıcılarda (hangisi daha yüksekse) en yüksek noktadır.

- Visual Studio aboneliklerinin ek kullanıcılara atanması maksimum kullanım düzeyini yükseltir.  
- Visual Studio abonelikleri yöneticileri, bir atamada azalma ve diğerinde artış oluşturacak bir birey için atanan abonelik düzeyini değiştirebilir. Bir abonenin atanan abonelik düzeyini düşürdüğünde, kişinin yalnızca üst düzey abonelikteki herhangi bir şeyi kullanmayı derhal durdurması ve kaldırması gerekir. 
- Visual Studio abonelikleri yöneticileri, orijinal atama tarihinden bu yana 90 gün geçtiyse, bir aboneden diğerine abonelikleri yeniden atayabilir. 
    > [!NOTE]
    > Yapay olarak yüksek düzeyde maksimum kullanımdan kaçınmak için, bunu her zaman önce varolan aboneliği kaldırarak ve sonra yenisini ekleyerek yapın. 
- Kuruluşunuzun maksimum kullanımını izlemenize yardımcı olmak için Visual Studio [Abonelikleri Yönetim Portalı'nda](https://manage.visualstudio.com) [bir Maksimum Kullanım](maximum-usage.md) raporu bulunmaktadır. 

## <a name="monthly-subscriptions-open-license-or-open-value"></a>Aylık Abonelikler, Açık Lisans veya Açık Değer
Abonelikleri Açık Lisans ve Açık Değer gibi programlar aracılığıyla veya Aylık olarak Visual Studio Marketplace aracılığıyla atayabilirsiniz. Bu durumda, kullanıcıların (çalışanlar veya harici yükleniciler) Visual Studio lisanslı yazılımlarla etkileşime girdiği ay boyunca ek kullanıcılar için siparişinizi işlemeniz gerekir.

## <a name="enterprise-mpsa-and-select-agreements"></a>Kurumsal, MPSA ve Select Anlaşmaları
Microsoft Kurumsal Sözleşmeler (EA), MPSA ve Select Plus Anlaşmaları, Visual Studio yazılımlarını zaman içinde nasıl kullandığınız ve lisanslamanız konusunda esneklik sağlar. Visual Studio yöneticileri, yazılım lisanslarını anlaşma döneminde belirlenen maksimum kullanıma getirmek için yıllık True-Up siparişi vermelidir.
