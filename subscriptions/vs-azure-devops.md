# <a name="azure-devops-services-benefits-for-visual-studio-subscribers"></a>Visual Studio aboneleri için Azure DevOps Hizmetleri avantajları

Etkin bir Visual Studio abonesi, çeşitli özellikler, Azure DevOps Hizmetleri'nde, bir avantajı olarak, aboneliğiniz ile birlikte kullanabilirsiniz. Her Azure DevOps Services hesabındaki hesap oluşturduğunuz veya başka biri tarafından eklenen üyesi olduğunuz aynı özelliklerin kullanmanız mümkün olacaktır.

Oturum açtığınızda Azure DevOps kullanarak hizmetler için Visual Studio aboneliğinizi etkinleştirme için kullanılan kimlik veya [alternatif kimlik](vs-alternate-identity.md), biz bunu otomatik olarak tanıyacak.

[Ücretsiz bir Azure DevOps Services hesabı için kaydolun](https://visualstudio.microsoft.com/team-services/)

## <a name="eligibility"></a>Uygunluk

| Abonelik düzeyi                                                 |     Kanallar                                            | Faydası                                                          | Yenilenebilir?    |
|--------------------------------------------------------------------|---------------------------------------------------------|------------------------------------------------------------------|---------------|
| Visual Studio Enterprise (standart, yıllık bulut, aylık bulut)   | VL, Azure, perakende, seçili NFR<sup>1</sup>, FTE  | Temel, Test Yöneticisi, paket yönetimi, CI/CD şirket içinde barındırılan       |  Evet          |
| Visual Studio Professional (standart, yıllık bulut, aylık bulut) | VL, Azure, perakende                                        | Temel                                                            |  Evet          |
| Visual Studio Test Professional (standart)                         | Toplu Lisans, perakende                                              | Temel, Test Yöneticisi                                              |  Evet          |
| MSDN platformları (standart)                                          | Toplu Lisans, perakende                                              | Temel, Test Yöneticisi                                              |  Evet          |

<sup>1</sup>*içerir: değil satılamaz (NFR), Microsoft değerli ortak (MVP), bölge Yöneticisi (RD), Visual Studio Endüstri ortağı (VSIP), Microsoft iş ortağı ağı (Kurumsal), BizSpark, MCT yazılımı ve Hizmetleri Geliştirici.   Hariç tutar: MCT yazılım ve Hizmetleri düşünün.*

Azure DevOps Services özellikleri hakkında daha fazla bilgi edinin:

- [Temel](https://visualstudio.microsoft.com/team-services/compare-features/)
- [Test Yöneticisi](https://marketplace.visualstudio.com/items?itemName=ms.vss-testmanager-web)
- [Paket Yönetimi](https://marketplace.visualstudio.com/items?itemName=ms.feed)

Emin değil hangi aboneliği, kullanmakta olduğunuz?  Bağlanma [ https://my.visualstudio.com/subscriptions ](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) size atanan tüm abonelikleri görmek için.
Tüm aboneliklerinizi görmüyorsanız, bir veya daha çok farklı bir e-posta adresine atanmış olabilir.  Bu Aboneliklerdeki görmek için bu e-posta adresiyle oturum açmanız gerekir.
