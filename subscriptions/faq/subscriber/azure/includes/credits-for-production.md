## <a name="can-i-use-my-monthly-azure-credits-to-run-production-applications"></a>Üretim uygulamalarını çalıştırmak için aylık Azure kredilerimi kullanabilir miyim?

Visual Studio abonelerinin aylık Azure kredileri özel olarak yalnızca bireysel geliştirme/test kullanımına yöneliktir. Üretim iş yüklerini çalıştırmak için [kullandıkça öde](https://azure.microsoft.com/offers/ms-azr-0003p/) fiyatlandırması gibi ücretli bir Azure aboneliğine yükseltmeniz gerekir.
