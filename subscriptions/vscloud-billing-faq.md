# <a name="visual-studio-cloud-subscriptions-billing-faq"></a>Visual Studio bulut abonelikleri faturalandırma SSS
Her Visual Studio aboneliğinin avantajlarını anlamak için [bulut aboneliği avantajlarını ve fiyatlandırmayı](https://visualstudio.microsoft.com/vs/pricing/) bulut ve standart Visual Studio abonelikleri arasındaki karşılaştırmalar, abone avantajlarıyla ilgili ayrıntılar ve daha fazlası ile karşılaştırdıktan emin olun.

## <a name="general-purchasing-questions"></a>Genel satın alma soruları
### <a name="q-can-i-buy-visual-studio-cloud-subscriptions-using-a-purchase-order"></a>S: Satın Alma Siparişi kullanarak Visual Studio bulut aboneliklerini satın alabilir miyim?
C: Hayır. Tüm Visual Studio bulut abonelikleri Azure aboneliği kullanılarak satın alınmalıdır. (Azure faturalandırma hesabınız olarak düşünün.)

### <a name="q-what-types-of-azure-subscriptions-can-be-used-to-buy-visual-studio-cloud-subscriptions"></a>S: Visual Studio bulut abonelikleri satın almak için ne tür Azure abonelikleri kullanılabilir?
C: Çoğu Azure aboneliği kullanılabilir - Kurumsal Sözleşmenize bağlı Azure aboneliklerini [(EA)](https://azure.microsoft.com/pricing/enterprise-agreement/)destekleriz, Bulut Çözüm Sağlayıcıları (CSP'ler tarafından ayarlanan Azure abonelikleri), Microsoft Open License satıcıları aracılığıyla ayarlanan Azure aboneliklerini ve Kullandıkça Öde Azure aboneliklerini destekleriz.

[Azure Ücretsiz Deneme](https://azure.microsoft.com/pricing/free-trial/) sürümü ve Visual Studio aboneliklerinde abone avantajı olarak dahil edilen abonelikler de dahil olmak üzere bazı Azure abonelik türleri kullanılamaz.

### <a name="q-am-i-required-to-buy-other-azure-services"></a>S: Diğer Azure hizmetlerini satın almam gerekiyor mu?
C: Hiç de değil. Visual Studio bulut aboneliklerini yalnızca Azure üzerinden satın almak istiyorsanız, bunu yapabilirsiniz.

## <a name="enterprise-agreement-ea-customers"></a>Kurumsal Sözleşme (EA) müşterileri
### <a name="q-can-i-use-an-enterprise-agreement-to-buy-visual-studio-cloud-subscriptions"></a>S: Visual Studio bulut abonelikleri satın almak için Kurumsal Sözleşme kullanabilir miyim?
C: Evet, yapabilirsin. EA'nız için oluşturulan bir Azure aboneliği için sahibi veya katılımcısı olmanız gerekir. Lütfen Visual Studio bulut abonelikleri için alışverişlerinizi doğrudan Visual Studio Marketplace'te yaptığınızdan emin olun. SatınAlma Siparişi kullanarak Visual Studio bulut aboneliklerini satın alamazsınız.

### <a name="q-how-can-i-tell-whether-i-have-the-necessary-privileges-to-buy-services-in-the-visual-studio-marketplace-through-my-organizations-enterprise-agreement"></a>S: Kuruluşumun Kurumsal Sözleşmesi aracılığıyla Visual Studio Marketplace'te hizmet satın almak için gerekli ayrıcalıklara sahip olup olmadığımı nasıl anlayabilirim?
C: Doğru ayrıcalıklara sahip olup olmadığınızı belirlemek için en kolay yaklaşım, Visual Studio Marketplace'te sunulan bir hizmet için **Satın Al** düğmesini tıklatmaktır.
Şu anda oturum açmanızla bağlantılı olan azure abonelikleri listesinden bir Azure aboneliği (fatura hesabı olan) seçmeniz gerekir.
Azure aboneliğinin adı fatura hesabı türüne ("Ödeme Yap", "Kurumsal Sözleşme" vb.) bağlı olduğundan, Azure aboneliğinin Kurumsal Sözleşmenizin bir parçası olup olmadığı genellikle açıktır.

Diğer bir yaklaşım da [Azure Kurumsal Portalı'nı](https://ea.azure.com)ziyaret etmeye çalışmaktır.  Başarılı bir şekilde erişebiliyorsanız, kurumsal yönetici veya Hesap Sahibi rolüne zaten sahipsiniz demektir. Kurumsal Sözleşme'de yalnızca Hesap Sahipleri yeni Azure fatura hesapları ayarlayabilir. Azure Kurumsal Portalı'na erişemiyorsanız, lütfen Kurumsal Yöneticinizin kim olduğunu öğrenmek için kuruluşunuz içinde bilgi alın ve bu kişiden sizi Azure Kurumsal Portalı'na Hesap Sahibi olarak eklemesini isteyin.  Bu kişiyi bulamıyorsanız, bir [destek bileti gönderebilir](https://support.microsoft.com/supportrequestform/cf791efa-485b-95a3-6fad-3daf9cd4027c) ve iletişim bilgilerini isteyebilirsiniz.  Destek bileti için kuruluşunuzun adını ve Kurumsal Sözleşme kayıt numaranıza ihtiyacınız vardır.

### <a name="q-can-i-use-the-azure-monetary-commitment-funds-from-my-enterprise-agreement-to-buy-visual-studio-cloud-subscriptions"></a>S: Visual Studio bulut abonelikleri satın almak için Kurumsal Sözleşmemdeki Azure Parasal Taahhüt fonlarını kullanabilir miyim?
C: Hayır, bu ön ödemeli fonlar Visual Studio bulut aboneliklerini satın almaya uygun değildir. EA'nızın Visual Studio bulut aboneliklerini satın alması için oluşturulan bir Azure aboneliğini seçtiğinizde, ücretler bir sonraki "aşım" faturanızda görünür. Genellikle bu aylık olarak gerçekleşir, ancak bazı EA müşterileri için geçmiş kurallar nedeniyle, bir aşım faturası birkaç ay için düzenlenmeyebilir. Ne kadar ek satın alma işleminin (Azure Parasal Taahhüt fonları için uygun olmayan satın almalar) bir aşım faturayı tetiklediğini bilmeniz gerekiyorsa, lütfen EA'nız için bir lisans uzmanına danışın.

## <a name="how-charges-are-processed"></a>Ücretler nasıl işlenir?
### <a name="q-how-are-monthly-cloud-subscription-charges-processed"></a>S: **Aylık** bulut abonelik ücretleri nasıl işlenir?
C: İlk satın alma işleminde, geçerli ayın kalan günlerini kapsayacak şekilde eşit olarak verilen bir miktarı faturalandırıyoruz. Örneğin, 15 Nisan'da 10 Visual Studio Professional aylık bulut aboneliği satın alındıysa, ayın %50'si kaldığı için 5 birim ücret alabiliriz (30 günlük ayın 15 günü).
Mayıs ayının ilk günü ve bundan sonraki her ay iptal edene kadar, 10 birimin tamamı faturalandırılır.

Daha sonra ödenen miktarı artırdığınızda, artırılacak birimleri de geçerli aydaki kalan günleri kapsayacak şekilde çoğaltıyoruz. Yani 10 Mayıs'ta 1 daha fazla Visual Studio Professional aylık bulut aboneliği satın aldıysanız, biz yaklaşık 0,677 adet (21 gün Mayıs 31 günlük ay kalan) fatura olur.

### <a name="q-how-are-annual-cloud-subscription-charges-processed"></a>S: **Yıllık** bulut abonelik ücretleri nasıl işlenir?
C: Her satın alma işleminde, satın alınan miktarın tamamını hemen fatura ediyoruz. Ücretler yıl boyunca yayılmaz ve prorating yoktur. Yılın farklı saatlerinde yıllık bulut abonelikleri satın alırsanız, aboneliklerin farklı aylarda yenilenmesi gerekir. Microsoft toplu lisanslama sözleşmesi satın alma da yaygın olduğu gibi, bir müşterinin yıllık bulut aboneliklerinin tümlerini birbiriyle uyumlu hale getirmeyiz.

### <a name="q-how-do-cancelations-work"></a>S: İptaller nasıl çalışır?
C: Visual Studio bulut aboneliğini iptal ettiğinizde, otomatik yenilemeyi iptal emziyorsunuz. Abonelik normal yenileme tarihine kadar devam eder ve sonra sadece süresi doluyor.
Son kullanma tarihi sonunda, Visual Studio abonesi artık Visual Studio'yu veya abonelikten diğer avantajları kullanamaz.

Aylık bulut aboneliklerinde, iptaller bir sonraki ayın ilk günü yürürlüğe girer. Yalnızca aylık bulut aboneliklerinizin bazılarını iptal ederseniz, doğru kişilerin etkin abonelikler atanmış olmaya devam etmesini sağlamak için sonraki ayın ilk günü kullanıcıları kaldırdığınızdan emin olun.

Yıllık bulut abonelikleri için iptaller, ilk satın alma işleminden sonraki 12 ayı izleyen ayın ilk günü veya son yıllık yenileme ücretinden 12 ay sonra yürürlüğe girer. Örneğin, 3 Ocak 2018'de Visual Studio Professional yıllık bulut aboneliği satın aldıysanız, 1 Şubat 2019'a kadar etkin kalır ve bu abonelik bir yıl daha otomatik olarak yenilenir. O tarihten 1 Şubat 2020'ye kadar herhangi bir zamanda iptal ederseniz, abonelik 1 Şubat 2020 tarihinde sona erer. Yıllık bulut abonelikleriyle abonelik yılı boyunca kısmen iptal etmek için indirim yapılmaz.

### <a name="q-what-kind-of-volume-discounts-are-available-for-visual-studio-subscriptions"></a>S: Visual Studio abonelikleri için ne tür toplu indirimler kullanılabilir?
C: Her abonelik *türünde* 6'ncı ve sonraki tüm aboneliklerde %5 indirim alırsınız:

* Visual Studio Professional aylık
* Visual Studio Professional yıllık
* Visual Studio Enterprise aylık
* Visual Studio Enterprise yıllık

Örneğin, 6 Visual Studio Professional aylık abonelik ve 5 Visual Studio Enterprise aylık abonelik satın alırsanız, 5 Professional'da normal fiyatı ödeyecek, 6. Abonelik.

Ayrıca, indirim yalnızca belirli bir aylık fatura dönemindeki ücretler için geçerlidir. Yani bir ay içinde 5 Visual Studio Professional yıllık abonelikleri satın alırsanız ve daha sonra 5 daha sonraki ay satın alırsanız, tüm 10 abonelikleri normal fiyat ödeyeceksiniz.

> [!NOTE]
> Microsoft artık Cloud Subscriptions'da Visual Studio Professional Annual abonelikleri ve Visual Studio Enterprise Annual abonelikleri sunamaz. Mevcut müşterilerin aboneliklerini yenileme, artırma, azaltma veya iptal etme deneyimi ve yeteneğinde herhangi bir değişiklik olmayacaktır. Yeni müşteriler Visual Studio [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) satın almak için farklı seçenekleri keşfetmek için gitmek için teşvik edilir.

## <a name="other-questions"></a>Diğer sorular
### <a name="q-can-i-use-the-monthly-azure-devtest-individual-credit-as-a-visual-studio-subscriber-to-buy-more-visual-studio-cloud-subscriptions"></a>S: Daha fazla Visual Studio bulut aboneliği satın almak için Visual Studio abonesi olarak aylık Azure DevTest bireysel kredisini kullanabilir miyim?
C: Hayır, Visual Studio Marketplace satın alma işlemlerini ödemek için [aylık Azure DevTest bireysel kredinizi](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/) Visual Studio abonesi olarak kullanamazsınız. Visual Studio bulut abonelik satın alımları kredi kartınıza faturalandırılır.
Alışveriş yapmadan önce [harcama sınırınızı kaldırmanız](https://azure.microsoft.com/pricing/spending-limits/)gerekir.

### <a name="q-whats-the-difference-between-annual-and-monthly-cloud-subscriptions"></a>S: Yıllık ve aylık bulut abonelikleri arasındaki fark nedir?
C: Aylık bulut abonelikleri arasında Visual Studio'nun yanı sıra Azure DevOps Hizmetleri ve TFS'nin kullanımı yer almaktadır. Yıllık bulut abonelikleri de buna sahiptir, ancak geliştirme ve test için windows ve diğer Microsoft yazılımlarının kullanımı da dahil olmak üzere abone avantajlarını da içerir, Azure ile denemeler yapmak için kullanılacak aylık Azure DevTest bireysel kredisi bulutta geliştirme ve test yapma, eğitim, destek ve çok daha fazlası.
[Bulut aboneliği avantajlarını ve fiyatlandırmayı karşılaştırın](https://visualstudio.microsoft.com/vs/pricing/)

### <a name="q-do-i-get-new-versions-of-visual-studio-if-i-buy-a-visual-studio-cloud-subscription"></a>S: Visual Studio bulut aboneliği satın alırsam Visual Studio'nun yeni sürümlerini alabilir miyim?
C: Evet. Yeni sürümler yayımlandıkça, bunları karşıdan yükleyebilir ve çalıştırabilirsiniz. Ayrıca önceki sürümleri de çalıştırmaya devam edebilirsiniz.

### <a name="q-can-i-buy-visual-studio-cloud-subscriptions-from-my-software-reseller"></a>S: Visual Studio bulut aboneliklerini yazılım satıcımdan satın alabilir miyim?
C: Evet, satıcınız Bulut Çözüm Sağlayıcısı (CSP) programına katılıyorsa yapabilirsiniz. Onlara sor.
