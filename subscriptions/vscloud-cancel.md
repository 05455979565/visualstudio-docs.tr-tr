# <a name="buying-visual-studio-cloud-subscriptions---cancelling-subscriptions"></a>Visual Studio bulut Abonelikleri - iptal abonelik satın alma

Visual Studio bulut aboneliğini iptal etmek için oturum açmak için Visual Studio [Abonelik Yönetim Portalı](https://manage.visualstudio.com/) orijinal satın alma yapmak için kullanılan Azure aboneliğini erişimi olan bir hesapla.
1.  Sol üst köşedeki uygun Azure aboneliğini seçtiğinizi doğrulayın.
2.  Sol gezinti bölmesinde seçin **azaltmak** iptal etmek istediğiniz abonelik altında.
3.  Hesabınızla ilişkilendirilmiş abonelik sayısını ayarlamak için bir pencere ile sunulur.
4.  İstenen miktar veya (iptal etmek ve sıfır 0) abonelik sayısını azaltmak**devam**.
5.  Siparişinizi gözden geçirmek **Onayla**ve you'redone!

A: Evet kalan abonelik süresi Avantajlarınızı hala erişebilir.
